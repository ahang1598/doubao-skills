# 行业结构分支（Industry Structure Playbooks）

判定本行业属于哪种结构类型后，按对应分支取分析侧重与必查项。一个行业可能带多个标签，但应按**决定价值分配的瓶颈**来选主分支。判定方法见 `task-router.md` 第三节。

## 目录

- 分支选择
- 产业链型（制造/硬科技/新能源/医药/化工）
- 网络型（平台/互联网/SaaS 生态）
- 资产负债型（银行/保险/券商）
- 单店模型型（连锁零售/餐饮/消费医疗）

## 分支选择

按决定性瓶颈选择：

- 价值沿上中下游分布、利润在某环节集中 → **产业链型**（行业研究最常见）。
- 价值来自流动性、网络效应、多边激励、用户黏性 → **网络型**。
- 价值来自息差/承保利润、资产质量、杠杆、风险定价 → **资产负债型**。
- 价值来自单店模型、密度、SOP、复购、坪效 → **单店模型型**。

若多个分支适用，选一个**主分支**（决定价值分配的那个）加一个**叠加分支**。主分支应是决定价值分配的，而非最时髦的标签。政策若能创造或摧毁该市场，叠加一层"政策驱动"（见 `analysis-framework.md` 第七节）。

## 产业链型（制造 / 硬科技 / 新能源 / 医药 / 化工）

**最重要的问题**：这个行业的利润集中在哪个环节、正在往哪迁移，哪个环节有定价权、哪个环节在被压价？

**先查**：

- 产业链上中下游各环节的构成、各环节毛利率与价值占比（利润池）。
- 各环节集中度（CR3/CR5）、头部玩家与阵营、卡脖子环节与国产化进度。
- 产能投放节奏、产能利用率、价格走势、库存位置（周期性环节）。
- 技术路线竞争（如电池技术、半导体制程、面板代际）与渗透率。
- 上游原材料/能源成本对各环节毛利的传导。

**判断**：

- 中游整机/组装常因同质化与价格战被压到低毛利；上游核心元器件/材料或高壁垒环节（如海缆、特种光纤、关键设备）保住高毛利。
- 利润池迁移方向是这类行业的核心判断——点名"剪刀差"在扩大还是收敛。
- 规划产能不等于已释放产能；国产化率高的环节未必有高毛利。
- 技术路线切换会重塑各环节价值，渗透率拐点是关键 T 信号。

**盈利模型**：单位经济（单位售价−单位成本=单位利润）或成本结构拆解；ROE 杜邦 + 现金流三指标；资本开支与营运资本是这类行业的关键。

**必做表**：产业链全景表（上游/中游/下游分层 + 各层价值/盈利特征 + 关键节点份额/毛利率/国产化率注释）、各环节利润池表、头部玩家份额表。

## 网络型（平台 / 互联网 / SaaS 生态）

**最重要的问题**：网络效应与用户黏性是否真实形成、能否把流量转成可持续的货币化与高留存？

**先查**：

- 用户规模/活跃/留存、付费转化、单用户价值（ARPU/LTV）、获客成本（CAC）。
- 多边结构（供给侧 vs 需求侧）、抽佣率（take rate）、变现路径。
- 网络效应的来源（双边、跨边、数据网络）与切换成本。
- 平台对供给/需求方的议价能力、监管与合规风险。

**判断**：

- 市场大可能是"很多人各关心一点点"，要看少数高价值场景的迫切性。
- 网络效应是否复利、是否被多归属（multi-homing）削弱。
- 增长靠产品力还是靠补贴——区分真实留存与买来的活跃。
- 数据只有在专有、难复制、合法可用且因果上提升产品时才算护城河。

**盈利模型**：单用户经济（LTV/CAC、回收期、留存曲线）；成本结构（获客、内容/供给补贴、技术）。

**必做表**：关键运营指标表（规模/活跃/留存/付费/ARPU/CAC）、变现结构表、竞品禀赋对比表。

## 资产负债型（银行 / 保险 / 券商）

**最重要的问题**：盈利来自息差/承保利润还是规模扩张，资产质量与风险定价是否稳健，杠杆与资本约束如何？

**先查**：

- 息差（NIM）/承保利润率、成本收入比、信用成本/赔付率。
- 资产质量（不良率、拨备覆盖、准备金充足率）、资本充足率/偿付能力。
- 规模（总资产/保费/AUM）、杠杆水平、ROE/ROA。
- 监管政策（资本要求、利率市场化、行业准入）。

**判断**：

- 资产负债型行业的"利润"不是卖货毛利，而是利差与风险定价之差。
- 高 ROE 可能来自高杠杆而非高盈利能力——杜邦拆解要看清杠杆贡献。
- 信用成本/赔付的周期性会吞噬利差，资产质量是真实风险所在。
- 利率市场化、监管资本要求是关键政策变量。

**盈利模型**：成本结构变体（息差/承保利润率、成本收入比、信用成本/赔付率）；ROE 杜邦（重点看杠杆与风险成本）。

**必做表**：盈利能力与资产质量表（NIM/成本收入比/不良率/拨备/资本充足率）、可比机构 ROE 杜邦表。

## 单店模型型（连锁零售 / 餐饮 / 消费医疗）

**最重要的问题**：单店模型是否跑通、密度与复购能否支撑扩张，规模化后单店经济是否被稀释？

**先查**：

- 单店模型（单店收入/坪效/翻台率/客单价/单店盈亏平衡）、门店数与扩张节奏。
- 同店增长（SSSG）、复购率/会员留存、毛利率与门店层面利润率。
- 选址密度、供应链与 SOP、加盟 vs 直营结构。
- 品牌力、客流来源、租金/人工成本趋势。

**判断**：

- 单店模型跑通且可复制，是这类行业能否规模化的前提。
- 规模扩张可能稀释单店经济（密度过高自相蚕食、下沉市场客单价低）。
- 同店增长比总收入增长更能反映健康度——总收入靠开店堆出来的要警惕。
- 复购与品牌决定长期价值，价格战拉来的客流留不住。

**盈利模型**：单店模型拆解（单店收入−单店成本=单店利润）+ 门店数 × 单店利润；成本结构（租金/人工/原材料/折旧）。

**必做表**：单店模型拆解表、同店增长与门店扩张表、可比公司门店经济对比表。
