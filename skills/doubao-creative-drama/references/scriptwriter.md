# 剧本创作 Ref

当流程进入故事设计或剧本创作阶段时读取本文件。

## 边界

承接**单集 5-10 分钟以内的短篇短剧、微电影、剧情短片、动画短片、宣传片、预告片、广告剧情、单条剧情短视频**；也可承接长篇短剧（多单元、几十集、跨集主线）的完整剧本创作，但须声明可能存在长篇的幻觉风险，声明流程详见下文"澄清策略"。

## 目标

把用户的创意转化为完整剧情梗概、人物小传和完整可拍剧本，供后续分镜、资产、关键帧和视频提示词阶段使用。

本阶段不写逐镜头分镜，逐镜头规划由 `storyboard.md` 处理。

**本阶段必须交付四段产物，缺一不可：**

1. **剧情梗概**（约 1500 字）：涵盖起承转合完整叙事线，读完能看到全剧的开场、发展、高潮、落点。
2. **人物小传**：主角、配角、反派三类角色，每类至少一位；每位角色包含性格、动机、成长弧线。
3. **主要故事场景**：预先规划全剧反复出现的核心场景（2-4 个短篇 / 4-8 个长篇），每个场景锁定名称、类型、空间概念、情绪基调、关键陈设。**产物四完整剧本中的所有场景必须从本清单中选择，禁止在剧本正文里自由新增未登记的场景**（如剧情确需新增，须先回填本清单并向用户声明再继续剧本创作）。
4. **完整剧本**（分集分场专业格式）：包含集数、场次、场景、时间/内外景、人物、动作提示（△）、对白、旁白（VO）、情绪节拍标记；长篇剧集需在开头向用户澄清"长篇幻觉声明"。

四段产物按顺序输出，每段之间用 `---` 分隔。短篇（≤10 集）四段一次性输出；长篇（>10 集）产物一（含分集大纲）、产物二、产物三一次性输出，产物四改为"分批输出、逐批确认"，详见"产物四：完整剧本"章节的"长篇分批输出规则"。

## 澄清策略

剧情设计或剧本设定开始前，必须确认或默认以下参数：

- 故事主题或核心梗概。
- 题材类型：都市逆袭、甜宠、搞笑、奇幻、科幻、历史、广告剧情、微电影、独立短片等。
- **单集总时长**：5-10 分钟；用户未指定时默认 6 分钟。
- **剧作基调偏好**（默认询问用户，二选一）:
  - **微电影三幕式**：开场钩子 → 事件发展/冲突 → 高潮反转 → 落点，聚焦单一核心事件、人物弧完整，适合大多数 5-10 分钟叙事。
  - **短视频钩子驱动**：3 秒强钩子 → 快速冲突堆叠 → 爽点集中释放，节奏更快、反转更密，适合都市逆袭、掉马甲、爽剧类。
- 目标受众与情绪价值：爽感、悬疑、治愈、搞笑、燃点、情感共鸣等。
- 主角身份与核心动机。
- 中心冲突与反派 / 对立力量。
- 情绪基调与视觉方向。
- 结局方向：开放、反转、惩罚、留白或行动号召。
- 禁用内容或敏感边界。

如果用户只给模糊 idea，问 1-3 个最关键问题。用户要求"你来定"时，选择合理默认值并继续。

### 长篇幻觉风险声明（>10 集时强制前置）

如果澄清阶段判定用户诉求指向**长篇短剧**（多单元、几十集、跨集主线，且集数 >10 集），必须在正式进入产物生成之前完成一次幻觉风险声明并让用户确认，作为进入生成的门禁：

- 一次性向用户明确声明超过 10 集之后模型可能出现的**剧情漂移与人设漂移**风险（关键伏笔遗忘、角色动机前后不一、道具/场景连贯性失守）。
- 声明模板（可原样输出）：
  > 提示：您要求生成 N 集完整剧本。当集数超过 10 集时，模型可能出现剧情或人设漂移（例如伏笔遗忘、角色动机前后不一、关键道具/场景失守）。生成过程中如您发现某一集与前文剧透、人设或世界观出现矛盾，请随时指出，我会针对该处进行回填校正或整段重写。
- 主动给出兜底建议（供用户在澄清阶段就选择替代路径）：
  - 拆成独立短篇/单集样片单独下单；
  - 改用专门的长篇短剧生产工具；
  - 保留长篇但同意承担漂移风险，改由用户在生成过程中随时点位纠错。
- **同步告知长篇分批输出机制**：即使用户选择继续生成长篇，也不能一次性输出全部剧集，而是首批 3-5 集供用户查看确认后再继续下一批（细则见下文"长篇分批输出规则"）。
- 用户明确知晓并选择继续后，才可进入四段产物生成；长篇每集内部同样使用下文的分集/分场剧本格式，且必须在完整剧本正文最前面再复述一次声明模板。

## 输出要求

四段产物按下面结构输出。

### 产物一：剧情梗概（约 1500 字）

- 单一连贯叙事文本，不用小标题堆砌，也不用过多分点。允许用 3-5 个自然段落对应"起承转合"。
- 覆盖：故事起点与主角初始处境、核心冲突登场、事件升级与转折、高潮反转、落点/情感闭环。
- 必须包含：主角核心动机、反派/对立面的具体阻挠手段、主角的关键抉择、结局形态（爽感释放 / 情感和解 / 反转揭示 / 行动号召）。
- 短篇 5-10 分钟：1500 字左右完整覆盖单集。
- 长篇多集：1500 字覆盖全剧主线（不逐集展开），另外配一份"分集大纲"表（每集一句话概括核心事件），供完整剧本阶段落地。

### 产物二：人物小传

至少覆盖主角、配角、反派三类角色，每位一段小传，段内使用以下颗粒度：

```markdown
#### [角色名]（[主角 / 配角 / 反派]）
- **一句话定位**：[年龄 + 身份 + 关键人设标签]。
- **性格**：[3-5 个具体性格特质，避免抽象形容词，用行为倾向描述]。
- **动机**：[核心动机 + 底层驱动力（欲望/恐惧/执念）]。
- **成长弧线**：[开场状态 → 中段动摇/受挫 → 结局蜕变，写清转折触发事件]。
- **视觉方向**：[大方向的形象、气质与识别符号，细节由资产阶段落地]。
- **关键道具/记忆点**：[标志性物件或行为习惯]。
```
反派小传特别要求：
- 必须写清"合理性动机"（反派为什么这样做，避免脸谱化）。
- 必须写清"惩罚落点"（反派在剧中如何得到实质惩罚或明确失败）。

### 产物三：主要故事场景

在动笔完整剧本之前，先把全剧反复出现的核心场景一次性锁定，作为产物四完整剧本场次表头 `场景名 [日/夜] [内/外景]` 的取用来源。

**数量约束：**

- **短篇（≤10 集，5-10 分钟单集）**：核心场景控制在 **2-4 个**，跨场景切换过多会显著抬升资产成本与视频生成风险。
- **长篇（>10 集）**：核心场景控制在 **4-8 个**，允许每几集出现 1-2 个次要场景，但仍需在本清单中提前登记。

**输出格式：**

```markdown
| 场景编号 | 场景名称 | 场景类型/功能 | 空间概念方向 | 情绪氛围基调 | 叙事性关键陈设 | 常见时间/内外景 |
|---|---|---|---|---|---|---|
| S01 |  | [主场景/次场景/回忆场景/梦境] |  |  |  | [日 内 / 夜 外 / …] |
| S02 |  |  |  |  |  |  |
```

**字段颗粒度：**

- **场景名称**：2-6 字为宜，需与后续剧本正文场次表头一字不差（如 `茶室`、`旧公寓客厅`、`天台`）。
- **场景类型/功能**：主场景（反复出现的日常/工作/家庭空间）/ 次场景（推动关键剧情的少量出现空间）/ 回忆或梦境（视觉上与现实形成区隔的特殊空间）。
- **空间概念方向**：只写大方向（材质、体量、光源、时代感），细节由资产阶段落地。
- **情绪氛围基调**：本场景默认承载的情绪（压抑/温暖/悬疑/冷冽/怀旧等）；同一场景可承载不同情绪，但需列出主基调。
- **叙事性关键陈设**：与关键道具、伏笔、剧情反转有直接关系的物件（如"书架下层的深棕色铁盒"、"茶室长桌上的落款签"）。
- **常见时间/内外景**：给出该场景在剧中最常出现的 `日/夜 + 内/外景` 组合，便于剧本正文表头对齐。

**产物四完整剧本的场景取用约束（强制）：**

- 完整剧本每一场戏的表头 `场次编号 场景名 [日/夜] [内/外景]` 中的 `场景名`，**必须严格从本清单的 `场景名称` 中选择**，禁止在剧本正文里凭空新增未登记的场景。
- 如果剧情确需新增场景，必须先暂停剧本创作，回填本清单（新增一行 `S0N`），并向用户简要说明新增场景的必要性；用户确认后再继续剧本创作。
- 场景登记不足时（如短篇只登记 1 个场景就动笔完整剧本），必须先回到本产物补齐 2-4 个场景再进入剧本创作。

**样例：**

```markdown
| 场景编号 | 场景名称 | 场景类型/功能 | 空间概念方向 | 情绪氛围基调 | 叙事性关键陈设 | 常见时间/内外景 |
|---|---|---|---|---|---|---|
| S01 | 旧公寓客厅 | 主场景（父亲遗物空间） | 昭和风老式公寓，深色木地板+落地书架+百叶窗斜光 | 压抑、怀旧 | 书架下层的深棕色铁盒、父亲旧照片、《平凡的世界》 | 日 内 |
| S02 | 旧公寓天台 | 次场景（结局落点空间） | 老式楼顶天台，水泥地+晾衣绳+夕阳 | 释怀、温暖 | 修好的旧怀表、姑姑的旧布袋 | 傍晚 外 |
| S03 | 都市写字楼工位 | 主场景（主角日常压抑空间） | 现代 open-space 工位，冷白日光灯+电脑屏 | 冷冽、疏离 | 电脑上的姑姑来电提醒、办公桌抽屉里的父亲遗信复印件 | 日 内 |
| S04 | 老家餐桌回忆片段 | 回忆场景 | 昏黄暖光下的方桌，家常菜+旧电视机 | 温暖、遗憾 | 那本《平凡的世界》摆在饭桌一角 | 夜 内 |
```

### 产物四：完整剧本（分集分场专业格式）

按下节"专业剧本格式规范"输出。长篇（>10 集）必须在剧本正文最前面加一次"长篇幻觉声明"（模板见"边界"章节）。

**场景取用硬约束**：完整剧本每一场戏的场次表头 `场景名`，必须严格从产物三"主要故事场景"清单中选取；如剧情确需新增场景，先回填产物三再继续。

#### 长篇分批输出规则（>10 集时强制，短篇不适用）

一次性长文本极易在中后段出现漂移，因此长篇产物四采用**分批输出、逐批确认**：

- **每批 3-5 集**（单集 3 分钟以内可 5 集/批，单集 5 分钟以上建议 3 集/批）；首批必须在剧本头之后立即复述"长篇幻觉声明"，后续每批开头以 `## 第 N 批：第 X 集 - 第 Y 集` 标注区间，并简要复述"上一批已发生的关键事件与埋下的伏笔"。
- **每批之间必须停下等确认**：输出后主动询问用户对人设稳定性、伏笔清晰度、节奏基调的意见，收到"继续/下一批/ok"等确认表达后才可进入下一批；若用户指出漂移，先针对该集回填校正或整段重写，再次确认后再继续。
- **伏笔标注**：每批结尾如遇关键伏笔尚未回收，明确标注 `【伏笔待回收：XXX（预计第 X 集）】`。

**输出格式示例**：

```markdown
## 产物四：完整剧本（第 1 批：第 1 集 - 第 3 集）

《XX 短剧》
类型：长篇短剧（共 20 集）
...（剧本头）

> 提示：您要求生成 20 集完整剧本。当集数超过 10 集时，模型可能出现剧情或人设漂移...（复述声明）

第一集
...

第二集
...

第三集
...

---
**首批交付说明**：
以上是第 1 批（第 1-3 集）的完整剧本。请您查看：
1. 主角/反派的人设是否与人物小传一致？
2. 首集钩子是否够强？前 3 集节奏是否合适？
3. 已埋下的伏笔（如 XXX、YYY）是否清晰？
4. 场景是否都取自产物三"主要故事场景"清单？是否需要新增场景？
5. 是否需要针对某一集进行调整或重写？

您确认无误后，回复"继续"或"下一批"，我会输出第 2 批（第 4-7 集）。
```

## 专业剧本格式规范

完整剧本必须严格采用**分集分场专业剧本格式**，让下游分镜阶段能一场戏对一段镜头列表。**不建议再使用"章节 + 100 字梗概"这类概要式写法作为完整剧本的最终形态**——概要式内容归在"剧情梗概"产物里。

### 剧本头（每部剧本开头一次）

```markdown
《[剧本标题]》
类型：[短篇短剧 / 微电影 / 剧情短片 / 动画短片 / 广告剧情 / 长篇短剧]（如为长篇，注明总集数）
风格：[王家卫风格的电影 / 是枝裕和风格的电影 / 3d玄幻 / 2d水墨 等]
标签：[题材关键词 + 情感关键词，例如"豪门联姻 + 先婚后爱 + 救赎"]
人设：[一句话高浓缩人设对撞，例如"年轻洒脱活泼女×年成熟稳重古板男"]
单集时长：[X 分钟]  |  剧作基调：[微电影三幕式 / 短视频钩子驱动]
```
长篇短剧在剧本头之后立即输出一次"长篇幻觉声明"。

### 风格字段填写指引与真人电影大师风格参考库

**风格路径判定与写法**：`风格` 字段先判断走"真人电影"还是"非写实类"，再决定具体锚点。

- **切换到"非真人"仅两条强信号（命中任一即可）**：①用户直接指定画面制作形式（3D 动画/2D 动画/水墨/国漫/日漫赛璐璐/绘本/MG 动画/皮克斯/毛毡定格/粘土动画等）；②内容本身无法用真人拍摄（非人形主角、极端抽象、低龄童话绘本形态）。**题材本身（仙侠/玄幻/修真/科幻/末世/赛博朋克/古风）不构成非真人信号**——《仙剑奇侠传》《三体》《长安十二时辰》《银翼杀手》皆为真人。仅凭题材切换到 3D/2D 是错误判断。
- **无非真人强信号时默认走真人电影**，风格字段直接写"**某某风格的电影**"（如 `王家卫风格的电影`、`是枝裕和风格的电影`、`大卫·芬奇风格的电影`）。须主动从下文六大流派中挑选 1 位大师作为锚点，禁止留白写"影视级布光""电影风格""高级质感"等模糊表述，否则会导致下游资产阶段与关键帧阶段风格漂移。若用户点名要求某位大师风格，一字不差写清并在后续资产、关键帧、prompt 阶段全程沿用；若一位大师有多个明显不同风格阶段，可追加代表作限定（如 `王家卫《花样年华》风格的电影`）。
- **模糊时主动澄清**：题材/形式无法从其它信号推断时（如"短篇玄幻广告"既可真人也可 3D 动画），澄清阶段主动回问画面形式偏好，不建议私自决定；实在无法推断（如"随便写个都市短剧"）默认取 `是枝裕和风格的电影` 作为最保守锚点。
- **走非真人路径时**：风格字段不套用真人电影写法，使用"京都动画风格""吉卜力风格""上美影水墨风格"等**动画流派**作为风格锚点。

**真人电影六大流派参考库**（按澄清阶段确认的题材/情绪基调选择 1 位主锚点大师）：

#### 一、东方生活流 / 家庭治愈（对应微电影 B1-B2 情感共鸣型 / 治愈自愈型）
- **是枝裕和**：家庭日常、柔和自然光、餐桌戏、克制的情感高潮。
- **李安（早期华语时期）**：东方伦理细腻笔触、温润暖色、克制的情感张力。
- **侯孝贤**：长镜头 + 自然光 + 生活流、台湾新电影气质、少切镜多定机。
- **岩井俊二**：青春过曝、逆光空气感、蓝白清冷色、手持轻晃。

#### 二、港风霓虹 / 怀旧胶片（对应微电影 B4 怀旧色调 · 复古胶片美学）

### 分集与分场格式

- **分集标题**：`第X集`，独占一行，不带集名（可在剧情梗概里给集名，剧本正文用集号即可）。
- **场次编号**：格式为 `X-Y`，X 为集号、Y 为该集内场次序号（从 1 起）。
- **场景表头**：`场次编号 场景名 [日/夜] [内/外景]`，例如 `1-1 茶室 日 内`、`1-2 茶室门口 日 外/内`。**`场景名` 必须严格从产物三"主要故事场景"清单的 `场景名称` 中选取，与其一字不差**；剧情如需新增未登记场景，先回填产物三再继续。
- **人物**：单独一行，列出该场出现的人物；如有画外音使用"角色名+VO"形式，如 `白父VO`。
- **动作提示**：以三角形符号 `△` 起首，描述人物动作、场面调度、道具状态、镜头暗示。**动作提示必须客观、可拍摄，不写心理活动、不写情绪形容词**（情绪需通过动作外化）。
- **对白**：`角色名: 台词内容`；括号内可加简短情绪/动作提示，如 `白清枞:（失笑）盲婚哑嫁，你也能接受？`；台词若为特写、内心独白等特殊呈现，用 `（特写）` 或 `（内心）` 前置。
- **旁白/画外音**：`角色名VO: 内容`。
- **情绪节拍标记**：允许在关键台词或动作提示末尾用 `【节拍：XXX】` 标记情绪节拍（如 `【节拍：破冰】`、`【节拍：反转】`、`【节拍：爽点释放】`），供分镜阶段判断镜头节奏。
- **场次结束**：空一行进入下一场次。

### 分集分场格式样例（微电影向）

微电影通常单集独立、以情感沉淀取胜，样例展示"留白 + VO 克制 + 象征化对立面"的典型写法。

```markdown
《最后一封信》
类型：微电影（1 集）
风格：都市真人
标签：都市治愈 + 父子和解 + 遗物开启
人设：情绪克制的都市白领儿子×一辈子沉默的老式父亲
单集时长：6 分钟  |  剧作基调：微电影三幕式

第一集

1-1 旧公寓门口 日 外
人物：林远
△（远景）林远背对镜头站在旧公寓门口，右手推开吱呀作响的木门，尘埃在门缝斜光中缓缓浮动。
△林远停在门框中央三秒，没有立即迈步进去。
林远VO：父亲走后的第七天，我第一次踏进这里。【节拍：开场钩子】

1-2 旧公寓客厅 日 内
人物：林远
△林远缓缓走进客厅，视线扫过书架上父亲的旧照片，眉头轻蹙。
△（近景）林远走到书架前，右手抬起，指尖轻抚玻璃相框，停在父亲肩膀上。
△（特写）指尖在玻璃相框上停留三秒后缓慢滑落，指腹沾了一层薄灰。

1-3 旧公寓客厅 日 内
人物：林远
△（林远 POV，俯拍）书架下层，前景是林远的左手，一个深棕色四角磨损的铁盒静置层板中央。
△（第三人称切回，大特写）铁盒表面缓缓浮出微小光斑，从盒内传出规律的滴答声。【节拍：悬念揭示】

1-4 旧公寓客厅 日 内
人物：林远
△林远从书架抽出那本《平凡的世界》，翻开书页——一枚小铜钥匙从书脊夹层滑落到桌面。
△林远盯着桌面的钥匙看了很久，没有说话。
△（特写）钥匙在斜光下有细微反光，桌面浮尘缓缓沉落。

1-5 旧公寓天台 傍晚 外
人物：林远、姑姑
△林远手里握着修好的旧怀表，站在天台边缘。夕阳把两人剪影拉长。
△姑姑站在他身后三步远的位置，没有说话，从旧布袋里掏出那本翻旧的《平凡的世界》。
姑姑：（低声）你爸让我把这个交给你。
△（特写）林远缓缓打开怀表，按下侧面按钮。
怀表VO：（沙哑）……对不起。
△林远蹲下身，肩膀开始颤抖。姑姑没有走过去，只是站在原地。
林远VO：三年了，我第一次听见他这么小声地说话。【节拍：情感闭环】
```

> 微电影样例说明：动作提示允许"停三秒""指尖缓慢滑落""浮尘缓缓沉落"这类留白性描述；VO 只出现在开场与结尾，中段用画面 + 环境音承担；对立面（父亲的沉默）以"怀表 VO + 未寄出的书本"作为象征化道具登场，不需要人格化反派登台。短剧向的丰满对话样例（打脸/亮相/破冰节拍密集）见文末"完整输出样例参考（长篇短剧）"章节。

### 单集内部结构与章节层继承

单集完整剧本仍需遵循下文"单集章节结构"要求：3-5 个叙事章节。**章节层不写在剧本正文里**，而是通过场次的排列顺序自然承担叙事功能，供 `storyboard.md` 阶段读取时切分。可以在剧情梗概或分集大纲中标注"第一章 场次 1-1 至 1-3 / 第二章 场次 1-4 至 1-6..."的映射说明，方便后续分镜切分对齐。

### 长篇分集大纲附表

长篇短剧在剧情梗概之后、完整剧本之前，必须附一份分集大纲表：

```markdown
| 集号 | 集名 | 核心事件 | 情绪节拍 | 关键伏笔/回收 |
|---|---|---|---|---|
| 第1集 |  |  | 开场钩子 |  |
| 第2集 |  |  | 冲突展开 |  |
| ... |  |  |  |  |
```

分集大纲的作用是给完整剧本的每一集提供锚点，降低超过 10 集之后的漂移风险；用户发现漂移时，可用大纲表定位应回填的伏笔。

## 单集章节结构

短篇单集必须切分为 **3-5 个叙事章节**，每个章节承担一个明确的叙事功能。章节数量与总时长的推荐配比：

| 单集总时长 | 章节数 | 单章节时长 | 每章节 15s 片段数 |
|---|---|---|---|
| 5 分钟 | 3 章节 | 约 100 秒 | 6-7 段 |
| 6 分钟 | 4 章节 | 约 90 秒 | 6 段 |
| 8 分钟 | 4 章节 | 约 120 秒 | 8 段 |
| 10 分钟 | 5 章节 | 约 120 秒 | 8 段 |

**章节功能分配**（详细写作原则见下节 A1/B1）：

- **微电影三幕式**：铺陈（20-25%，世界观 + 开场钩子）→ 事件与冲突（40-50%，主角陷入困境）→ 高潮与落点（25-30%）；4-5 章节时把冲突拆成"展开 + 危机加剧"或加"破局准备"章节。
- **短视频钩子驱动**：3 秒钩子 + 铺垫（15-20%）→ 加剧压迫、主角蓄力（30%）→ 身份/能力暴露、首波打脸（30%）→ 终局清算、爽点释放（20-30%）。

## 写作原则与爆款法则

短剧（钩子驱动 / 爽剧）与微电影（三幕式 / 情感沉淀）的创作逻辑差异较大，下面分两类分别给出。**根据澄清阶段确认的"剧作基调偏好"选择对应法则；混合基调项目可交叉引用**。

### A. 短剧向法则（钩子驱动 / 爽感释放）

适用于：都市逆袭、掉马甲、豪门打脸、年代手撕极品、爽剧类。

#### A1. 结构与节奏
- **黄金开局定律**：短篇必须在前 15-30 秒内抛出核心矛盾或强悬念，让主角陷入危机或揭示极端处境。
- **高频反转与波折**：每章节至少一次反转；不建议出现超过 15 秒没有信息增量或情绪推进的镜头。
- **收束优先**：短篇结尾必须给出明确落点（惩罚、反转、行动号召、情感闭环），不建议用"待续/悬而未决"作为短篇收尾——那是长篇钩子写法。

#### A2. 爽感与情绪
- **压迫要具体，释放要痛快**：极致的压抑带来极致的释放。对手的阻挠必须有具体恶劣的行径，主角的反击必须是降维打击。
- **反差原则**：人设反差（弱小外壳+强大内核）、情节反差（预期用来打破）、台词反差（不说"该说的"，说"最解气的"）。
- **共鸣与痛点外化**：精准击中现代人的痛点（如职场霸凌、背叛），让主角以最痛快的方式予以回击。

#### A3. 人物与台词
- **立体主角与灰度配角**：赋予主角合理的执念或软肋，赋予反派行为逻辑的"灰度"。
- **用动作外化情绪，不靠解释性台词**：直接利用动作和视觉来传递信息（例如：不要说反派多坏，直接让他踩碎主角的救命药）。
- **台词千人千面**：台词匹配角色身份、教育背景和社会地位。
- **反派惩罚铁律**：短篇必须让反派在本集内得到实质惩罚或明确失败，不建议写成"跨集才收拾"。

### B. 微电影向法则（三幕式 / 情感沉淀 / 意境余韵）

适用于：都市治愈、亲情和解、独立成长、社会议题、怀旧片、艺术短片、品牌微电影、影展参赛短片。

#### B1. 结构与节奏
- **单一核心事件定律**：一部微电影只承载**一个核心事件 / 一次情绪转变 / 一个价值追问**，不建议塞多条主线。核心事件可以极小（一次告别、一封未寄出的信、一次深夜通话），但必须让主角在开合之间发生真实变化。
- **经典三幕式骨架（默认推荐）**：铺陈（约 25%，建立主角处境与情感缺口）→ 事件与冲突（约 50%，事件登场并把主角推到临界点，中段设 1 次翻转或撬动情绪的关键契机）→ 高潮与落点（约 25%，主角做出内在选择并完成情感闭环）。
- **允许留白与呼吸**：微电影可以有 2-3 秒的静态凝视、无对白空镜、环境音铺陈；这是与短剧最大的节奏差异。**不建议**因为担心用户"划走"而删掉留白，留白是微电影情绪沉淀的必需成本。
- **落点优先"情感闭环"而非"打脸释放"**：结尾可以是和解、释怀、觉悟、一次未说出口的告别、一次无声的目光对视；不追求"反派实质惩罚"，允许对立面（如"父亲的沉默""过往的自己"）以象征性/非人格化方式登场。

#### B2. 情感与意境
- **压抑不必具体到"作恶"**：微电影的对立面可以是"沉默""距离""时间流逝""社会规则""过去的自己"，不必人格化为一个坏人。冲突层级可以从"人 vs 人"降级为"人 vs 环境 / 人 vs 自我 / 人 vs 时间"。
- **情绪靠"物件 + 光影 + 留白"承载**：一封信、一块怀表、一张旧照片、傍晚的斜光、雨夜的路灯——这些"情感锚点"承担的比台词更多。写剧本时应优先给锚物件足够的镜头与陈设，而非塞更多解释性对白。
- **允许"淡"，不建议"平"**：淡 = 情绪浓度低但仍有暗流；平 = 没有情绪推进。淡是微电影的美学；平是失败的微电影。判断标准：某场戏删掉是否会让主角的情感转变链条断掉？如果不会，则说明这场戏太平。
- **意境优于爽点**：允许结尾不给"回答"，只给"追问"（如"你会怎么选？"、"如果那天他没走呢？"），把余韵留给观众。

#### B3. 人物与台词
- **主角内驱大于外驱**：短剧主角为"打脸/复仇/掉马甲"这些外部目标驱动；微电影主角更多为"和解/找回自己/放下"这些内驱目标驱动。写小传时优先写清"未和解的情绪 / 长期回避的问题 / 心里的一块石头"。
- **配角"少即是多"**：微电影主要角色数量建议控制在 **2-4 位**（主角 + 1-2 位关键他者 + 可选 1 位画外/回忆角色），不建议塞太多群像。每个角色都要为主角的内驱变化服务。
- **台词克制、去戏剧化**：微电影台词更接近生活对白，允许沉默、允许答非所问、允许一句话说一半。避免总结陈词式台词（如"我终于明白了……"），这些应由观众自己得出。
- **VO / 独白使用节制**：VO 是微电影的常用手法，但建议只出现在开场与结尾，中段用画面 + 环境音替代 VO；不建议整片用 VO 讲故事。
- **反派/对立面可以象征化**：允许用"父亲的旧照片""空掉的座位""褪色的门牌"来代表对立面，不必安排一个人格化反派。

#### B4. 视觉与听觉
- **光影是第一叙事语言**：微电影通常靠自然光、逆光、剪影、色温冷暖切换承担情绪转折；写剧本时可在 `△` 动作提示中给出光影意图（如"斜光扫过书架"、"夕阳把两人剪影拉长"），供分镜阶段落地。
- **色调风格建议**：治愈系→柔光+暖色低饱和；怀旧系→复古胶片+颗粒感；独立文艺→高级暗调+侧光；社会议题→纪实感+原生光。
- **音乐优先"氛围垫底"**：微电影音乐建议以钢琴、大提琴、氛围电子、极简弦乐为主，避免情绪激烈的鼓点；音乐建议只出现在铺陈铺开、事件登场、落点三个关键节点，中段允许纯环境音。

## 常见题材与受众心理参考库

### A. 短剧向 · 核心受众心理学分析与对应题材

短剧的核心爽感往往根植于深层的社会心理，可从**个人主义**与**集体主义**两个维度进行剖析：

#### A1. 个人主义爽感（打破规则、掌控命运、极致自我）
此类爽感的核心在于"我"的意志、能力和情感欲望的彻底释放。在欧美市场（海外短剧）尤为突出，国内亦有受众。

*   **心理痛点**：现代社会中个体受制于复杂的规则、缺乏对人生的掌控感、渴望被独一无二地偏爱。
*   **爽感机制**：主角拥有绝对的信息差或宿命般的羁绊，能够随心所欲地改写命运，或体验极致纯粹的情感。
*   **对应热门题材**：
    *   **重生/预知/无限流**：利用"信息差"降维打击，弥补遗憾。短篇里通常聚焦一次关键的"重来一遍反杀"事件。
    *   **穿越/异世界 (Isekai/Time Travel)**：主角利用现代文明的知识、科技或系统降临古代/魔法世界，享受智商和见识上的绝对优越感。
    *   **末世囤货/生存基建**：在秩序崩塌的末日世界中，主角凭借金手指疯狂囤积物资，享受绝对安全感。
    *   **狼人/天选伴侣（海外爆款）**：强调"唯一性"与"宿命感"，满足极度渴望被偏爱的个人情感。
    *   **吸血鬼/黑帮大佬（海外爆款）**：危险、迷人且拥有绝对权力的非传统男性角色与平凡女主的极限拉扯。
    *   **禁忌之恋/离婚觉醒**：对抗规则，追求真我。
    *   **系统/修仙升级流**：主角独自获得神秘系统或逆天功法，短篇通常聚焦一次关键突破。

#### A2. 集体主义爽感（阶层跃迁、打脸围观、重塑秩序）
此类爽感的核心在于"面子"、"地位"与"认同"，通过他人的震惊、跪拜或悔恨来完成。国内市场占据绝对统治地位。

*   **心理痛点**：现实生活中的阶级焦虑、被社会/家族/圈层轻视、集体荣誉受损。
*   **爽感机制**：主角隐忍不发，随后突然暴露极其显赫的身份或惊人财富，让所有轻视者付出代价。
*   **对应热门题材**：
    *   **掉马甲/隐世大佬（战神/龙王/首富）**：先抑后扬的"扇耳光"逻辑，短篇里适合聚焦一次核心亮身份事件。
    *   **真假千金/豪门上位**：通过血缘或联姻进入最高权力结构。
    *   **年代/四合院/年代知青**：在复杂的邻里关系和家族纠纷中斗倒"极品亲戚"。
    *   **婆媳伦理/手撕极品亲戚**：聚焦中国式家庭的结构性压迫。
    *   **萌宝/天才儿童**：单亲妈妈带着天才宝宝归来。
    *   **衣锦还乡/家族正名**：为受辱的父母报仇，让整个家族获得至高尊重。

#### A 段市场偏好补充
- **国内受众**（出海华人/下沉市场）：更吃"集体主义爽感"，安排"围观群众"和"势利眼配角"衬托主角。
- **海外受众**（欧美/拉美）：更吃"个人主义爽感"，强化男女主宿命拉扯、荷尔蒙张力与独立觉醒。

### B. 微电影向 · 核心受众心理学分析与对应题材

微电影的核心价值不是"爽感释放"，而是**情感共鸣 + 意境沉淀 + 价值追问**。受众看完之后带走的是"一段情绪 / 一个念头 / 一次自我审视"，而不是"痛快感"。

#### B1. 情感共鸣型（人际关系与情感和解）
此类微电影的核心在于让观众看见自己的影子。

*   **心理痛点**：现代人际疏离、亲密关系损耗、来不及说出口的话、日渐拉远的家人距离。
*   **共鸣机制**：把观众自身"未说出口的遗憾 / 迟到的道歉 / 想联系却没勇气拨出的电话"具象化为一次事件，让观众在主角身上完成一次"代偿"。
*   **对应热门题材**：
    *   **亲情和解 / 父子父女**：常见于"父亲遗物 / 母亲的病历 / 一次错过的葬礼"，主角借由一件旧物完成迟到的情感闭环。
    *   **异地/异国离别**：机场、火车站、跨年夜的视频通话；核心冲突是"距离侵蚀情感"。
    *   **老友重逢 / 十年再见**：借由一次意外重逢，回望青春或彼此走散的路径。
    *   **临终告别 / 阿尔茨海默题材**：以"记忆流失"或"时间不多了"作为情绪撬动器。
    *   **离婚 / 分手后的独处**：不写复仇，写"如何与自己重新相处"。

#### B2. 治愈自愈型（都市孤独与自我修复）
此类微电影不写与他人的关系，写主角与自己的关系。

*   **心理痛点**：都市青年的倦怠、抑郁边缘、对"意义感"的怀疑、被 KPI 与算法压得喘不过气。
*   **治愈机制**：让主角在一个平凡的日常事件中（一次深夜散步、一场落雨、一段陌生人对话）重新找回小确幸或活下去的理由。
*   **对应热门题材**：
    *   **深夜食堂 / 便利店哲学**：陌生人短暂交集带来的情感修复。
    *   **辞职返乡 / GAP 一年**：主角短暂逃离都市，在乡野/海边/雪山重新充电。
    *   **一日限定关系**：一次共伞、一次借宿、一次接力照顾走丢的宠物；短暂而完整的温暖。
    *   **独居美学 / 老年独居**：把独处过成生活美学，反焦虑叙事。
    *   **宠物告别 / 猫狗视角**：以宠物寿命为镜子，让主角看见自己被治愈的过程。

#### B3. 社会议题型（现实洞察与价值追问）
此类微电影承担一定的社会观察责任，常见于影展、公益短片、独立创作。

*   **心理触点**：观众对某种社会现象既熟悉又无力表达，微电影替他们"说出来"。
*   **表达机制**：不给答案，只呈现一段生活切片；结尾允许开放式追问。
*   **对应热门题材**：
    *   **打工人题材**：外卖员、网约车司机、深夜清洁工、程序员猝死边缘、35 岁裁员。
    *   **教育焦虑 / 鸡娃**：家长与孩子在补习班门口的一次对话。
    *   **老龄化 / 空巢老人**：一部无人接听的电话、一顿只为一个人做的年夜饭。
    *   **性别议题**：职场性骚扰、单亲妈妈、彩礼争议、家暴幸存者。
    *   **原生家庭 / 代际创伤**：一段没有终点的和解。
    *   **AI 与人性**：机器人陪伴的老人、被算法推送困住的青年、被大模型替代的岗位。

#### B4. 品牌 / 商业微电影型（品牌植入 + 情感叙事）
此类微电影服务于品牌传播，需要在情感故事中自然嵌入品牌价值观。

*   **执行要点**：品牌应作为"情感转折的助手"出现（一杯咖啡 / 一次快递 / 一个 app 提醒），**不建议**让品牌直接出面说教。落点必须与品牌价值观自然对齐，不建议在结尾加广告 slogan。
*   **常见方向**：
    *   **年货 / 春节回家系列**：交通、通讯、饮品、家电品牌常用方向。
    *   **母亲节 / 父亲节 / 感恩节系列**：食品、化妆品、家居品牌常用。
    *   **职场女性 / 独立女性系列**：美妆、服饰、金融品牌常用。
    *   **公益纪录短片**：教育、医疗、扶贫品牌常用，注重真实感。

#### B 段市场偏好补充
- **国内微电影受众**（社交媒体/短视频平台/公众号推广）：更吃"亲情和解 / 城市青年治愈 / 社会议题共鸣"三条主线；落点建议朝"共情、共鸣、共叹"三个方向收束，而非制造话题冲突。
- **海外微电影受众**（影展/短片平台/YouTube 独立创作）：更吃"独立视角 + 社会洞察 + 视听语言"，允许更极致的作者表达、更慢的节奏、更含蓄的落点；开放式结局耐受度远高于国内。
- **品牌微电影地域差异**：国内注重"节点营销契合度"（春节、母亲节、双十一），海外注重"品牌价值观陈述"（可持续、多样性、社区）。
- **影展参赛**：注重原创性、社会议题厚度、视听语言完整度；单集时长建议 8-15 分钟（部分影展接受 5-10 分钟），需符合具体影展征片规则。

## 产物二：人物小传

#### 林远（主角）
- **一句话定位**：32 岁都市白领，父亲刚刚过世的独生子。
- **性格**：情绪克制、习惯性回避、责任感强、外冷内软、对细节敏感。
- **动机**：完成父亲遗物整理这一"最后的义务"，以此彻底翻篇——却在过程中被迫直面从未和解的亲子关系。
- **成长弧线**：开场机械化处理遗物 → 中段被铁盒滴答声与旧照片撬动情绪 → 结局戴上怀表并允许自己哭出来，完成迟到的和解。
- **视觉方向**：干净利落的都市白领形象，深色针织衫 + 简洁风衣，眼神克制、下颔线绷紧。
- **关键道具/记忆点**：父亲遗信、修好的旧怀表。

#### 姑姑（配角）
- **一句话定位**：50 岁父亲的姐姐，家族中唯一还与林远保持联系的长辈。
- **性格**：唠叨、直白、心直口快、护短、爱哭。
- **动机**：替沉默一辈子的弟弟把话说给林远听，让侄子不要重蹈覆辙。
- **成长弧线**：开场电话催促 → 中段来公寓帮忙整理时旁敲侧击 → 结局在天台下静静等他哭完，什么都没多说。
- **视觉方向**：微胖，短发烫过，穿深色针织外套，永远拎着一个旧布袋。
- **关键道具/记忆点**：布袋里那本被翻旧的《平凡的世界》。

#### 记忆中的父亲（反派/对立面：以"父亲的沉默"作为对立力量）
- **一句话定位**：60 岁老式父亲，早年丧妻独自抚养林远，去世时刚满 60。
- **性格**：寡言、要强、情感表达笨拙、控制欲强、藏事儿。
- **动机**：他不是坏人，但他用"沉默"逼迫儿子按自己的方式生活，是对立力量本身。
- **合理性动机**：他用一辈子扛下所有情绪，是因为他相信"男人不该把软弱说出口"，这种价值观最终把父子关系推向深渊。
- **惩罚落点**：他没能亲口说出"对不起"，只能借助怀表内的一段短短录音在死后被儿子听见——他的沉默在剧末被儿子的一声哭泣彻底"打脸"。
- **成长弧线**：只出现在回忆和信件中；从"寡言父亲"→ "写下从未寄出的信" → "在怀表里录下最后一句对不起"。
- **视觉方向**：只出现在旧照片、回忆片段和天台夕阳的画外音里。
- **关键道具/记忆点**：修好的旧怀表、未寄出的信、书脊夹层的钥匙。

## 产物二：人物小传

#### 白清枞（主角）
- **一句话定位**：22 岁豪门二小姐，退役特种兵出身、军牌不离身的洒脱活泼女。
- **性格**：外壳散漫爱迟到、说话带刺、见谁都插兜；内核忠诚仗义、遇事第一反应挡在人前、对亲近之人极度护短。
- **动机**：不愿被家族当作联姻筹码，想在结婚前逼周晟安暴露真实底牌；底层驱动是对父亲将其工具化的长期愤怒与自我价值证明。
- **成长弧线**：开场用叛逆姿态试探男主 → 中段被雨夜救援与腕伤真相触动、开始真正参与这段婚姻 → 结局主动坦白军牌来历，完成从"被安排的人"到"共同选择的人"的转变。
- **视觉方向**：休闲西服 + 内搭白 T + 战术腰带、马尾、素颜、军牌吊坠贴身佩戴。
- **关键道具/记忆点**：脖颈军牌吊坠（与男主手腕伤疤同源）、习惯性敲桌、走路插兜。

#### 周晟安（男主 / 关键他者）
- **一句话定位**：32 岁跨国集团继承人,曾服役特种部队、手腕留旧伤疤的成熟稳重古板男。
- **性格**：极度守时、对白话不多、以规则与克制回应挑衅；对底线之内的事有近乎冷酷的耐心,对底线之外则一秒翻脸。
- **动机**：以联姻为名接近白家,实为回收多年前军中恩情;底层驱动是对救命恩人（女主父亲的战友,也是女主军牌来源）的旧诺。
- **成长弧线**：开场以"任务"心态坐在联姻桌前 → 中段在雨夜救援中露出军人本能、被女主看穿身份 → 结局承认自己不是来完成任务、而是来还一个未寄出的"谢谢"。
- **视觉方向**：深色三件套西装、银色腕表、发型一丝不苟、右手腕内侧一道旧疤。
- **关键道具/记忆点**：银色腕表(每场戏都会抬手看时间)、右手腕旧伤疤(第 10 集揭示是当年替女主父亲挡子弹留下)。

#### 白远山（反派 / 对立面）
- **一句话定位**：55 岁白氏集团掌门人、把女儿视作联姻筹码的功利型父亲。
- **性格**：表面儒雅、席上敬酒滴水不漏；私下强推流程、对女儿的意愿零耐心、对下人从不抬眼。
- **动机（合理性）**：白氏正处于被并购危机，联姻是他判断唯一能保住白家的路径；他相信"父亲有权替女儿决定命运"。
- **惩罚落点**：终局董事会上,女主当众公开白远山早年隐瞒军中战友遗孤(即男主)的旧账,他被董事会架空、失去实控权。
- **视觉方向**：深灰订制西装、金丝眼镜、右手无名指旧戒指(其亡妻遗物,反衬他的功利)。
- **关键道具/记忆点**：白家餐桌上的红酒杯（每次施压必先斟酒）、订婚文件夹（每集反复推到女主面前）。
