# 分镜切分 Ref

当已有完整故事并由用户确认/由用户提供，需要拆分为**章节 → 片段 → 镜头**三层结构时读取本文件。

## 承接边界重申

本 skill 主承接单集 5-10 分钟以内的短篇短剧、微电影、剧情短片、动画短片、宣传片、预告片、广告剧情、单条剧情短视频；也可承接多集长篇短剧（长篇需先在 `scriptwriter.md` 阶段完成"长篇幻觉声明"与分集大纲）。分镜阶段以"每集为单位"独立切分：短篇直接对该集切分，长篇按当前处理集单独切分并记录集号。

## 输入要求

需要具备：

- 当前处理集的完整剧本（分集分场专业格式，来自 `scriptwriter.md` 产物四）。
- 单集内叙事章节切分（3-5 个），可从 `scriptwriter.md` 产物一（剧情梗概 + 分集大纲附表）中的"第一章 场次 X-1 至 X-3 / 第二章 场次 X-4 至 X-6..."映射说明中读取。
- 人物小传中的主角、配角、反派清单（来自 `scriptwriter.md` 产物二）。
- 主要故事场景清单（来自 `scriptwriter.md` 产物三），完整剧本场次表头 `场景名` 可直接与该清单交叉验证；关键道具与场次表头 `[日/夜] [内/外景]` 也从此获取。
- 项目风格和画幅比例，如已确定。

如果剧本四段产物任何一段缺失，返回 `scriptwriter.md`。

## 澄清策略

写分镜前确认或默认：

- 单集总时长：默认 6 分钟，可选 5-10 分钟。
- 章节数量：3-5 个，与 `scriptwriter.md` 已确认的章节结构对齐。
- 每章节 15s 片段数量（默认单位）：按章节时长 / 15s 换算，向下取整（例如 90 秒章节 = 6 段，120 秒章节 = 8 段）。
- **单段视频时长开关（当前锁定 15s）**：本 skill 承接的下游生视频模型目前仅支持 15s，因此当前所有分镜片段**统一按 15s 切分**，无需向用户澄清。30s 分支作为**项目侧预留全局开关**保留在下文"分镜节奏速查表""输出规则""30s 版本合并说明"中；未来生视频模型支持 30s 后，由项目侧一次性把开关切换到 30s，届时预留分支将自动生效，无需用户逐次确认。若用户主动明确要求 30s 或 15s + 30s 双版本，才按其指定切分。
- 镜头密度按剧作基调分档（详见下文"分镜节奏速查表"）：**短剧向**每个 15s 片段 6-10 个镜头（钩子/打脸/高潮场景可上探到 10-12 个）、30s 片段 12-20 个镜头；**微电影向**每个 15s 片段 3-5 个镜头（允许 1 个 5-8 秒呼吸镜头）、30s 片段 6-10 个镜头（允许 1-2 个 8-12 秒长镜头）。
- 台词、旁白、字幕、音效、音乐要求。
- 是否优先使用低风险连续动作以适配 AI 视频生成。

## 分镜规划规范

- **章节继承**：单集章节结构由 `scriptwriter.md` 阶段确认，分镜阶段直接继承，不重新设计章节数量或叙事功能。
- **章节内片段切分**：每个章节内部按 15s（当前默认，全局锁定）连续切分，片段时间戳在章节内首尾相连；30s 分支作为项目侧预留开关保留,未来生视频模型支持 30s 后由项目侧统一切换。
- **章节间时间戳连续**：整集时间戳从 `00:00` 开始，跨章节仍然连续，不因为章节切换而重置。
- **双版本要求**：如果需要同时输出 15s + 30s 两个版本，必须在同一分镜表中先按 15s 完整切分，然后额外给出 30s 版本的合并规则（通常是相邻 2 个 15s 片段合并为 1 个 30s 片段），确保两版本共享同一批镜头信息。
- **独立短篇 / 微电影 / 广告剧情 / 单条短视频**：直接按上述章节+片段规则切分，不再套用长篇的"大单元/小高潮"节奏。

## 输出规则

- 默认按当前批次输出用户指定的章节；未指定时默认输出全部章节。
- 每章节按其规划的时长切分，切分单位当前**全局锁定 15 秒**；30s 与双版本分支保留为项目侧预留开关，未来生视频模型支持 30s 后由项目侧一次性切换,或用户主动指定时启用。
- 镜头密度需与剧作基调对齐（详见下文"分镜节奏速查表"）：**短剧向**每个 15s 片段 6-10 个镜头（钩子/打脸/高潮场景可上探到 10-12 个）、30s 片段 12-20 个镜头；**微电影向**每个 15s 片段 3-5 个镜头（允许出现 1 个 5-8 秒的呼吸镜头）、30s 片段 6-10 个镜头（允许出现 1-2 个 8-12 秒的长镜头）。
- 每个镜头必须包含**绝对时间戳（从整集 00:00 起算）**、景别、视角类型（第三人称客观视角 / POV 第一人称视角 / POV 与第三视角切换）、机位运动、拍摄角度与主体朝向（正面/侧面/背面等）、画面动作、台词、旁白、音效、音乐、字幕和转场中适用的信息。**时间戳必须首尾相连，确保片段总时长严格等于 15 秒或 30 秒**。

## 分镜节奏速查表

**分镜密度与镜头节奏须与 `scriptwriter.md` 澄清阶段确认的"剧作基调偏好"对齐**：短剧向（短视频钩子驱动）以"极快"和"快"为主，参考行业惯例——抖音爆款关键帧间隔约 1.8 秒、快手每 3.2 秒新的视觉刺激点、"暴力剪刀流"单镜头控制在 2-3 秒以内、前 15 秒平均每 2 秒 1 镜；微电影向（三幕式）允许"中速"与"慢速"，并可安排 5-8 秒静态凝视 / 空镜 / 环境镜头。**同一片段内可混合使用，但应避免长时间停留在单一节奏档位**。

### A. 短剧向(钩子驱动)节奏

| 单个镜头时长 | 节奏 | 对应单段镜头数(15s / 30s) | 典型剧情场景 | 表达情绪 |
|---|---|---|---|---|
| **1-1.5秒** | 极快(暴力剪刀流) | 10-12 / 20-25 | 钩子开场、冲突爆发、动作连击、打脸瞬间、身份揭示、高潮清算 | 窒息、爽点密集、信息轰炸 |
| **1.5-2.5秒** | 快(短剧常用主档) | 6-9 / 12-18 | 对话交锋、情绪递进、常规叙事、伏笔铺陈 | 急促、悬念叠加、节奏紧凑 |
| **2.5-4秒** | 中速(短剧下限) | 4-6 / 8-12 | 关键台词特写、爆点前的呼吸、反转停顿 | 蓄势、震撼、决定性瞬间 |

**短剧向下限硬约束**：15s 片段总镜头数**不得少于 6 个**（对齐"每 2 秒 1 镜"的行业底线；单镜头平均时长 >2.5 秒会显著降低留存），高潮/打脸/开场钩子建议 8-12 个；30s 片段总镜头数不得少于 12 个。若某段实际低于该下限，须回到 `scriptwriter.md` 检查是否误取了微电影向节奏，或该段本应作为微电影向"呼吸段"处理。

### B. 微电影向（三幕式 / 情感沉淀）节奏

微电影的分镜密度整体低于短剧。**允许"呼吸镜头"存在**——2-3 秒的静态凝视、无对白空镜、环境镜头是微电影情绪沉淀的必需成本，不建议因为担心用户"划走"而删掉。

| 单个镜头时长 | 节奏 | 对应单段镜头数（15s / 30s） | 典型剧情场景 | 表达情绪 |
|---|---|---|---|---|
| **3-5秒** | 中速（微电影常用档位） | 3-4 / 6-10 | 日常互动、情感铺陈、对白交流 | 平稳、含蓄、情绪暗涌 |
| **5-8秒** | 慢速 | 2-3 / 4-6 | 静态凝视、环境空镜、光影转折、关键台词沉淀 | 意境、留白、余韵 |
| **8-12秒** | 极慢 / 长镜头 | 1-2 / 2-4 | 长凝视、无对白独角戏、大远景空镜、结尾落点 | 沉浸、克制、追问 |

**微电影分镜密度建议**：15s 片段总镜头数控制在 **3-5 个**（短剧通常 6-10 个，高潮段可上探 10-12 个），30s 片段控制在 **6-10 个**（短剧通常 12-20 个）；关键情绪节点可安排 1 个 5-8 秒的"呼吸镜头"作为情绪撬动器。

## 镜头语言与运镜规范

在编写分镜时，必须准确使用影视级的景别和运镜术语，以精确控制观众的情感代入和叙事焦点。

### 1. 景别控制

景别决定了观众与主体的距离，直接影响情绪：

| 术语 | 定义 | 核心功能与情绪指向 |
|---|---|---|
| **大特写** | 只展现人物面部的局部，如眼睛、嘴唇。 | 强调微表情，放大极致情绪（爱、恨、惊恐）。 |
| **特写** | 展现人物从头肩的部分。 | 聚焦面部表情，传递强烈情绪，拉近心理距离。 |
| **近景** | 展现人物胸部以上。 | 既能看清表情，又能兼顾部分肢体语言。 |
| **中景** | 展现人物膝盖或腰部以上。 | 常用于对话场景，展现人物间的互动关系。 |
| **全景** | 完整展现人物全身及其周围小范围环境。 | 交代人物的完整动态与环境的直接互动。 |
| **远景** | 人物在画面中占比极小，环境成为主体。 | 强调环境氛围，表现人物的渺小、孤独或宏大叙事。 |
| **大远景** | 人物在画面中几乎成为一个点。 | 极致地渲染环境，常用于史诗感开场或结尾。 |
| **过肩镜头** | 从一个角色的肩膀后方拍摄另一个角色。 | 增强对话的代入感，让观众感觉身临其境。 |
| **第三人称客观视角** | 从角色外部观察人物动作、空间关系和场面调度。 | 交代人物站位、动作结果、环境关系，适合多数叙事镜头。 |
| **POV 第一人称视角** | 模拟某个角色眼睛看到的画面，镜头即该角色的主观视线。 | 提供沉浸式体验，强化压迫、惊吓、亲密互动、追逐、观察或手部操作。 |
| **POV 与第三视角切换** | 同一片段内在主观视线和客观外部镜头之间切换。 | 先用 POV 建立代入感，再用第三人称交代动作结果或空间关系，增强节奏变化。 |

### 1.1 POV 与第三视角切换规则

- 分镜可以采用 `POV 第一人称视角` 与 `第三人称客观视角` 来回切换的形式编排。
- 使用 POV 时必须明确 POV 所属角色，例如"林远 POV 第一人称视角"，避免只写"第一人称"。
- POV 镜头应从该角色眼睛高度出发，写清视线方向、被观看对象的位置、可见手部 / 手持道具 / 衣袖 / 伤痕等前景元素；POV 镜头中不能出现该角色完整正脸。
- 第三人称镜头负责交代空间关系、人物站位、动作结果和场面调度；如果从 POV 切回第三人称，必须写清切换后的机位、主体朝向和动作承接。
- POV 与第三视角来回切换时，角色服装、发型、道具持有状态、伤痕、湿衣服、光影方向、场景锚点必须连续，不能因为视角变化造成前后矛盾。
- 禁止把 POV 写成自拍、监控视角、无人机视角或第三人称过肩镜头；如需要过肩镜头，必须明确写"过肩镜头"，不要写 POV。

### 2. 导演级运镜语言

必须明确指出机位状态。**不建议**含糊其辞。

**基础运动运镜（控制叙事节奏与焦点）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **推镜** | 聚焦细节、强调重要性、拉近心理距离、制造紧张感。适用于角色做出关键决定前或发现重要线索。 |
| **拉镜** | 揭示环境、展现人物与场景的关系、营造疏离感。适用于故事结尾或展现角色身处宏大环境中的孤独。 |
| **摇镜** | 水平或垂直巡视场景，跟随主体移动。适用于扫视群像或从高处摇到低处。 |
| **移镜** | 轨道镜头，水平/向前/向后平移。创造强烈的空间感和沉浸感，适用于跟随角色行走。 |
| **升降镜** | 摇臂镜头，垂直大幅度升降。营造宏伟或压抑感，如从地面升至上帝视角。 |

**情绪表达运镜（外化人物内心世界）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **环绕运镜** | 围绕主体做弧形运动。产生强烈戏剧张力，适用于双人对峙或全方位展示关键物品。 |
| **手持感运镜** | 模拟手持摄影机的轻微晃动。增强纪实感、主观感和紧迫感，适用于追逐、打斗或混乱人群。 |
| **希区柯克变焦** | 推拉镜头同时进行反向变焦。背景产生急剧扭曲，适用于角色得知惊天秘密时的极度震惊或恐惧。 |
| **呼吸感镜头** | 带有轻微的周期性焦距变化。产生"呼吸"的生命感，适用于长时间凝视的静态特写。 |

**动作与冲击力运镜（武戏场景）**
| 运镜术语 | 导演意图与适用场景 |
|---|---|
| **甩镜转场** | 极快速地摇动镜头产生动态模糊。用于创造快速、生硬的转场，连接两个高速场景。 |
| **格挡震动** | 在撞击、爆炸或重物落地瞬间产生剧烈震动。增强物理打击感。 |
| **子弹时间** | 主体动作极度放慢甚至静止，镜头高速环绕。全方位展示决定性瞬间（如躲避子弹）。 |
| **冲刺跟拍** | 极快速侧面平移跟拍。画面充满动态模糊和颠簸感，营造强烈的速度与激情。 |

### 3. 拍摄角度与主体朝向

必须明确镜头拍摄的主体朝向，这对于 AI 视频生成时的角色一致性和空间关系至关重要：

| 角度与朝向 | 核心功能与适用场景 |
|---|---|
| **正面** | 展现角色的完整面部表情，与观众建立最直接的视觉交流。适用于重要台词、情绪爆发。 |
| **侧面 (左侧/右侧)** | 强调角色的轮廓或动作，常用于双人对话交锋或表现角色在思考、观察。 |
| **半侧面 (3/4侧)** | 最常用的角度，既能展现表情又有立体感，适用于大多数日常叙事和互动。 |
| **背面** | 营造神秘感、疏离感，或引导观众视线看向角色所看的前方。适用于角色离开、隐藏真实情绪。 |
| **仰拍** | 从下往上拍。赋予主体高大、威严、压迫感，适用于展现反派的嚣张或主角的觉醒。 |
| **俯拍** | 从上往下拍。使主体显得渺小、弱势、无助，适用于角色受挫或展现全局环境。 |
| **平视** | 与主体视线齐平。客观、平等的视角，适用于常规叙事。 |

### 4. 光影与色调设计

在关键分镜中，需补充光影以塑造氛围：

| 维度 | 分类 | 常用词汇/适用场景 |
|---|---|---|
| **光线质感** | 硬光/柔光/伦勃朗光 | 硬光（轮廓分明，制造冲突感）、柔光（干净治愈）、伦勃朗光（经典人物肖像，脸颊有三角光斑）。 |
| **光线方向** | 顺光/侧光/逆光/底光 | 侧光（强烈明暗对比）、逆光（勾勒金色轮廓、剪影）、底光（从下往上打，制造恐怖悬疑氛围）。 |
| **色调风格** | 饱和度/色温/特殊风格 | 低饱和度（情绪内敛、忧郁）、冷色调（赛博朋克/悬疑）、复古胶片色（港风怀旧）。 |

## 动作规则

- **肢体细化 + 程度量化**：写清身体部位，必须补充幅度、速度、力度（如：缓慢抬手、快速转头、用力蹬地、微微低头）。
- 优先使用低缓连续小动作：缓慢行走、轻轻抬手、顺势坐下。
- 除非剧情必须，避免狂奔、大跳、翻滚、复杂打斗等大动态不稳定动作。
- 写清动作过渡，让前后镜头因果衔接。必须写明惯性与承接关系（如：借着转身惯性顺势抬手、从停顿状态自然过渡到举手）。
- **情绪具象外化表达**：用具体的身体细节表现情绪，**不建议使用"很悲伤""非常愤怒"等抽象词汇**。
  - 悲伤：低头、肩膀微微颤抖、眼眶泛红、手指无意识地攥紧衣角、泪水在眼眶里打转。
  - 喜悦：嘴角抑制不住地上扬、眉眼舒展、脚步变得轻快。
  - 紧张/焦虑：频繁地看手表、手指不停敲击桌面、呼吸急促、无意识地啃咬指甲。
  - 愤怒：双拳紧握、下颔线紧绷、胸口剧烈起伏、眼神如刀、从牙缝里挤出话语。
  - 释然：长长地舒了一口气、紧绷的肩膀完全放松、脸上露出淡淡的微笑。

## 连续性规则

- **因果逻辑清晰**：每个镜头都要能逻辑衔接到下一个镜头。在每个镜头的末尾必须写明转场方式（如：`[硬切]`、`[叠化]`）。前后镜头必须存在严密的因果联系（例如：上一个镜头有人扔出文件，下一个镜头必须是另一个人低头看文件后产生反应）。
- 如果上个镜头抛出道具，下个镜头要交代落点或人物反应。
- 台词长度必须适配镜头时长（3秒内只能说一两句短语）。
- 旁白应克制，只在必要时使用，并在画面内容后使用『』标注。
- **角色与道具命名强一致性**：在所有分镜中，对同一个角色或道具必须使用**完全一致的称呼**（例如：必须全程使用"黄衣小哥"或"林远"，不建议中途改用"他/那个男人/少年"等代词或别称）。
- **不建议使用群体量词概括角色**：当多个角色同时出场或互动时，**不建议使用数字或量词概括**。
  - 不推荐：`三个人面面相觑`、`众人大惊失色`、`反派们冲了上来`。
  - 更推荐：`黄衣小哥，粉裙少女，灰头发老太 面面相觑`、`周凯，李总，黄毛 大惊失色`。完整写出当前镜头内出现的所有具体角色名称。

## 交接

分镜由用户确认/由用户提供后，路由到 `assets.md` 建立角色、场景和道具资产。

**关于关键帧位置的说明：**
分镜阶段只负责镜头级信息，**不预先指定"首帧/尾帧"位置**。哪些镜头需要被生成为静态关键帧图，将在 `frame.md` 阶段由 agent 依据行业关键帧判定标准（决定性瞬间、情节转折、身份揭示、状态突变等）从每个片段的镜头列表中动态锁定，可能是 1 张，也可能是多张。分镜表只需保证每个镜头信息完整，供关键帧阶段挑选。
