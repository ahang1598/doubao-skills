# 非理工科研究想法的评估框架

当研究想法不属于典型的理工科/技术类（CS、工程、材料、化学等），而是属于人文社科或金融/经济学时，`five-dimensions.md` 里的五维框架（更准/更快/更稳/更省/更广）不适用。按下面的替代框架评估。

先判断属于哪一类，再用对应的维度打分。判断依据是**研究方法**，不是学科名——传播学做话语分析走人文思辨，传播学做问卷调查走社科实证。

---

## 一、人文社科（文学、历史、哲学、政治、社会、教育、传播、艺术、马克思主义理论等）

### 五个评估维度

**1. 理论贡献（Theoretical Contribution）**

这个想法在理论层面推进了什么？是提出了新的解释框架、对已有理论的延伸或修正、还是用新材料检验了一个旧假说？

打分：9-10 = 提出一个全新的概念工具或解释框架，能改变该方向的讨论方式；6-8 = 对已有理论的非平凡延伸、修正或跨域移植；3-5 = 应用已有理论到新材料，但理论本身无推进；1-2 = 没有理论视角，纯描述。

**2. 方法创新（Methodological Innovation）**

研究设计或材料使用方式有没有突破？不是说用了多少新技术，而是有没有用不同的方式看待材料——新的文本分析策略、新的档案组合、新的跨学科视角、新的田野方法。

打分：9-10 = 开创性的方法设计，产生了之前不可能产生的发现；6-8 = 有意识的方法论选择，和研究问题匹配良好；3-5 = 常规方法，无特别的设计考量；1-2 = 方法与研究问题不匹配。

**3. 经验发现（Empirical Discovery）**

揭示了什么之前没有被记录、系统化、或充分解释的现象或规律？不一定是实验数据——一手文献的新发现、被遮蔽的历史事实、质性材料中的新模式都算。

打分：9-10 = 发现了一个重要的、被忽视的现象，有改变该领域认知的潜力；6-8 = 系统化了一个已知但未充分记录的模式；3-5 = 证实了已有发现，但没有新信息；1-2 = 没有经验发现，纯理论推演。

**4. 解释力（Explanatory Power）**

能不能用更少的假设解释更多的现象？和已有解释相比，是否有更高的内洽性、更少的例外、或更精确的边界条件？

打分：9-10 = 统一了之前互相竞争的多个解释；6-8 = 比最好的已有解释多覆盖了一个重要的反例或边界条件；3-5 = 和已有解释在覆盖力上持平；1-2 = 比已有解释更窄。

**5. 社会意义（Social Significance）**

对政策、实践、公共话语、或特定社群有没有可操作的启示？不是说每个研究都必须"有用"，但如果有，这是一个加分项。

打分：9-10 = 直接回应一个正在被讨论的政策或公共议题，能改变决策者的判断框架；6-8 = 对某个实践领域有间接但可辨识的启示；3-5 = 理论意义为主，社会意义有限；1-2 = 纯学术内部讨论。

### 打分纪律

和理工科五维一样：默认 5 分起评，加分要有依据（材料证据或机制论证）。人文社科的机制论证形式不同——不是"消融实验证明了核心模块有效"，而是"这个理论框架能解释对手理论解释不了的反例 X"。

### 致命伤替代

以下致命伤替代 `fatal-flaws.md` 中的 STEM 版本：

1. **文献对话不足** — 替代"跟最接近的工作比没有新意"。该领域的核心文献（经典理论、奠基性研究、近年重要争论）没有被认真讨论，让审稿人怀疑作者不了解该方向的学术脉络。
2. **研究问题不可操作化** — 替代"关键主张验证不了"。研究问题太宏大（"全球化如何影响文化"）或太模糊，无法设计出可以回答它的研究方案。
3. **证据与主张不匹配** — 替代"基线不是真基线"。所使用的材料（文本、案例、访谈、问卷）不足以支撑论文的核心主张。比如用 3 个案例就声称发现了"普遍规律"。
4. **方法与范式冲突** — 新增。用实证方法回答思辨问题（用回归分析回答"美是什么"），或用思辨方法回答实证问题（用理论推演代替田野调查来解释特定社区的行为）。
5. **概念含混** — 新增。核心概念没有清晰定义，在不同段落里含义漂移。人文社科论文的核心概念是论文的地基，地基不稳全篇崩。

其余通用致命伤（说不清谁在乎、摊子铺太大、从不谈失败、数据或伦理过不去、能力不匹配）仍然适用。

## 二、金融与经济学

### 五个评估维度

**1. 识别策略质量（Identification Quality）**

因果推断的识别策略是否可信？如果论文声称 A 导致 B，它用什么策略排除了反向因果、遗漏变量和选择偏差？

打分：9-10 = 自然实验或准实验设计，外生性论证清晰且难以挑战（如政策断点、地理边界、意外事件）；6-8 = 工具变量或双重差分，外生性论证合理但有已知局限；3-5 = 控制变量回归，没有识别策略讨论；1-2 = 相关性分析声称因果。

**2. 经济显著性（Economic Magnitude）**

发现的效应在经济意义上大不大？统计显著 ≠ 经济显著。一个 t 值为 8 但效应量是 GDP 的 0.001% 的发现，对政策没有意义。

打分：9-10 = 效应大到足以改变政策讨论或投资决策；6-8 = 效应在经济意义上不可忽视；3-5 = 效应存在但偏小；1-2 = 只有统计显著性，经济意义不明。

**3. 机制新颖性（Mechanism Novelty）**

揭示了什么新的经济机制或传导渠道？不是说"发现了一个新的相关关系"，而是说"解释了为什么 A 影响 B，通过什么具体的渠道"。

打分：9-10 = 发现了一个之前没有被记录的传导渠道，并用中介分析或结构模型做了分解；6-8 = 用已知渠道框架解释了一个新现象；3-5 = 记录了相关关系，没有讨论机制；1-2 = 黑箱回归。

**4. 数据贡献（Data Contribution）**

用了新的数据集、或者用已有数据发现了新的 stylized fact？手工搜集的微观数据、首次被用于学术研究的行政数据、跨国面板的新组合，都是数据贡献。

打分：9-10 = 构建了一个新数据集，本身就是该领域的基础设施；6-8 = 首次使用某个行政数据或微观调查回答该问题；3-5 = 使用公开数据集；1-2 = 使用被大量研究过的经典数据集，没有新发现。

**5. 政策相关性（Policy Relevance）**

发现对政策设计有没有直接可用的含义？能不能告诉政策制定者"如果你做 X，效果会是 Y"？

打分：9-10 = 直接回答一个正在被辩论的政策问题，且识别策略足以支撑因果推断；6-8 = 对政策讨论有间接但可辨识的启示；3-5 = 学术意义为主；1-2 = 和政策无关。

### 打分纪律

同理：默认 5 分起评。金融/经济学的加分依据特殊之处在于：识别策略的可信度比结果的大小更重要。一个效应小但识别干净的发现，比一个效应大但内生性严重的发现更值得做。

### 致命伤替代

1. **内生性未处理** — 替代"关键主张验证不了"。论文声称因果关系，但没有可信的识别策略来排除反向因果或遗漏变量。
2. **样本选择偏差** — 替代"基线不是真基线"。样本的选取方式导致结果不能推广（只用上市公司数据研究"企业行为"、只用某银行客户数据研究"消费者行为"）。
3. **p-hacking 嫌疑** — 新增。只报告了显著的结果、试了大量回归规格但只展示了最好看的那个、没有 robustness check。
4. **缺少 robustness** — 新增。核心结果没有用替代指标、替代样本期、替代回归规格做稳健性检验。金融/经济学的审稿人会要求至少 2-3 个 robustness check。
5. **经典文献遗漏** — 替代"跟最接近的工作比没有新意"。该方向的奠基性论文或最近的 survey 完全没有被引用或讨论。

## 三、怎么判断用哪套框架

在 idea-evaluate 的第一步"先复述一遍"时，顺带判断研究范式：

| 信号 | 范式 | 用哪套维度 |
|---|---|---|
| 有实验、有基准、有消融、有模型架构 | STEM/技术类 | 用原版五维（更准/更快/更稳/更省/更广） |
| 有文本分析、史料、概念辨析、话语分析、思辨论证 | 人文思辨 | 用人文社科五维 |
| 有问卷、访谈、统计分析、案例研究、田野调查 | 社科实证 | 用人文社科五维（侧重经验发现和方法创新） |
| 有回归、工具变量、DID、RDD、面板数据、金融变量 | 金融/经济学 | 用金融五维 |
| 有法条、案号、判例、司法解释、制度比较 | 法学 | 用人文社科五维（侧重理论贡献和社会意义） |
| 分不清 | 先问一句 | "这项研究的核心方法是什么？是做实验、做调查、做文本分析、还是做理论推导？" |

如果一个想法跨范式（比如"用 NLP 技术分析政治话语"），按其**核心贡献**归类：如果核心贡献是 NLP 方法的创新，走 STEM；如果核心贡献是政治学的新发现，走人文社科。

> **致命伤补充**：社科实证（问卷/统计/案例）除人文社科致命伤外，其统计类硬伤（样本选择偏差、p-hacking、缺 robustness check）参照上文金融/经济学的致命伤替代表；法学除人文社科致命伤外，补案号错误、判决编造这类专属硬伤。
