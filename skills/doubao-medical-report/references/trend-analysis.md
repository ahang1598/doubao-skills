## 趋势分析

## 必要步骤

对每个重复出现的指标：

1. 按日期排序。
2. 统一项目名和单位。
3. 保留每次检查的参考范围。
4. 判断每个值相对其参考范围的位置。
5. 计算绝对变化和百分比变化。
6. 标记趋势状态。
7. 解释临床意义和管理含义。

百分比变化、均值、标准差、日期间隔和多点统计使用 `calculator`。

## 趋势状态

- `new_abnormal`：过去正常，本次异常。
- `persistent_abnormal`：连续两次或多次异常。
- `worsening`：远离目标或参考范围。
- `improving`：向目标或正常范围改善。
- `fluctuating`：上下波动，没有明确方向。
- `stable`：无明显临床意义变化。
- `insufficient_data`：可靠数据少于两个时间点。

## 临床意义判断

不要把所有数值变化都解释成医学变化。需要考虑：

- 变化幅度。
- 是否跨过参考阈值。
- 是否持续存在。
- 相关指标是否同向变化。
- 检查条件是否变化：空腹/非空腹、用药、急性疾病、不同实验室方法。

## 跨领域趋势模式

重点识别组合风险：

- 代谢风险：BMI/腰围 + 空腹血糖/HbA1c + TG/HDL-C + 血压。
- 心血管风险：血压 + LDL-C/apoB + 糖代谢 + 吸烟。
- 肾脏风险：肌酐/eGFR + 尿蛋白 + 血压 + 糖尿病。
- 脂肪肝/代谢相关肝病：BMI + TG + 血糖 + ALT/GGT + 超声脂肪肝。
- 甲状腺结节：大小变化 + TI-RADS 分级 + 原报告建议。

## 可视化要求

完整报告中：

- 重复数值指标优先使用飞书图表助手趋势折线图；无法插入图表助手卡片时，用趋势表替代并说明原因。
- 有参考范围时显示正常区间。
- 当前值要醒目标注。
- 异常等级使用低/中/高或轻度/中度/重度等克制标签，不用恐吓式颜色措辞。
- 多年数据用历年异常对比表。
- 每个趋势图都要有 `图表解读`，说明变化方向、临床管理含义和不能过度解读的部分。

所有图表必须能从证据表复现。关键数值不确定时，先请用户确认；确认前不生成图表。
