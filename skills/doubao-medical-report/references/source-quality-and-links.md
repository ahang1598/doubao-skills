## 来源质量与链接校验

## 来源优先级

医学解释、阈值、分级、随访建议和紧急风险判断需要外部来源时，按以下顺序选择：

1. 国家级权威来源：国家卫生健康委、国家疾控局、国家药监局、国家医学质控/标准相关机构等官网。
2. 权威指南和共识：中华医学会及其分会、国家级诊疗指南、行业标准、专家共识。
3. 教材和权威工具书：人民卫生出版社教材、专业教科书、权威医学参考书。
4. 权威期刊文献：核心期刊、同行评议期刊、指南原文或指南解读文献。
5. 专业机构官网：三甲医院、大学附属医院、专科中心、疾病控制/公共卫生机构官网。
6. 其他医院官网或权威科普平台：仅在上级来源不足时补充使用。

不要使用问答社区、论坛、百科式未署名内容、营销软文、广告页、无发布日期或无机构署名转载内容、与引用结论不匹配的页面、无法打开或需要登录才能阅读核心内容的链接。

## 覆盖维度

参考文档应覆盖报告中的关键医学维度，而不是只堆链接：

- 检验/检查项目含义。
- 异常阈值或分级。
- 风险和就医优先级。
- 复查和随访。
- 生活方式管理。
- 安全边界。

每个核心结论至少有一个可追溯来源；同一来源可支持多个相邻结论，但不能用单个泛泛科普页支撑全部报告。

## 链接呈现

飞书文档“信息来源/参考文档”章节必须使用可点击链接：

```xml
<a href="https://example.gov.cn/xxx">国家卫生健康委：标题</a>
```

每条来源至少包含来源标题、发布机构或期刊名、链接、支持的报告结论、发布日期或访问日期。不要只粘贴裸 URL。

## 链接检查

输出前必须检查：

- URL 使用 `https://` 或可信 `http://`。
- 页面能打开，HTTP 状态通常为 2xx 或 3xx。
- 页面标题、机构、正文内容与引用结论匹配。
- 不是登录页、错误页、广告页或搜索结果页。

如果链接不可访问或内容不匹配，删除该来源并换用更权威、可访问的来源。若无法找到可用外部来源，写明“外部来源不足，需医生确认”，不要用不可靠链接补位。

## 工具使用

- 使用 `general_search` 查找国家级网站、指南、教材/期刊或医院官网页面。
- 涉及药品、医生、医院时使用 `MedicalSearch`；若返回结果没有可点击 URL，可在“信息来源”中写可追溯标题和资源类型，同时不要伪造链接。
- 创建飞书文档草稿后，若草稿含外部 URL，运行 `scripts/check-reference-links.js draft.xml`。任何失败链接都要替换或移除后再交付。
