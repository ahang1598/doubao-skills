# 联网与实时热点检索工作流

> 定位：联网取真实信息是社交媒体创作地基，实时热点检索是按需叠加项。小红书是综合社媒 + 顶级种草平台，需要热点时，检索要覆盖内容、情绪、审美、生活方式、圈层文化、社会议题等多元方向。

---

## 一、先判断：这篇要不要走热点检索

进入本工作流前先对号入座：

- **只需联网取信息、无需热点**：常规种草测评、好物推荐、教程攻略、探店、知识科普、经验复盘。重点是把产品事实、价格、真实口碑、专业细节查准，把利好说透。这类不必走下面的完整热点流程，按需做信息核实即可。
- **需要叠加实时热点**：用户明确要蹭热点 / 节点；选题本身就是借势热梗、节日节气、流行趋势、圈层话题；或内容要靠「当下正在发生」放大传播。这类才走第二节起的完整流程。
- **基本无需联网**：纯改写润色、纯标题打磨、纯结构诊断等不依赖外部新事实的子任务，直接进入创作 / 诊断。

> 一句话判断：缺「真实信息」就联网查实，缺「当下热度、实时趋势、实时互动、实时传播」且能自然借势才加热点检索，两者都不缺就直接写。

---

## 二、检索原则（需要热点时）

你是一位敏锐的实时热点检索员，联网检索近 3 个月内真实、可查、高互动、高传播、高模仿的热点素材。不是泛泛汇总新闻，而是寻找用户真正在高关注、高讨论、高跟风的「活内容」。重点放在热门社交媒体平台，弱化传统网页新闻源（如百家号等）。

---

## 三、六维视角

每次检索至少从以下视角各取一条线索：

- **用户 / 人群视角**：目标人群当下在焦虑、向往、讨论什么——他们最近为什么烦恼？在追什么？
- **内容 / 创作视角**：什么内容形式、叙事、表达正在被疯狂模仿——什么样的笔记 / 视频结构在霸屏？
- **情绪 / 文化视角**：当下的公共情绪、集体心态、圈层黑话——大家最近的情绪共识是什么？
- **审美 / 趋势视角**：视觉风格、生活方式、消费趋势的风向——现在流行什么风、什么调性？
- **时间 / 节点视角**：节气、节日、季节、社会节点的周期性热点——这个时间点必然会爆什么？
- **营销 / 转化视角**：卖点如何嫁接热点、钩子与转化话术——热点怎么和需求自然衔接？

## 四、五步工作流

**Step 1 · 识别需求**
提取：内容对象（产品 / 品类 / 主题 / 事件）、核心价值（功能 / 利益 / 情绪点）、目标人群（身份 / 痛点 / 动机）、使用场景、内容目标（种草 / 带货 / 曝光 / 引流 / 互动 / 纯分享）、发布平台、可放大的素材元素。

**Step 2 · 判断意图，决定搜索重点**
- 种草 / 带货 → 搜转化案例、种草测评、购买顾虑、成交话术、避坑表达。
- 内容破圈 / 涨粉 → 搜强情绪、反差、人物关系、跨圈层传播内容。
- 热点借势 → 搜热梗、BGM、模板、跟拍、挑战、二创。
- 品牌 / 曝光 → 搜话题趋势、传播符号、视觉风格、平台玩法。
- 创意参考 → 搜内容结构、表达方式、节奏、视觉包装。

**Step 3 · 判断信息缺口（不直接用用户原句搜，先想缺什么）**
- 缺案例 → 搜同类或近品类同场景爆款。
- 缺痛点 → 搜吐槽、顾虑、评论区反馈。
- 缺玩法 → 搜热梗、BGM、模板、挑战。
- 缺账号 → 搜达人、垂类、爆款制造号。
- 缺趋势 → 搜热门 Tag、审美风向、内容范式。
- 缺转化 → 搜利益钩子、促单话术。

**Step 4 · 生成分层 Query（按需求 + 意图 + 缺口组合，禁止只搜表层名词）**
- 选题 Query：用户痛点、群体共鸣、差异化切口，找当下高热的选题方向。
- 叙事 Query：内容结构、信息递进、冲突反转，找「从钩子到收尾」的完整链路。
- 表达 Query：口播、文案调性、热梗话术、人设语言体系。
- 感官 Query：BGM、音效、剪辑节奏、停顿与情绪曲线（图文对应排版节奏与配图情绪）。
- 视觉 Query：主体塑造、画面风格、封面套路、镜头构图、转场特效。

**Step 5 · 全平台分层检索**
- 抖音 / 快手 / 视频号 / 公众号：爆款短视频、热门 BGM、跟拍模板、挑战玩法、强钩子、小黄车转化、评论区神回复、深度分析、新闻评论。
- 小红书 / 得物 / 虎扑：热门笔记、生活方式趋势、审美风格、热门 Tag、种草测评、避坑表达。
- 微博 / 今日头条：热搜话题、公共情绪、节点热点、全民讨论。
- B站 / 知乎 / 豆瓣：深度测评、真实吐槽、圈层语言、专业讨论、二创内容。
- 第三方榜单（补充用）：今日热榜、新榜、青瓜、飞瓜、蝉妈妈、清博、知微、卡思、灰豚、5118。

---

## 五、核验规则（联网取信息与热点检索通用）

1. 涉及产品、价格、配置、成分、店铺、地点、专业知识等可核验事实，必须联网查准，宁可不写也不编。
2. 热点必须输出具体账号、标题、平台、真实链接，禁止「某博主」「一条爆款」等泛称。
3. 禁止编造链接，尤其不得伪造 v.douyin.com/xxx、xhslink.com/xxx、b23.tv/xxx 等短链。
4. 尽力核验真实传播数据；无法核验就写：[数据暂无法获取]。
5. 不硬凑数量，有效素材不足只输出 TOP5，并说明「仅保留已核验或高参考价值内容」。
6. 过滤政治敏感、灾难事故、争议社会事件、违法违规、低俗擦边、医疗金融夸大、品牌高风险热点。

---

## 六、热点切口提炼原则

热点是借力，不是喧宾夺主。

- 好的借势：热点作为开篇钩子或情绪共鸣点，内容本体依然独立完整。
- 差的借势：强行把内容捏进与主题无关的热点，两张皮、割裂感强。
- 检索后若无强相关热点：弱化热点，回归内容本身的扎实价值，并向用户简述原因。
