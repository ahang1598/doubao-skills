# 爆款/优质文本笔记创作方法库

> 用途：承接系统化写作方法、诊断优化、账号定位与合规细则。需要展开具体方法时读取本文件对应章节 `xhs-note-proposal.samples.creation-methodology.md`。

## 一、什么是「爆款 / 优质笔记」

### 1.1 用户为何而来（写作的出发点）

用户打开小红书，本质是满足精神或物质需求。他们不在意文笔多完美，只在意「这篇能不能解决我的问题、降低我的选择成本、给我情绪认同或生活灵感」。永远从「用户需要什么」动笔，而非「我想表达什么」——个人表达有价值，但必须和用户需求挂上钩。

优质笔记至少命中一种用户价值：

- 帮选择：怎么买、怎么选、值不值、适合谁、不适合谁。
- 帮避坑：哪些别做、哪些别买、哪些容易被忽略。
- 帮提升：教程、步骤、清单、模板、流程、经验复盘。
- 帮共鸣：说出用户的焦虑、委屈、期待、松弛感与自我确认。
- 帮发现：新产品、新店、新路线、新风格、新生活方式。
- 帮表达：提供可转发、可收藏、可评论的观点和话术。

### 1.2 数据维度（量化判定，分两档）

只作诊断参考，不当平台铁律，更不据此承诺「一定爆」；需按赛道体量和账号阶段动态校准。带货类还要叠加「求链接评论占比」等转化信号。

- 赞藏比（互动 / 浏览）：优质线 ≥10%，爆款线 ≥5% 起步、越高越好；浏览:点赞:评论 约 100:10:1 为健康结构。
- 点赞总量：优质线 ≥1000，爆款线 ≥5000；单日互动涨幅 >50% 为优质信号。
- 浏览总量：优质线 ≥1 万，爆款线 ≥10 万；点击打开率 >5% 表现优异。
- 评论总量：优质线 ≥100（2 小时内），爆款线 ≥500；整体互动率 >3% 为优质水准。

### 1.3 体感维度（四步判定）

优质笔记同时满足：第一眼想点开（封面 + 标题）→ 点开愿读完（正文节奏与价值）→ 读完有动作（点赞 / 收藏 / 评论 / 搜索 / 下单）→ 看完记得住（人设或风格辨识度）。

## 二、爆款的底层逻辑与文本特点

### 2.1 底层逻辑：70% 成熟结构 + 30% 差异化

爆款很少凭空创新，多是「70% 对成熟结构的学习 + 30% 场景 / 人设 / 观点 / 证据的差异化」。拆解爆款时不抄原文，只提炼结构：标题用了什么钩子？封面主体与文字占比？开头是结论、痛点、反差还是故事？正文是步骤、清单、对比还是复盘？有没有真实证据（场景 / 价格 / 截图 / 前后对比 / 失败经历）？有没有互动入口？关键词是否落在标题、正文、标签、图片文字里？

### 2.2 四大共性（所有赛道通用的「骨」）

1. 站在用户立场：摒弃自我炫耀，只回答「用户能得到什么」。
2. 干货密度大于广告浓度：合集 / 技巧类更易起量；单品推广必须先输出足量干货，再柔性植入。
3. 情绪共鸣大于功能罗列：用场景化情绪词替代中性描述。
4. 真实可信的细节：信任建立在「具体到毛孔」的细节上（「阿姨会贴心提醒肉烤好了」），而非泛泛「很好用」。

### 2.3 差异化文本特性（按内容类型因类施写）

- 测评类：客观分析 + 真实使用感，核心抓手是客观对比 + 适合谁 / 不适合谁，适配美妆、家居、数码。
- 教程类：步骤清晰、可复制，核心抓手是照着做就能复现，适配美妆步骤、学习、生活技巧。
- 合集类：信息密度高、便于收藏，核心抓手是分类清楚、一站获取，适配书单、好物清单、攻略。
- 开箱类：直观、激发好奇，核心抓手是视觉好奇 + 第一感受，适配新品、小众好物。
- Vlog / 沉浸式：强代入、治愈解压，核心抓手是情绪流动 + 场景节奏，适配生活记录、护肤、学习。
- 探店类：环境 + 服务 + 推荐理由，核心抓手是价格 / 口味 / 路线 / 适合人群，适配餐厅、咖啡馆、景点。
- 互动型：抛问题、求帮助，核心抓手是让用户有话可说，适配求推荐、帮我选、晒图。
- 避坑 / 观点类：反常识、有判断力，核心抓手是反常识切口 + 取舍，适配知识、消费决策、行业观察。
- 意识流 / 自我表达：不完美、自省、哲思，核心抓手是脆弱真诚胜过完美攻略，适配旅行随笔、知识分享、个人心得、成长复盘、情绪。

## 三、模块 A · 选题：三层筛选法

逐层自问，三问皆过才算好选题：

1. 价值落地：这篇能解决什么具体问题？（「4 步修复干枯受损发质」「28 岁普通女生逆袭蜕变」）
2. 人群聚焦：哪类人会觉得「这就是写给我的」？锁定圈层普遍焦虑（职场内卷、带娃难题、身材焦虑），用平台数据工具（灵感榜等）核验热度。
3. 差异化切口：有没有新场景、新证据、新观点、新对比、新方法？（「用心理学拆解内耗」胜过单纯心灵鸡汤）

高潜力选题角度模板：

- 「大家都在夸，但我发现……」
- 「新手最容易踩的坑……」
- 「预算有限也能……」
- 「我试了 X 个，只留下这 X 个……」
- 「不是 X 没用，是你少了……」
- 「适合 X 人群，不适合 Y 人群……」
- 「看完再决定要不要买 / 做 / 去……」

## 四、模块 B · 标题：3 秒抓眼球（十大爆款公式）

标题要同时完成三件事：让用户看懂、让用户想点、让搜索能识别。避雷：字数 ≤20，核心关键词前置，拒绝堆砌无关词。

- 数字型（X 个方法 / X 步搞定）：3 步搞懂小红书起号底层逻辑。
- 反常识型（一般人都错了 / 别再这样做）：发了 100 条笔记却没人看，问题出在哪？
- 对比型（A 和 B 哪个更好）：首页推荐和搜索流量，哪个更值得做？
- 利他型（教你如何 / 模板合集）：新媒体运营日报模板分享，拿走不谢！
- 成长型（我的变化 / 真实复盘）：半年从 0 到涨粉 10W，这份总结送你。
- 踩坑型（错误经历 / 避免失败）：踩过 3 次限流坑后，我才懂这些不能做。
- 合集型（系列整理 / 合集）：爆款标题模板合集，建议直接收藏。
- 需求型（针对用户痛点）：小红书新手该从哪开始？别再瞎试了。
- 工具型（工具教程 / 指南）：剪映 + 新红 + 石墨，工具合集来了。
- 引导型（与用户互动）：你也在为起号烦恼吗？看完这篇你就懂了。

高频句式补充：

- 数字冲击：「160 小个子试穿 40 条牛仔裤整理出显瘦黄金版型」。
- 痛点直击：「熬夜党自救！6 个快速恢复气色小妙招」。
- 悬念反差：「闺蜜以为我花 2W 做的医美，实际平价护肤搞定」。

标题检查清单：

- 核心关键词前置，一个标题只放一个主钩子。
- 标题承诺必须与正文一致，不做标题党式失真。
- 少用「绝绝子 / 天花板 / 封神」等空泛词，除非贴合人设语境。
- 完整创作时按不同公式给 8–12 个分类备选，供挑选或做 A/B；只要标题的子任务同样多给备选，不附正文。

## 五、模块 C · 封面：强视觉吸睛

封面的任务：让用户 1 秒内判断「这篇和我有关，值得点开」。

- 信息分层：主标题大于辅助小字大于装饰元素；封面文字整体占比小于三分之一；一张图只讲一个核心主题。
- 对比冲击：Before & After 对比图、数据打卡记录（「30 天控油变化」）、价格差异、错误示范。
- 配色适配赛道：美妆护肤偏粉金柔和色系；知识干货偏蓝白极简干净风。
- 构图四原则：主体突出、留白呼吸、善用拼图（攻略 / 合集）、引导线或对角线增强纵深。
- 一致性成 IP：固定模板 + 统一色调 + 个人出镜（护肤 / 穿搭 / 美妆类真人更具信任感）。
- 常见封面类型：大字报（观点 / 避坑）、备忘录（清单 / 流程）、多拼图（合集 / 对比）、对比类（改造 / 护肤）、图表类（排名 / 数据）、场景类（探店 / 家居）、人物类（穿搭 / IP）、聊天类（求助 / 争议）、表情类。
- 封面文案示例：「新手别乱买」「我试了 18 个只留 5 个」「通勤女生照着穿」「敏感肌先看这篇」「预算 200 也能抄」。

## 六、模块 D · 正文：按内容选骨架的行文框架

正文不是把信息堆齐就行，而是一条「让人看下去、看进去、看完动手」的注意力滑梯。把它拆成三件事：用什么骨架承载内容、用什么心理杠杆推着读者往下走、每一段怎么落到具体。骨架不是模板填空，是按内容类型选最顺的那条叙事线；段落占比只是节奏参考，实际段数随内容自然生长，不强行三段、不套总分总。

### 6.0 先选骨架：按内容类型挑叙事线

没有万能结构，只有「这篇内容最该怎么被讲」。先认清本篇属于哪类，再选骨架，比硬套金字塔有效得多。

- 痛点解决型（PAS 线）：点破痛点 → 把痛感再往深里戳一层（不解决会怎样、你是不是也这样）→ 给出解法和方法。适合护肤、健康、职场、母婴等「用户有明确困扰」的内容。底层是损失厌恶——人对「失去 / 踩坑」的敏感度，约是「得到」的两倍，所以「再不改会怎样」往往比「改了能得到什么」更抓人；但只用来戳真问题，不制造焦虑、不贩卖恐惧。
- 种草转化型（亲历五步线）：我的苦恼 → 我的问题 → 我试的解法（产品自然登场）→ 我的真实体验与变化 → 我的总结 + 适合谁。这是小红书最稳的种草骨架，全程「我」在场，产品是解药不是主角。
- 干货教程型（Why-What-How 线）：先讲为什么值得做 / 为什么是它 → 是什么 → 怎么做（可复制的分步）。让用户既知其然也知其所以然，照着做能复现。
- 测评对比型（横纵对比线）：先立标准（看哪几点）→ 横向比同类或纵向比前后 → 给出适合 / 不适合人群。重点是客观、有取舍，敢点名缺点。
- 故事种草型（SCQA 线）：情境（本来日常）→ 冲突（出问题 / 不满足）→ 转折（遇到它）→ 结局（变化 + 体会）。靠故事降低广告感，把卖点藏进情节。
- 合集清单型（分类罗列线）：开篇给「为什么收藏这篇就够」的承诺 → 分类逐条输出 → 收尾给取舍建议。信息密度是它的命，钩子是「一篇省你翻 20 篇」。

> 选骨架口诀：有痛点走 PAS，要带货走亲历五步，教方法走 Why-What-How，做选择走对比，讲感受走 SCQA，攒信息走合集。混合内容就以主诉求定主骨架，其余作为段落穿插。

### 6.1 黄金开头：3 秒决定走或留

读标题的人是读正文的数倍，而点进来的人会在前 3 行决定去留。开头唯一的任务，是让人读下一行——把「价值 / 冲突 / 好处」前置，别拿寒暄和铺垫浪费第一屏。

6 种高效开头气口（按内容选其一）：

- 痛点开头：直说困扰，让对的人瞬间对号入座（「混油皮一到下午就脱妆卡粉，是不是你」）。
- 结论开头：先抛判断再解释，给急性子读者即时价值（「这 3 件通勤单品，闭眼买不会错」）。
- 反差开头：点破普遍误区，制造认知冲突（「贵的不一定好，我交了 2000 学费才懂」）。
- 结果开头：先亮变化 / 收益 / 前后对比，用成果勾人（「30 天，腰围少了 8cm，方法在文末」）。
- 身份开头：精准点名目标人群，制造专属感（「150 小个子、预算 300，这篇是给你的」）。
- 求助开头：当用户的诉求本身是求助时，扮演用户抛问题给社区（「真的有平价又持妆的粉底吗，求姐妹们砸答案」）。

> 开头红线：标题承诺什么，开头就接住什么——首屏和标题对不上，信任 3 秒崩塌，再好的正文也救不回来。

### 6.2 中段留人：心理杠杆 + 可选模块

中段是「沉淀留人」的主战场，每隔一段就要给读者一个继续看的理由。除了把信息讲清楚，靠四个心理杠杆维持牵引力：

- 好奇缺口：留一个「不看完会痒」的悬念，但承诺必须和正文一致（「尤其第 3 个，我后悔没早知道」）。
- 社会认同：让读者看到「和我一样的人」也在用 / 也踩过（「评论区好多敏感肌姐妹都翻车在这步」）。
- 损失厌恶：把「不这样做的代价」写具体，替用户避坑（「这一步偷懒，前面全白做」）。
- 情绪共鸣：先精准描述读者已知的真实场景，让人脱口而出「对，就是这样」，再给解法。

中段可选模块（按骨架取用，不必全上）：分步方法、产品 / 方案对比、真实使用场景、踩坑与修正、时间线与结果、个人故事 + 可复用经验、优缺点 + 适合 / 不适合人群。

> 可信度心法：一篇笔记别贪心地堆满卖点——专攻 1–2 个核心点，比面面俱到更能打；适度点出非硬伤的小缺点，反而让「全是优点」的内容显得真实可信。卖点全展示、零缺点，是用户一眼识破的硬广信号。

### 6.3 有力结尾：给一个明确的下一步

结尾不是「希望对你有帮助」式收场，而是把读者的注意力转成一个具体动作——收藏、评论、搜索或决策。CTA 越具体、心理成本越低，转化越高。

6 种结尾方式（按目标选其一或叠加）：

- 决策建议：直接给「适合……不适合……」，帮用户落地选择。
- 收藏提醒：给一个收藏的实在理由（「先收藏，下次照着这张清单逐项检查」）。
- 评论互动：抛一个低门槛真问题（「你是哪种肤质，我按情况补一版」）。
- 软转化：用关键词钩评论（「要这份清单 / 模板的，评论区扣『要』」），不硬求赞收藏。
- 后续预告：给个追更理由（「下篇拆怎么挑适合自己的色号」）。
- 金句收束：留一句能被记住、被截图转发的话，给整篇画一个记忆点。

### 6.4 文采六法（正文从「说清楚」升级到「读得爽」）

正文质量是笔记的命脉。同样的信息，平铺直叙是流水账，有文采才能让人一行行读下去。文采的本质，是把「信息点」写成「用户能感受到的生活变化」——是生活里长出来的漂亮话，不是堆辞藻。

1. 动词带画面：用能「看见」的动作替代抽象形容词。「冰箱很大很好用」不如「拉开门那一下，整层冷气糊上来，塞了一周的菜还有富余」。
2. 细节到毛孔：信任来自具体。「服务很好」不如「阿姨守在旁边，肉一到火候就喊我『好啦好啦快夹』」。
3. 长短句造节奏：短句抓情绪（「真香」「绝了」「直接破防」），长句铺画面，靠断句与换行造呼吸感，拒绝一逗到底。
4. 情绪有颗粒且克制：把「开心 / 满意」落到具体场景的具体反应（「下班想到家里有它等着，脚步都轻快了」）；不滥用「绝了 / 封神 / 天花板」，让细节替用户得出结论。
5. 金句收尾：结尾留一句能被记住、被截图转发的话，而非「希望对你有帮助」式客套。
6. 埋记忆点：每篇至少设计 1–3 个能被记住的点——金句、画面、反差、生活切片或强判断，而非通篇平均用力。

先具象、后抽象（落地顺序法则）：先写场景再写卖点、先写感受再写参数、先写冲突再写解决、先写细节再写结论。让每个判断都站在具体之上。

5 种正文气口（开场语感模板）：

- 真实复盘：「买之前我也纠结了很久，真正用下来才发现……」
- 生活切片：「下班回家第一件事，就是……」
- 反差惊喜：「本来只是随手试试，结果……」
- 踩坑醒悟：「踩过一次坑之后，我现在选……只看这几点」
- 审美叙事：「阳光落在……的时候，才知道这个选择有多对」

卖点转译（参数 → 体验）：

- 「低至 16 分贝」转译成「晚上开着几乎听不到，浅睡的人也不会被吵醒」。
- 「401L 容量」转译成「一周的菜、饮料、冻肉都能分区放下」。
- 「柔雾妆效」转译成「不是死哑光，是脸被轻轻磨平的干净感」。
- 「自重力调节」转译成「大人孩子换着坐，不用每次重新调」。
- 「50 米防水」转译成「洗澡、游泳、出汗都不用摘」。

画面感（形容词 → 画面）：

- 「舒服」写成「下班回家把自己陷进沙发里，腰被稳稳托住」。
- 「高级」写成「哑光表面不反光，放在原木色空间里很安静」。
- 「方便」写成「睡前手机、眼镜、耳机都有地方放，不用摸黑找线」。
- 「好吃」写成「外层焦香，里面还挂着汁，咬开会烫嘴」。
- 「治愈」写成「翻到那一页时，像有人替你把心里的皱褶抚平」。

分品类文采方向：

- 生活方式类：先写空间、日常、情绪，再自然带出产品——用户的理想生活才是主角，产品是配角。
- 家具家电类：参数转生活体验——尺寸对应空间、噪音对应睡眠、容量对应囤货、能效对应电费焦虑。
- 美妆护肤类：先写肤质 / 发质 / 妆容困扰再写产品；质地、肤感、妆效、香味要具体，但功效不得医疗夸大。
- 科技数码类：用「不吹不黑、对比旧设备、适合谁不适合谁」提升信任，参数必须转译成场景。
- 教育 / 母婴 / 知识类：先写焦虑或误区再给方法，信息要清楚但语气不能像讲课。
- 美食 / 探店类：开头要有口语冲击，菜品具体到口感、温度、分量、香气、排队、价格、适合场景。
- 书单 / 阅读类：可更有文学感但保持可读性，每本给一个情绪定位或精神价值，不只罗列简介。

> 取法范本：动笔前先把用户素材拆成“事实、场景、痛点、变化、证据、限制”，再判断本篇需要哪种表达力——动词更有画面？细节更具体？情绪更克制？节奏更轻快？所有表达都从本篇素材重新生成，不从博主笔记原文迁移句式、段落或故事。
> 红线：文采不等于堆辞藻。宁可朴素真实，也不空洞铺陈、不为修辞而修辞。

### 6.5 让用户停留：利好与差异化（种草 / 营销类的核心）

种草和营销内容能不能让人停下来看，取决于两件事：对用户的利好讲没讲透，以及有没有「别处刷不到」的差异点。

讲透利好，至少命中一类并落到具体场景（探店 / 测评类可一次叠加多类）：

- 价格利好：平价、性价比、平替、省钱、预算友好——写清省在哪、对谁划算、种草攻略写清价格优势（产品/地址/服务等高性价比）。
- 好用利好：实测有效、解决了哪个具体痛点、别处替代不了的体验。
- 省时高效利好：替用户省时间、降决策成本，一篇看完不必再做功课，清单 / 攻略「抄作业即可」。
- 颜值审美利好：好看、出片、有氛围、提升空间或自我形象，把视觉价值写进画面。
- 稀缺独特利好：别处买不到 / 抢手 / 小众宝藏 / 限定，或「我用过一圈只有它戳中这个需求」的独特性。
- 服务体验利好：探店 / 服务类写清环境、服务、流程、人情味。
- 社交认同利好：能晒、能聊、有谈资、彰显品味或身份认同，让用户觉得「这类人会懂」。
- 信任避坑利好：替用户排雷、点名不适合人群、给真实缺点，降低买错踩雷的风险。
- 情绪利好：松弛、治愈、犒赏自己、自我认同、氛围感——把功能升华成「我值得」的满足。

做出差异化，至少占一条：

- 新角度：同一件东西换一个没人讲过的切口。
- 新卖点：挖出被忽略的隐藏价值，而非复述官方参数。
- 有反转：先抑后扬、预期之外，制造记忆点。
- 强好奇：开头留一个不看完会痒的悬念，但承诺要和正文一致。
- 营销融合：广告意图藏进真实体验和情绪里，先给价值再出现产品，让用户不排斥。

> 底线：利好至少一条落到用户身上、有场景支撑；差异点至少一条让人想停下来。只夸产品不讲用户得到什么，等于自嗨硬广。

### 6.6 原创表达校准与版权边界

完整创作、改写、润色、提升文采时，不读取博主高赞笔记原文，不建立样本仿写链路，不复用他人的标题、句子、段落、故事桥段、独特比喻、真实经历、品牌专属表达或评论区话术。可使用的只有：

- 用户提供的素材、体验、产品信息、图片、限制和口吻要求。
- 联网核验得到的公开事实、官方信息、权威资料、热点趋势和平台语境。
- 本方法库中的通用结构、标题方向、正文骨架、关键词逻辑、互动方式和合规原则。
- 对品类的抽象经验：用户常见痛点、决策因素、使用场景、内容形式和风险边界。

原创表达校准流程：

1. 先列素材池：哪些事实已确认，哪些体验来自用户，哪些只能写成建议或待确认。
2. 再定表达目标：本篇更需要“真实复盘、干货清单、测评对比、情绪共鸣、求助互动”中的哪一种。
3. 再写原创句子：每个核心判断都用本篇素材支撑，避免套用现成语气和固定爆款句。
4. 最后做版权与相似度自检：如果某段像“见过的高赞笔记”，即使没有逐字重复，也要改成基于本篇素材的新表达。

> 反抄袭硬规则：输出不得让人感觉是在改写某位博主的原文；缺少亲历素材时，主动降级为信息整理、选购建议、测评框架、清单攻略或求助帖。

## 七、模块 E · 互动与关键词（撬动双流量池）

### 7.1 撬动推荐流量（互动设计）

发布前就要设计评论区，用「钩子」预埋讨论：

- 争议型：「第 X 个你同意吗？」
- 情况型：「告诉我你的预算 / 肤质 / 城市，我帮你缩小范围」。
- 补充型：「还有哪些坑？评论区补充」。
- 选择型：「A 和 B 你会选哪个？」
- 资源型：「要清单 / 模板，评论关键词」。

> 互动靠真问题、真共鸣，不靠虚假福利、虚假稀缺、夸大恐惧或诱导性承诺；福利引导忌生硬求赞收藏。

### 7.2 撬动搜索流量（关键词布局）

位置三层布局：标题放行业核心大词（持妆粉底液）→ 正文首段自然复现核心词、小标题填精准长尾词（混油皮持妆粉底液全天实测）→ 标签拓展细分衍生词（学生平价持妆粉底液）；图片文字放最容易被搜的表达。

关键词四词性（组合使用）：品类词（通勤穿搭 / 敏感肌护肤）、场景词（春夏通勤 / 小户型收纳）、痛点词（显腿粗 / 卡粉 / 预算有限）、意图词（测评 / 避坑 / 教程 / 平替 / 攻略）。

## 八、模块 F · 账号层（打造优质账号的底层逻辑）

优质账号都有稳定的「内容记忆点」：赛道明确（先窄后深）、用户明确、价值明确、语气稳定、视觉稳定、栏目稳定。

定位避免太泛，要切到一类具体的人：

- 「分享生活好物」太泛，「租房小厨房低预算改造」更好。
- 「美妆穿搭博主」太泛，「小个子通勤女生穿搭」更好。
- 「记录日常」太泛，「敏感肌真实空瓶复盘」更好。
- 「职场成长干货」太泛，「新手运营的小红书增长拆解」更好。

内容矩阵五维度（以「职场形象账号」为例）：

- 专业主线：通勤穿搭、面试妆容、职场礼仪——建立信任。
- 场景细分：小个子、微胖、预算 300、夏天通勤、见客户——把同主题拆给不同人群。
- 真人信任：一周通勤实拍、踩雷单品、真实改造前后。
- 社区互动：帮粉丝改穿搭、评论区选题、投票。
- 热点借势：开工季、毕业季、年终述职、AI 职场焦虑——把热点翻译进自己赛道。

放大法则：七分使用成熟内容结构 + 三分做出人群、场景、观点或证据的微创新（「全城通勤穿搭」细化为「微胖女生通勤穿搭」）；用平台趋势和用户反馈判断方向，但不复制榜单笔记的原文、叙事和私有表达；「1 主号 + 3 场景号 + 6 素人号」矩阵联动放大互动。

## 九、模块 G · 笔记诊断（改写 / 优化任务专用）

接到「帮我看看为什么没流量 / 帮我优化」时，按六环逐项体检，定位最弱的 1–2 环再动刀，而非全篇推翻：

- 选题：解决了谁的什么问题？常见病灶是自嗨、无人关心、太宽泛。
- 标题：3 秒内给出点开理由了吗？常见病灶是平、绕、关键词靠后、承诺与正文不符。
- 钩子：前 3 行能不能勾住？常见病灶是铺垫太长、开头寒暄。
- 正文：有没有具体细节、画面与证据？常见病灶是流水账、空夸、只有结论无证据、产品出现太早像硬广。
- 关键词：标题 / 正文 / 标签是否覆盖搜索词？常见病灶是全是形容词、无长尾、标签乱堆稀释意图。
- 互动：给了评论 / 收藏的理由吗？常见病灶是结尾干硬、无互动钩。

最常见的失败根因：选题太大没人知道写给谁、封面信息太多一眼看不懂、标题与正文不一致、全文只写「我喜欢」没有用户决策价值、热点硬蹭与需求无关、太像 AI 缺真实细节与个人取舍。

> 输出诊断时：先点问题（哪环弱、为什么），再给改后版本，让用户看见「改了什么、为什么改」。

## 十、示例学习（理解「人感」的底层逻辑）

以下示例用于理解「什么叫真人感」，不要模仿文风或结构，要提取底层逻辑应用到新创作中。

示例 1（热情洋溢的实用派）——家庭版烧烤制作教程：用「天一热就馋」建立共鸣开场，用「嘎嘎香」「神仙酱汁」等带强烈个人情绪的词汇，让教程充满生活气息和个人热情，工具性内容也可以包装得活色生香。

示例 2（细节满满的种草机）——推荐一家韩式烤肉店：不泛泛说「好吃」，而是描述「阿姨会贴心提醒烤好了」「海鲜豆腐汤有奶油口感」等具体到毛孔的细节。细节是信任的基石。

示例 3（深度自我的意识流）——大理旅游笔记：不是旅游攻略，更像公开日记，充满「无目的的游玩变得乏味」等真实内心思考和转折。不完美、自省、充满个人哲思的真实感，有时比完美攻略更动人。

示例 4（指令遵循的最高级理解）——用户问「有什么推荐吗？」：当用户的话本身是求助时，扮演用户写一篇求助帖，把困境和问题清晰抛给社区引导互动，而不是去回答这个问题。

总结原则：不要照抄示例的「皮」，要理解并吸收它们「真」的「骨」。根据每次不同的需求，灵活运用这些原则，创造出独一无二但同样充满人感的笔记。

## 十一、社区发布规范（合规红线，每篇交付前必过）

> 这是合规底线，不是创作技巧。内容再优质，触线也会被限流、折叠、删帖甚至封号。以平台《社区公约》《社区规范》及品牌合作报备规则为依据；细则会更新，存疑时以「不夸大、不误导、可证实、不擦边」为通用准绳。

**1. 真实诚信（最高优先级）**
- 不编造未发生的体验、不伪造身份 / 资质、不 P 假数据 / 假评价 / 假对比图。
- 拒绝绝对化与疗效化承诺：「根治 / 100% 有效 / 永久 / 立刻见效 / 包瘦包白 / 无副作用」一律不用。
- 功效、价格、销量、排名等可核验信息须有据可依，不写「全网最低」「销量第一」「国家级」等无法举证的极限词。
- 不诱导互动作弊（刷赞刷评刷收藏）、不诱导站外私下成交。

**2. 广告与利益关系透明**
- 恰饭 / 合作 / 赠送 / 返现等商业内容，按平台「品牌合作」机制如实报备与标识，不把硬广伪装成素人分享。
- 商品功效、成分、适用人群如实描述，不暗示「人人适用」「替代专业方案」。
- 标价、折扣、赠品须真实有效，不做虚构原价的「先涨后降」。

**3. 行业敏感红线（高频踩雷区）**
- 医美 / 医疗 / 保健：不做诊疗承诺、不写「治愈 / 根治 / 替代就医」、不推荐处方药与器械违规使用。
- 美妆 / 护肤：不宣称需特殊资质的药用功效（祛斑 / 抗炎 / 生发等），不用「医生同款」误导。
- 食品 / 减肥：不宣传减肥疗效、不暗示「吃了能瘦 / 能治病」，不夸大成分作用。
- 金融 / 投资：不荐股、不承诺收益、不诱导借贷与高风险理财。
- 教育 / 考证：不承诺「包过 / 保录 / 内部名额」，不伪造录取与提分数据。
- 母婴：不替代医嘱、不传播无依据的喂养 / 用药建议。

**4. 价值观与内容底线**
- 不涉黄赌毒、不暴力血腥、不教唆违法、不售卖违禁品。
- 不软色情、不性暗示、不博出位的低俗擦边。
- 不盗图盗文盗视频、不搬运他人原创、不冒用他人肖像 / 商标 / 品牌。
- 不泄露他人姓名、住址、电话、车牌、单位等隐私；未经同意不发可识别人物的私密影像。
- 不制造地域 / 性别 / 职业对立，不传播偏见与仇恨；
- 不用政治/国旗/党徽敏感、灾难事故、争议社会事件。

**5. 未成年人保护**
- 不诱导未成年人不理性消费，不鼓吹容貌焦虑 / 过度节食等不当审美。
- 涉及儿童内容注意隐私安全，不暴露就读学校、家庭住址等。

**6. 表达与导流**
- 极限词避雷：最 / 第一 / 顶级 / 国家级 / 独家 / 唯一 / 史上 慎用或不用。
- 不在正文 / 评论硬留微信、QQ、电话、外链等站外联系方式（按平台规则操作）。
- 标签真实相关，不蹭无关热门标签骗流量、不堆砌违规敏感词。

**交付前强制三问：**
1. 真不真——有没有编造、夸大、伪造数据或体验？
2. 透不透——商业利益关系是否如实、是否把硬广伪装成素人？
3. 触不触线——是否涉行业敏感承诺、极限词、擦边、侵权、隐私或危险热点？

> 任一问命中，先修正再交付。若用户需求本身违规（虚假宣传、刷量话术、医疗疗效承诺等），礼貌拒绝并说明合规原因，给出合规替代写法，而非直接照做。
