# 图片设计方法库

> 定位：本文件承接 `xhs-note-proposal.samples.image-cover-methodology.md` 的主工作流，负责把已经确定的图片任务落成具体设计。这里不重复判断“要不要出图、出几张、图从哪来”，只回答“整组图如何建立统一视觉系统、封面怎么抓点击、内页怎么承接正文、每张图用哪类画面方法、怎样转成 prompt 或版式执行”。

## 目录

- 一、设计方法工作流：从分镜到单张执行
- 二、视觉基准：七维拆解、视觉质检底线与设计原子
- 三、封面设计：封面原则、封面文案与爆款封面方法论
- 四、整组内页设计：内页与封面的关系、递进式/单点展开式
- 五、按画面风格选择专业方法：插画、摄影、手绘注解层、配色、排版
- 六、Prompt 转换：九要素填槽与模板

---

## 一、设计方法工作流

主工作流已经给出分镜清单：`序号 -> 图片任务 -> 主文案/主体 -> 素材来源 -> 设计方法`。进入本方法库后，按下面顺序把“图片任务”落成可执行设计：

1. **接收设计需求**：逐张确认图片任务、商业/非商业属性、素材来源、主文案/主体、是否包含零容错信息。先标记真实素材页、证据页、AI 生成页、版式文字页，避免后续混用方法。
2. **定整组视觉系统**：先读「视觉基准」，用七维拆解对标图，提炼一张轻量视觉卡：主色/点缀色、字体气质、背景质感、构图网格、关键词高亮方式、去 AI 味要求。同组图共用这张视觉卡。
3. **先封面定调**：封面决定整组点击气质。先按「封面设计」写封面文案，再从「爆款封面方法论」选一个主封面类型；不要同时堆多个类型。
4. **再排内页逻辑**：按「整组内页设计」判断内页与封面的关系，选择递进式、单点展开式或混合式；每张内页只承接一个正文要点。
5. **按单张任务选专业方法**：每张图再判断是实拍质感、实拍底图再设计、插画示意、版式文字、图表 / 清单、真实 UI / 截图、产品 / 人物 / 场景。对应读取「按画面风格选择专业方法」，把具体摄影、手绘注解、插画、配色、排版特征写进执行说明。
6. **转成 prompt 或版式指令**：AI 生成页用「Prompt 转换」九要素填槽；真实素材页写裁切、调色、排字、遮挡/打码要求；零容错文字、品牌、价格、参数优先后期排字或真实素材，不交给模型自由生成。
7. **回到主工作流自检**：生成或排版完成后，回到主工作流的「生成前后自检闭环」「输出与交付检查」逐张检查；有水印、乱码、规格文字、假 UI、畸变或文字错误时重排/重出。

## 二、视觉基准

### 2.1 图片七维拆解原则

无论是动手前搜爆款封面、还是出图后自检自己的封面，都按这七个维度逐项过一遍——七维齐了，封面才「长得像优质/爆款封面」，而不是一眼 AI：

- **主题文案**：主标题醒目、辅文简洁、层级清晰——主标题一眼抓住核心钩子，辅助说明退居其次，绝不平均用力、不堆满。
- **视觉主体**：核心图要抓眼且紧贴主题——主体（人 / 产品 / 成品 / 场景）占画面绝对重心，和文案讲同一件事，不跑题、不喧宾夺主。
- **色彩搭配**：主辅色统一、对比适中、烘托氛围——固定一套主色 + 1 个点缀色，明度 / 饱和拉开层次但不刺眼，颜色为内容情绪服务。
- **排版布局**：留白合理、元素有序、视觉流顺畅——文字与主体各居其位，留白引导视线，阅读路径自上而下 / 自左而右一气呵成。
- **字体规范**：标题用粗体醒目、与主视觉强关联、有辨识度但易识别；注解 / 副文用纤细字号弱化拉开层级；全图字体 ≤ 2–3 种，字号 / 行距统一成体系，忌花体扭曲牺牲辨识度。
- **辅助元素**：logo / 边框 / 贴纸 / 手写涂鸦 / 装饰只作点缀、不抢主体——能强化风格记忆点即可，宁少勿多，杜绝杂乱堆砌。
- **核心信息**：时间 / 地点 / 人群 / 诉求等关键信息完整且直观——攻略类标清路线 / 预算 / 天数，测评类标清品类 / 人群，让人不点开也能 get 重点。

> 使用方式：找 1–3 篇同品类高赞封面，按七维提炼共性作本篇视觉基调；出图后再按七维逐项自检，缺哪维补哪维。

---

### 2.2 视觉质检底线

本节定义“像平台真实优质图”的最低标准。字体、配色、构图、载体、封面类型、插画/摄影/排版技法和 prompt 写法，分别落到 2.3、3.4、五、六中执行。

- **文字层级清楚**：主标题必须一眼可读，副标题只补充人群、场景、结果、数量或方法；不把正文搬上图，不让所有文字同等粗大。
- **字体服务内容气质**：字形、粗细、描边和手写感要匹配正文情绪与品类信任感；不能用默认系统字、花体乱码、硬描边或与内容相冲突的字体。
- **只强调少数关键词**：高亮块、下划线、圈注、箭头只服务 1-2 个核心词；没有明确指向的装饰全部删掉。
- **色彩有主次**：先有主色、文字色、点缀色，再谈氛围；避免无理由彩虹色、塑料渐变、全图高饱和或黑白底硬堆字。
- **真实信息用真实载体**：聊天、备忘录、软件界面、品牌、包装、价格、菜单、证书、地点等高风险信息，优先真截图、真实素材或可控模板；不让 AI 凭空画证据。
- **实拍保持真实质感**：人物、产品、美食、场景要有自然光影、真实材质、合理阴影和轻微生活痕迹；避免悬浮、过度磨皮、塑料高光和过度完美。
- **画面干净高清**：3:4 竖版优先，1080x1440 起步；无模糊、压缩色块、水印、平台规格文字、用户名、二维码、乱码英文、假 logo 或模型残留。

> 用法：出图前把这些作为 prompt 与版式策略的红线；出图后按这七项逐张检查。具体怎么做到，回到 2.3 选设计原子，3.4 选封面类型，五选择画面技法，六转成 prompt。

---

### 2.3 优质封面设计原子

建立封面视觉系统时，先拆成可复用的设计原子。

- **主视觉标题**：一句最强钩子占最大视觉权重。可用超粗黑体、手写大字、衬线大字或海报字；字形要服务情绪，不能只靠放大。
- **副标题 / 次级说明**：只补充人群、场景、结果、时间、数量、方法，不重复主标题。常见位置是标题旁、底部横条、角标、便签、气泡或小标签。
- **品牌 / 身份标识**：只有真实授权品牌、真实栏目、真实账号资产才可出现；没有授权时用中性身份词、品类词或栏目感标签替代。
- **视觉图形载体**：先定主体是人物、产品、菜品、场景、实拍拼贴、手绘插画、纸张便签、信息图还是纯文字。载体决定封面第一眼是“可信”“好看”“有用”还是“有情绪”。
- **色彩系统**：先确定底色、主文字色、点缀色三件事。常见有效组合是浅底 + 黑字 + 亮色强调、实拍底 + 白字/黄字高亮、深底 + 少量高亮字；避免无理由彩虹色和塑料渐变。
- **版式构图**：先搭骨架再放元素。可选大标题压顶、主体居中、底部利益条、四宫格拼贴、纸卡叠放、菜单清单、海报留白、对角动线、左右对比等；每种骨架只服务一个阅读路径。
- **字体规范**：全图控制在 1 个主字体 + 1 个辅字体，最多 3 个字号层级。主标题负责截停，副文负责解释，小标签负责补充；不要所有文字都同等粗大。
- **细节辅助元素**：白描边、撕纸边、胶带、回形针、手写箭头、圈注、波浪线、勾选框、贴纸、小图标、手写注解只用来指路、强调或补充感受，不承担主信息。
- **真实质感**：真人、穿搭、探店、美食、产品优先保留真实摄影的光线、阴影、材质和轻微生活痕迹。AI 生成时也要复刻“真实图再设计”的感觉，而不是干净到像模板。
- **密度控制**：干货、攻略、合集可以高密度，但必须有清晰分组；情绪、观点、高级感、摄影价值型封面要低密度，留白本身就是设计。

> 使用方式：先用这些要点对标 1-3 张同赛道或平台爆款高赞图，再把共性写进本篇视觉卡；如果共性不明确，宁可降低设计复杂度，也不要拼贴一堆无关元素。

## 三、封面设计

### 3.1 封面：第二个标题

信息流里用户先看到图、再看到字。封面只有一个任务：让人 1 秒判断「这和我有关，值得点开」。

### 3.2 封面八项原则

#### 3.2.1 尺寸合规原则（基础底线，决定展示面积）
1. **首选3:4竖版**，推荐像素：1080×1440px；高清版 1242×1660px；信息流占屏比最大，比横版4:3多40%可视面积，点击率更高；次选1:1方图，4:3横图仅风景对比使用，不做主封面。
2. **安全区留白**：顶部200px会被账号头像遮挡，底部300px被笔记标题/互动按钮覆盖；核心人物、大字标题、产品放在画面中间60%区域，四边留6%-8%空白，避免裁切失效。
3. **高清无水印**：画质清晰、无模糊压缩，禁止第三方水印、杂乱贴纸，系统更易推荐；导出JPG/PNG。

#### 3.2.2 视觉抓眼0.3秒原则（信息流第一竞争力）
用户划屏平均3秒/条，封面必须瞬间跳出页面：

1. **强​​0.3 秒法则​**​：封面必须在 0.3 秒内传达核心价值，用大字标题、对比反差或人脸产品快速抓住注意力。
2. **主体绝对突出**：核心元素（人脸 / 产品 / 成品）占画面 70% 以上，居中构图；画面元素不超过 5 个，杜绝杂乱堆砌。
3. **情绪 / 效果钩子**：人物生动表情、前后对比、成品高光特写（美食光泽、妆容试色、改造反差），直观传递 “好看 / 有用 / 省钱”。


#### 3.2.3 信息极简传递原则（一眼看懂笔记价值）
封面文字是点击开关，拒绝冗长：
1. **默认必带大标题**：封面默认带标题、尤其是一句醒目大标题——用户还没点进来，就要先被这句大字截停、被吸引、愿意停留。主标题 4-14 个字，加粗醒目，优先放痛点、数字、结果；例：3个减脂早餐、踩雷避坑、省500租房攻略、如何快速提高封面点击率。只有当主体画面本身已极强（高级感产品大片、强氛围场景、真人大片）、大字反而破坏质感时，才弱化为压一两个关键词，但仍保留可读的核心钩子。
2. **分层信息**：只保留1个主标题 + 1-2行副标题补充，不堆砌多段文字；字体统一、无描边杂乱，保证核心信息清晰可读。
3. **拒绝图文不符**：封面信息必须和正文内容匹配，过度夸大虚假、图文不符，会降低完读、拉低账号权重。

#### 3.2.4 构图聚焦原则（降低用户识别成本）
1. **爆款构图创意方法**
- 居中构图​​：主体放在画面正中央，稳定聚焦，适合单品特写或人物肖像，新手最不容易出错。
​- ​对角构图​​：图片和文案分别放在两个对角，制造动感和视觉冲击力，适合打造个性人设。
​- ​上下分割​​：将画面分成上下两个板块，上图下文或上文下图，层次分明且信息清晰。
​- ​包围式构图​​：用文字或元素环绕主体，包裹感强，能突出核心内容并营造氛围感。
- ​​散点构图​​：多张图片按一定规律分散排列，活泼有层次，适合多品类合集或生活类内容。
​​- C 字型排版​​：文字沿 C 型路径环绕主体，视觉有引导性，适合知识教程类干货内容。

#### 3.2.5 搜索适配原则（获取长尾搜索流量）
封面视觉+文字预埋关键词，用户搜词能匹配到你的笔记：
- 封面大字包含赛道关键词（美妆、美食、穿搭、租房、油皮底妆、露营穿搭、减脂餐）；
- 品类、场景、人群词直观展示，方便系统打精准标签。

#### 3.2.6 合规避坑原则（防止限流、违规）
1. 禁止低俗暴露、夸张医美对比、医疗功效、虚假夸大宣传；
2. 不出现二维码、微信、价格诱导、导流话术；
3. 避免过多大字遮挡主体、色彩刺眼刺眼、密集杂乱排版；
4. 不要直接搬运网图、他人实拍图，易判侵权限流。

#### 3.2.7 图文封面原则

封面是图文笔记的流量入口，直接决定作品点击率，核心要求为画面独立完整、信息表意清晰，无需点开内页就能让用户快速读懂内容核心主题；封面视觉优先级最高，是整张图文的核心视觉重心，第一眼需抓住用户眼球。
- 内页配图无需和封面争夺视觉焦点，整体色调、滤镜、排版、字体需和封面保持统一视觉调性，保证笔记整体风格连贯；
- 内页图片可差异化补充内容：封面主打整体效果、核心卖点、场景氛围感，内页侧重拆解细节、操作步骤、成品特写、对比实拍等延伸信息，丰富内容层次。

### 3.3 封面文案写法

写封面文案前，先做一个判断，再谈怎么写得抓眼球，最后让它和正文标题形成呼应。

**第一步：与正文标题形成呼应，不是简单复制**

- **承诺一致、措辞可异**：封面文案从正文标题 / 核心钩子提炼，可换更短更画面化的说法适配版面，但给用户的承诺必须和标题、正文对得上。
- **错位互补、信息不重复**：标题铺信息、封面给钩子，两者各补一半更高效——标题写全「敏感肌平价面霜测评，我试了 18 支」，封面就压最狠的那句「18 支只留 5 支」，而不是原样照搬标题。
- **关键词呼应**：封面与标题共用 1–2 个核心关键词（品类词 / 人群词 / 利好词），强化记忆点，也利于搜索流量承接。

**第二步：若要重写，就写得凝练抓眼球**

- 短、狠、具体：「什么穿搭一看就是海淀人」「试了 18 个只留 5 个」「通勤女生照着穿」「敏感肌先看这篇」「预算 200 也能抄」。
- 一张封面只讲一个核心钩子，主标题控制在一行能读完（建议 ≤14 字），多余的话拆成副标题小字或干脆删掉。
- 关键词加粗 + 亮色高亮，制造视觉重音；聚焦主标题，全部加粗等于没有重点。
- 善用「反问 / 共情句式 / 痛点句式 / 数字 / 身份指向 / 利好承诺」等气口起势：戳痛点（「闷痘肌别再瞎涂」）、给数字（「3 步搞定」）、点人群（「学生党码住」）、抛利好（「省一半钱」）。
- 可留悬念 / 制造好奇，但承诺必须和正文叙事保持一致，不做标题党式失真。

### 3.4 爆款封面方法论

按内容任务选择一种主封面类型，再调用 2.3 的设计原子补足标题、主体、色彩、构图、字体和细节。每张封面只设一个主视觉策略，其他元素只做辅助，避免多种类型互相抢戏。

#### 3.4.1 真人大字截停型

- **适用**：美妆护肤、保险科普、考试学习、装修避坑、职场干货、观点提醒等需要“真人站出来说一句重话”的内容。
- **视觉结构**：真人半身 / 上半身居中或偏一侧，常有手势（交叉拒绝、点赞、指向、拿电脑）；主标题压在顶部或上半屏，超大字号，常用白描边或阴影从实拍背景里跳出；底部放利益句 / 警告句 / 反问句；辅助元素用箭头、圈注、表情、勾选框、手绘线条增强情绪。
- **执行要点**：主标题必须短，通常 2-8 个字最强；人物轮廓可加粗白描边但不能塑料抠图；背景保留真实居家 / 办公 / 店铺场景，统一压暗或虚化，让文字可读；亮色只突出 1-2 个关键词。
- **慎用**：真实人物、OOTD、妆容上脸优先用户实拍；AI 生成人像只做泛化角色，不冒充真实本人或真实达人。

#### 3.4.2 纯文字大字报型

- **适用**：职场、求职、学习、观点、避坑、方法论、强情绪表达。
- **视觉结构**：文字是唯一主视觉，白底 / 浅色底 / 纯色底承载短句判断；主标题用超粗黑体、手写粗字或粗糙印刷字，关键词用亮黄、蓝、红、荧光绿等色块高亮；箭头、圈注、波浪线、感叹号、简笔图标增强“手做感”。
- **执行要点**：最多 3 个信息层级：主判断、补充说明、行动提示；每层字号差明显。可使用粗边框、白纸块、撕纸边、荧光色条，但不要把每个词都高亮。
- **慎用**：大字报不是把正文搬上封面。文案必须重新压缩成一句强判断或一个强疑问。

#### 3.4.3 纸卡便签 / 手写笔记型

- **适用**：冷知识、职场观察、情绪观点、清单总结、学习笔记、求职经验。
- **视觉结构**：米白纸、横线纸、便签、剪贴纸、胶带、回形针、手写字、下划线构成“私人笔记感”；主标题可像写在纸上，副文像笔记条目；插画或小图只做陪衬。
- **执行要点**：底色要像纸，不要纯白发光；字体用手写感但必须清晰；划线只划关键词；纸张投影、轻微倾斜、胶带压角能增强真实感。
- **慎用**：如果信息很多，优先分成清单卡片，不要挤成一坨手写长文。

#### 3.4.4 极简观点卡型

- **适用**：认知观点、品牌/营销/AI/职场类深度内容、情绪克制型笔记。
- **视觉结构**：大面积留白或深色纯底，主文字居中或偏中下，只有一处关键词高亮；装饰极少，如引号、短横线、细箭头、轻微纹理。
- **执行要点**：用少量文字建立权威感，字号层级克制，配色控制在 2-3 色；高亮只标一个词或一句关键短语；留白要足，靠秩序感而不是堆元素吸引人。
- **慎用**：不适合需要展示产品、美食、穿搭效果的内容；这类封面靠文案判断力，文案弱就不要用。

#### 3.4.5 插画情绪型

- **适用**：自媒体、亲子、心理、求职、轻松科普、情绪向干货。
- **视觉结构**：扁平手绘人物 / 动物 / 物件作为情绪载体，配大字标题；常用浅底、柔和撞色、粗线条、夸张表情、手绘箭头和小图标。
- **执行要点**：插画风格要统一，线条粗细、颗粒感、上色方式一致；角色只承接一种情绪，不要让插画抢走标题；适合做“低攻击性但高共鸣”的封面。
- **慎用**：职场硬核、金融法律、严肃商业内容不要过度可爱化，除非账号人设本来如此。

#### 3.4.6 旅行攻略拼贴型

- **适用**：城市攻略、景点路线、几天几夜、当地人推荐、打卡地图。
- **视觉结构**：多张真实地点图拼成杂志 / 地图感；主标题居中覆盖在大色块或白底贴纸上；地点标签、定位 pin、路线箭头、手写地名、印章、邮戳、胶片边等元素建立旅行感。
- **执行要点**：地点图必须真实可核验，不用 AI 冒充具体景点；主标题要压住信息，如“城市名 + 攻略/路线/深度游”；图片之间色温统一，标签只标关键地点。
- **慎用**：不要搬运他人游记原图；著名地标和真实地点优先真实照片或授权素材。

#### 3.4.7 穿搭合集拼贴型

- **适用**：OOTD、季节穿搭、通勤穿搭、约会穿搭、韩系/法式/学院风合集。
- **视觉结构**：多个人物全身图或单品图抠出，放在纸感 / 布纹 / 软色背景上；每套配编号、Look 标签、单品清单、手写箭头、胶带、拍立得、干花、咖啡等生活方式细节。
- **执行要点**：真人图比例统一，脚底对齐或按主次排列；主 look 最大，其他 look 环绕；文字标签小而清晰，不挡穿搭轮廓；整组滤镜和服装色系要统一。
- **慎用**：真人穿搭高度依赖实拍，AI 生成容易身体比例和服装结构失真；拿不到实拍时可改成单品平铺或中性搭配示意。

#### 3.4.8 单品平铺 / 产品质感型

- **适用**：穿搭单品、配饰、家居小物、好物推荐、产品质感展示。
- **视觉结构**：主体物件在干净材质背景上有序平铺，几乎不靠大字；光线、阴影、材质纹理是主要价值。可用少量品牌 / 品类标签，但不遮挡产品。
- **执行要点**：俯拍或平拍角度统一，主体之间留足呼吸感；背景材质可选地毯、水泥、木纹、纸面；如果是品牌产品，外观、logo、包装必须真实。
- **慎用**：不要用 AI 伪造具体品牌包、鞋、衣服、包装和 logo。

#### 3.4.9 探店 / 菜单拼贴型

- **适用**：美食探店、菜单推荐、店铺合集、人气榜、必吃清单。
- **视觉结构**：真实店内环境做背景，食物抠图或实拍菜品像贴纸一样分布；主标题像手写招牌或菜单标题；菜名、推荐理由、勾选框、白描边、手绘边框、菜单牌增强“点单参考”感。
- **执行要点**：食物图要真实、有食欲，白描边让菜品从背景里跳出；菜名短，最多补一句口感/卖点；同页菜品不宜太多，优先展示 3-8 个重点。
- **慎用**：具体门店、菜名、价格、菜单、招牌不能凭空造；用户没有素材时优先免版权实拍或改成非具体店铺的美食灵感图。

#### 3.4.10 食物实拍大片型

- **适用**：菜品本身就是内容价值的美食笔记、餐厅记录、食谱成品展示。
- **视觉结构**：主体食物占大画面，靠色泽、油光、蒸汽、餐具、桌面光线吸引；文字很少，可只放菜名、口感短句或手写标注。
- **执行要点**：优先使用真实食物照片；调色强调自然暖光、真实油光、清晰纹理；文字压在暗部或留白处，不遮挡主菜。
- **慎用**：AI 生成精致摆盘和食物质感容易假，尤其汤汁、肉纹、烘焙切面、拉花；确实生成时只做氛围示意，不当作真实成品证据。

#### 3.4.11 产品 / 食物广告拼贴型

- **适用**：新品、促销、好物种草、面包甜品、饮品、零食、产品卖点页。
- **视觉结构**：实物大图 + 高饱和色块 + 粗黑标题 + 英文 / 标签 / 贴纸 + 手绘箭头，整体接近海报广告；主体通常有白描边或荧光描边，卖点被压缩成短标签。
- **执行要点**：主产品最大，卖点标签围绕主体；色彩可以更强，但必须有主色统领；中英文字只做装饰时要少而准，不能出现乱码。
- **慎用**：商业产品必须核实品牌、包装、功效、价格；AI 不得生成假 logo、假包装、假促销信息。

#### 3.4.12 图表 / 结构化信息型

- **适用**：学习成长、金融科普、职场方法、生活决策、测评对比、步骤教程。
- **视觉结构**：标题给强判断，下面用清单、矩阵、表格、流程、勾选项、编号承载信息；配色克制，信息分组清楚，读者一眼知道“可收藏”。
- **执行要点**：先画信息结构，再做视觉；每块信息只讲一个点，编号和图标统一；重点词用单一亮色，不要每个格子不同色。
- **慎用**：图表里的数据、价格、时间、比例、结论需要事实依据，不能为了好看编造。

#### 3.4.13 真实 UI / 对话截图型

- **适用**：聊天故事、备忘录清单、软件教程、社群问答、求助/复盘类内容。
- **视觉结构**：靠真实系统界面建立可信感；备忘录、聊天气泡、状态栏、按钮、头像、时间戳等细节必须像真实截图。
- **执行要点**：优先真机截图或可控模板；隐私打码，昵称和头像中性化；文案短，气泡节奏自然，有转折或结论。
- **慎用**：不要让 AI 凭空画微信、iOS、软件界面，容易出现假按钮、错位图标、乱码和系统栏错误。

#### 3.4.14 猎奇创意 / 反差型

- **适用**：娱乐、冷知识、观点冲突、强反差标题、热点借势。
- **视觉结构**：用夸张字形、反常识画面、符号化道具、强对比色制造停顿；信息通常只保留一个冲突点。
- **执行要点**：创意要服务正文钩子，不能为了怪而怪；标题承诺必须能在正文兑现。
- **慎用**：不低俗、不擦边，不使用恐吓式医疗/金融/法律暗示。

> 类型选择：真人说重话走真人大字；观点强判断走纯文字 / 极简卡；清单经验走纸卡便签；城市路线走旅行拼贴；穿搭合集走 Look 拼贴；单品审美走平铺质感；探店菜单走菜品拼贴；菜品本身好看走食物实拍；卖点种草走广告拼贴；硬核干货走图表结构；聊天 / 备忘录走真实 UI。手绘注解、描边、箭头、圈注属于可叠加的设计层，按「五、按画面风格选择专业方法」调用，不作为独立封面类型。混合内容以封面主诉求定一个主类型，其余只作辅助。

## 四、整组内页设计

封面之外的 辅图/配图 是把正文核心要点逐张视觉化，让用户滑动时跟着节奏走。

### 4.1 先判断：辅图/配图 与封面的关系（出图前必做）

动手排内页前，先想清楚整组图是「同一套视觉系统」，再决定内容怎么铺开：

**第一问：辅图/配图要不要和封面排版风格一致?**

- **默认一致（强烈建议）**：同一篇笔记的内页与封面共用一套视觉语言——同色调、同字体气质、同版式骨架（文字位置、对齐方式、边距）、同滤镜调色。一致才像「一个系列」，滑动时不割裂，也强化账号记忆点。
- **可适度变化、但不破系统**：辅图/配图 可以比封面信息更密、留白更少（封面要钩人靠留白，内页要讲透靠信息量），但变化只发生在「信息密度与版式角色」层面，色调 / 字体 / 边距网格这套底层不能换。
- **何时允许不同**：辅图/配图 里嵌入真实截图、实拍图、前后对比图时，这些「证据页」可以跳出版式卡片直接铺满——真实感优先，但仍用统一的标题条 / 角标 / 配色把它收进系列里。

**第二问：辅图/配图怎么铺开——递进式 还是 单点展开式?**

按正文结构选一种主线，别混着排导致逻辑乱：

- **递进式（多要点并列 / 步骤流程）**：每张配图讲一个独立要点，张张平行推进，像翻一本目录清晰的小册子。适合清单合集、教程步骤、横向测评、避坑要点——用统一卡片版式 + 编号（No.1–No.N / Step1–StepN）强化「跟着走」的节奏。
- **单点展开式（一个核心论点层层深入）**：围绕一个主题逐张深挖——是什么→为什么→怎么做→效果证据→结论。适合单品深度测评、一个方法的完整拆解、一篇观点的论证。版式上可用「主线索贯穿 + 每张递进一层」，靠内容深度而非数量取胜。
- **混合式**：长内容可先单点展开立论，再用递进式铺要点清单收尾；以主诉求定主线，另一种作穿插。

## 五、按画面风格选择专业方法

生图不是一句「小红书风格」就完事。先回到本张图片的任务和封面类型，再判断它需要哪种画面风格，最后套对应的真实作图业务方法论，让 AI 复刻「专业的人会怎么做」，而不是凭空发挥。常见五类：

- **插画 / 手绘 / 卡通风**：套优质插画方法论。明确画风流派（扁平插画 / 厚涂 / 线稿淡彩 / 治愈系 / 国潮 / 像素风等），写清统一线条质感、统一上色方式、统一光影逻辑；角色 / 物体造型简洁有记忆点，留白透气；同组多张锁定同一画师风格关键词，保证系列统一。适合知识科普、情绪向、教程示意、抽象概念可视化。
- **真实人物 / 场景 / 产品图**：套专业摄影拍摄方法论。写清机位与景别（特写 / 半身 / 全身 / 俯拍 / 平拍）、光线（自然光 / 柔光 / 侧逆光，忌生硬塑料光）、景深与焦段（大光圈虚化背景突出主体）、真实质感与生活痕迹；产品图讲究主体居中、背景干净、质感反光真实。让画面像「专业摄影师实拍」，而非渲染图。强依赖真拍的（美食大片、OOTD 全身）优先回到真实 / 免版权实拍。
- **手绘注解层 / 涂鸦标注层**：这不是一种固定封面模板，而是叠加在不同封面类型上的设计方法。适用于真实 / 免版权实拍、拼贴、攻略、穿搭、探店、好物、教程等画面，当主体已经真实清楚但缺少笔记感、信息指向或情绪温度时使用。先判断这张图需要“解释什么”或“强调什么”，再选择描边、箭头、虚线、圈注、短句注解、少量闪光 / 爱心 / 热气等元素。注解必须指向真实可见对象或真实信息点，语气和品类匹配：美食写口感和香气，穿搭写版型和搭配点，好物写材质和使用感，攻略写路线和避坑点，空间写氛围和光线。它的任务是建立视线引导和人感，不是把画面装饰满。
- **色彩搭配**：参考专业配色体系——可借鉴经典名画、知名电影调色、优质海报 / 品牌视觉的配色方案（如莫兰迪低饱和、孟菲斯撞色、电影青橙调、名画的古典色盘）。给 AI 写明「参考 XX 配色：主色 + 辅色 + 点缀色」的具体色彩关系，而不是笼统「好看的颜色」；同组图共用一套色卡保持调性统一。
- **排版布局**：学习参考优质海报 / 杂志封面的排版语言——网格对齐、主次分明的字号层级、留白节奏、视觉引导线、信息分组。版式文字型封面（大字报 / 图表 / 三明治）尤其要照海报级排版结构生成：标题区 / 信息区 / 点缀区各就各位，对齐统一，专业不业余。

> 策略口诀：先定图片任务和封面类型，再定风格（插画 / 实拍 / 版式），最后判断是否需要叠加手绘注解层、配色法或海报排版法。注解层只服务信息和情绪，不替代封面主类型。

## 六、Prompt 转换（封面类型 -> 设计原子 -> 可执行提示词）

文生图 prompt 不是把九要素机械塞满，而是把前面已经确定的“图片任务、封面类型、设计原子、素材来源”翻译成模型可执行的视觉指令。优先保证画面结构、真实质感和可后期排版空间；精准中文、品牌名、价格、参数等零容错内容，优先后期排字或使用真实素材。

### 6.1 Prompt 生成顺序

1. **先定执行方式**：判断本张图是完整文生图、真实素材再设计、AI 背景 + 后期排字、还是模板排版；再判断是否需要叠加手绘注解层。需要精准中文、真实 UI、品牌 / 价格 / 参数时，不让模型自由生成文字。
2. **选封面类型**：从「爆款封面方法论」选 1 个主类型，如真人大字截停、纯文字大字报、旅行拼贴、穿搭合集、探店菜单、食物实拍、图表结构等。
3. **填设计原子**：明确主视觉标题、副标题/次级说明、视觉图形载体、色彩系统、版式构图、字体规范、细节辅助元素。
4. **写画面主体**：主体必须具体到人 / 物 / 场景 / 食物 / 纸卡 / 拼贴 / 图表，而不是只写“小红书风格”。
5. **规划可叠加设计层**：如需手绘注解 / 涂鸦标注，先写清 `对象 -> 注解角度 -> 短句 -> 指向方式 -> 摆放位置`，再放进 prompt；不需要时不要硬加。
6. **写可控边界**：只出现指定文字；不出现平台规格、尺寸标注、页眉角标、用户名、水印、二维码、logo 乱码、假英文、模型签名。尺寸、比例、分辨率只放工具参数，不进入画面描述。
7. **写后期要求**：如果需要排中文，写清“预留文字区 / 后期叠字 / 用指定文案替换占位”，不要依赖模型长文本一次性生成正确。

### 6.2 九要素填槽

使用手绘注解 / 涂鸦标注层时，不必改动主封面类型，只在 ⑨ 里补充设计层规划。推荐先写成：`对象 -> 注解角度 -> 短句 -> 指向方式 -> 摆放位置`。例如：`饮品 -> 冰感 -> 冰冰的 -> 瓶身描边 + 短箭头 -> 左上留白`。

| # | 要素 | 填什么 | 取料来源 |
|---|---|---|---|
| ① | 主视觉标题 | 一句核心钩子，决定最大视觉权重；可要求预留标题区或只生成短字 | 标题 / 核心钩子 |
| ② | 次级说明文案 | 人群、场景、结果、数量、方法；可作为后期排字 | 正文利好 / 适用人群 |

| ⑥ | 色彩系统 | 底色、主文字色、点缀色；全图 2-4 色为主 | 对标视觉 / 品类调性 |

### 6.3 参数与通用尾缀

尺寸与比例只在生成工具参数中设置：`3:4 竖版，建议 1080x1440 或更高清`。不要把“小红书 3:4 / 3:4 / 1080x1440 / 竖版封面 / cover / poster”等规格词写进画面 prompt。

每条画面 prompt 都加：`画面干净，只出现指定文案；无多余水印、logo、用户名、二维码、引用出处、尺寸标注、平台字样、风格标签、页眉角标、模型签名、乱码英文、假 UI 文本；顶部和底部边缘无任何非内容所需小字；图上中文用词准确、字形清晰、无错别字、无乱码；高分辨率，文字边缘干净，主体不畸变。`

### 6.5 文字与后期排版规则

- **短字可生成，长字后期排**：主标题、短标签可以让模型尝试；长句、品牌名、价格、参数、地点、UI 文本、表格数据优先后期排字。
- **占位比乱码好**：需要精准文字但模型不稳时，先生成干净背景、主体和留白区，再用指定文字排版覆盖。
- **同组图复用视觉卡**：同一篇的主色、点缀色、字体气质、边距、描边、标签样式、手写元素要复用，避免每张都像不同账号。
- **反向检查 prompt**：如果画面描述里出现“小红书 3:4”“3:4”“1080x1440”“竖版封面”“cover/poster”等规格词，先移到工具参数区；不要只补一句“禁止入画”。如果写了品牌 / 价格 / 地点，必须确认来自真实素材。

> 红线：不编造数据、参数、价格、品牌信息；不伪造他人肖像、商标、菜单、证书、截图；不搬运他人笔记图、网图或不明来源图；不生成擦边、低俗、敏感画面；真实信息类以真实素材为准，不用 AI 凭空捏造「证据图」。
