# 创意手册

新写分镜、重写、大幅优化，或用户要求“更抓人、更有创意、更像平台原生内容”时读取本文件。

## 创意流程

正式写表格前，内部先发散 2-3 个真正不同的方向：真实生活痛点版、反常识实验版、情绪故事版、社交关系版、视觉奇观点版、平台原生版。

选择最终方向时检查：是否贴合用户目标、第一镜是否有观看理由、核心信息是否能拍成动作、拍摄成本是否可控、是否符合目标平台气质。

最终组合：`钩子策略 + 叙事结构 + 情绪弧线 + 视觉证明 + 镜头衔接 + 平台语感`

## 开场钩子

| 钩子 | 用法 | 避免 |
| --- | --- | --- |
| 反常动作 | 出乎意料但相关的动作 | 与主题无关的猎奇 |
| 痛点特写 | 具体烦躁、尴尬、紧迫或犹豫瞬间 | 泛泛问“你有没有遇到过” |
| 强对比 | 前后、左右、旧新、错误/正确 | 只说优点，没有画面对比 |
| 生活冲突 | 迟到、低电量、包太乱、文档空白等压力 | 慢吞吞介绍背景 |
| 反常识设问 | 挑战常见习惯或误区 | “你想知道它好不好用吗” |
| 悬念切入 | 只露出部分信息，通过动作揭晓 | 开头把结论全说完 |
| POV 第一视角 | 让观众进入行动现场 | 一上来中景主播站着说 |
| 声音钩子 | 用有意义的声音打开注意力 | 从头到尾无变化的泛用 BGM |

时间规则：15s 前 2-3s 出钩子；30s 前 3-5s 出钩子；60s+ 前 5s 出钩子，并在 20-25s 处放次钩子或转折。

## 叙事结构

- 痛点/冲突 -> 发现 -> 展示/验证 -> 结果/记忆点。
- 场景进入 -> 问题出现 -> 使用动作 -> 前后对比 -> 收束。
- 观点抛出 -> 生活例子 -> 反差证明 -> 态度落点。
- 结果前置 -> 倒回过程 -> 揭示原因 -> 行动引导。

## 平台语感

| 平台 | 内容气质 | 镜头策略 | 台词/花字 |
| --- | --- | --- | --- |
| 抖音 | 强钩子、快节奏、反差 | 多特写，变化快，前置冲突 | 短句、强花字、少解释 |
| 快手 | 真实、直接、人情味 | 生活场景和人物关系优先 | 口语直接，少广告腔 |
| 小红书 | 真实体验、审美细节 | 暖光、桌面/居家/通勤细节 | 像分享经验，不像硬推 |
| 视频号 | 完整叙事、情绪价值 | 铺垫完整，情绪落点稳 | 句子可稍完整，有观点 |
| B站 | 过程感、解释感、轻梗 | 保留过程逻辑，可用屏幕录制 | 解释可稍长，但要有节奏 |

## 镜头衔接

使用动作承接、视线承接、道具承接、声音承接、景别递进、景别反差、光色承接或自然遮挡转场。避免无意义跳切、连续三个同机位镜头、前后镜头没有任何连接。

## 反平庸检查

- 第一镜去掉主题关键词后，还会有人想看吗？
- 第一镜是不是只是主播说话或产品静置？
- 有没有一个相关但没预料到的镜头？
- 核心卖点是否通过动作被看见，而不只是台词说出来？
- 有没有真实生活细节？
- 镜头之间有没有动作、道具、声音、视线、色彩或花字承接？

## 节奏速查

| 总时长 | 建议镜头数 | 单镜时长 | 典型结构 |
| --- | --- | --- | --- |
| 15s | 4-6 | 2-3s | 钩子 -> 核心证明 -> 收尾 |
| 30s | 7-10 | 3-4s | 钩子/冲突 -> 展示/验证 -> 转折/收尾 |
| 60s | 12-18 | 3-5s | 钩子 -> 问题 -> 展示 -> 次钩子 -> 收束 |

无台词时写 `（无台词）`。
