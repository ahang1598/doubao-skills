## 1. 内容要求

完整公众号文章应整理成结构完整、排版清楚、方便复用的公众号写作分区，包含正文、封面图、备选标题和内容验真。

文档的核心目标是：让用户一眼看懂“文章写给谁、核心选题是什么、主要观点是什么、结构如何展开、最终成品是什么、发布前还要检查什么”。

<a id="toc-2"></a>
## 2. 格式排版要求

公众号文章排版应服务内容气质和阅读体验。生成完整公众号文章或公众号文档时，模型应根据文章类型、账号定位、目标读者和传播目的，自动选择最匹配的排版风格，而不是默认输出普通纯文本。

除非用户明确指定排版风格，否则模型应在内部完成判断，并让标题、小标题、段落、配图建议、重点句和结尾引导保持同一种视觉逻辑。若内容风格不明确，默认优先选择“极简留白风”或“现代杂志风”，保证文章干净、耐读、有发布感。

<a id="toc-3"></a>
### 2.1 通用排版原则

正文应适合手机端阅读，段落不宜过长，一段尽量只表达一个重点。

文章开头应快速进入主题，可以用顶图、导语、痛点场景、核心观点或反常识判断建立阅读兴趣。

小标题要有设计感和信息量，不能只写“第一部分”“第二部分”。中长文可以统一使用“01 标题”“02 标题”“03 标题”的形式，增强秩序感和视觉识别度。

文中可在合适位置加入配图建议，例如顶图、章节图、案例图、数据图、情绪氛围图，但配图必须服务内容，不要为了装饰强行插图。

重点句子可以用少量加粗或黄色高亮标出，但必须克制。高亮只用于核心观点、关键判断、金句、重要提醒或转化信息，不能满篇都是重点。

结尾应根据文章目标自然收束，可以是观点升华、留言互动、关注引导或购买引导。

<a id="toc-4"></a>
### 2.2 自动匹配的排版风格

<a id="toc-5"></a>
#### 2.2.1 现代杂志风

适合人物故事、社会观察、生活方式、品牌故事、城市文化、热点深度文和有质感的观点文章。

主要排版特色：

- 开头设置顶图，先建立视觉氛围。
- 标题和导语有杂志感，适合营造情绪和阅读期待。
- 小标题统一使用“01 xxx”“02 xxx”的编号形式。
- 正文中穿插与内容相关的配图、场景图或案例图。
- 重点句可使用黄色高亮或单独成段，增强记忆点。
- 文章整体要有节奏感，不能像资料堆砌。

当文章需要“有审美、有故事、有深度、有传播感”时，优先选择这种风格。

<a id="toc-6"></a>
#### 2.2.2 极简留白风

适合个人成长、职场思考、情绪共鸣、轻观点文、作者个人表达和咨询顾问类文章。

主要排版特色：

- 以白底、短段落、留白和自然节奏为主。
- 不使用过多装饰，重点靠文字本身打动读者。
- 小标题数量克制，可以不用编号，也可以使用简洁短句。
- 重点句单独成段，少量加粗即可。
- 配图不宜太多，可只保留顶图或少量氛围图。
- 结尾适合用一句有余味的判断或一个温和提问收束。

当文章重在“共鸣、信任、思考和耐读感”时，优先选择这种风格。

<a id="toc-7"></a>
#### 2.2.3 知识卡片风

适合知识科普、方法论、工具教程、AI 应用、法律财经教育类解释文、干货清单文。

主要排版特色：

- 结构清楚，适合用“问题—解释—例子—结论”的方式展开。
- 小标题可以使用“01 先搞清楚一个问题”“02 最容易踩的坑”这类明确表达。
- 可适当使用要点卡片、步骤清单、对比模块和误区提醒。
- 重点句可以高亮成“核心结论”或“记住这一点”。
- 配图优先选择流程图、示意图、数据图、工具截图或案例图。
- 结尾适合整理成行动建议、判断标准或收藏清单。

当文章重在“讲清楚、能收藏、能复用、能指导行动”时，优先选择这种风格。

<a id="toc-8"></a>
#### 2.2.4 高端商务风

适合企业公众号、B2B 品牌、咨询公司、科技公司、金融产业、SaaS 服务、报告解读和解决方案文章。

主要排版特色：

- 整体排版稳重、克制、专业。
- 开头可设置“核心摘要”“主要判断”或“本文看点”。
- 小标题强调逻辑感，可使用“01 行业变化”“02 核心挑战”“03 解决路径”。
- 文中适合加入数据图、案例图、流程图、方案图或报告截图建议。
- 重点句可用加粗或浅色高亮，但不能过度营销化。
- 结尾适合引导下载报告、预约咨询、了解方案或联系顾问。

当文章需要体现“专业度、可信度、行业洞察和商业价值”时，优先选择这种风格。

<a id="toc-9"></a>
#### 2.2.5 文艺清新风

适合读书号、心理号、女性成长号、校园号、生活方式号、艺术文化号和节日推文。

主要排版特色：

- 视觉轻柔，段落有呼吸感。
- 开头可以用生活场景、季节感、人物片段或一句温柔判断切入。
- 小标题可以更有画面感，不必过度功能化。
- 配图适合选择插画、风景、生活细节、书页、手写感图片等。
- 重点句可以单独成段，不一定强高亮。
- 结尾适合陪伴式表达、轻互动或温和引导。

当文章重在“氛围、情绪、审美和陪伴感”时，优先选择这种风格。

<a id="toc-10"></a>
### 2.3 风格选择规则

模型生成公众号文章时，应先判断内容属于哪一类：

1. 观点、成长、情绪共鸣类文章，优先选择极简留白风。
2. 人物、品牌、生活方式、热点深度类文章，优先选择现代杂志风。
3. 科普、教程、方法论、工具类文章，优先选择知识卡片风。
4. 企业、产业、报告、解决方案类文章，优先选择高端商务风。
5. 读书、心理、艺术、校园、节日类文章，优先选择文艺清新风。

如果用户指定了参考账号或参考文章，应优先提取其主要排版特征，而不是机械照搬。例如参考浪潮工作室一类排版时，可重点借鉴“开头顶图、编号式小标题、文中配图、重点句黄色高亮、整体杂志化”的视觉逻辑，但不能复制原文结构、表达和独特设计。

好的公众号排版不是样式越多越好，而是让内容气质、读者需求和发布目的保持一致。

<a id="toc-11"></a>
## 3. 配图要求

生成完整公众号文章或公众号文档时，模型必须根据文章内容自行配置合适的头图、文中插图和封面图。图片不是可有可无的装饰，而应服务文章的阅读氛围、信息表达和传播效果。

图片来源可以包括：

- 模型自行生成的图片。
- 联网搜索到的公开图片。
- 用户提供的图片素材。
- 根据文章内容生成的信息图、流程图、示意图或封面图。

无论图片来自生成还是联网搜索，都必须与文章主题、排版风格、读者预期和内容基调一致，不能出现违和感、生硬感或单纯为了凑图而插图。

<a id="toc-12"></a>
### 3.1 头图要求

头图应出现在正文开头，用于建立文章的第一视觉印象。

头图应符合文章整体风格：

- 现代杂志风文章，头图应有质感、氛围感和视觉冲击力。
- 极简留白风文章，头图应干净克制，不喧宾夺主。
- 知识卡片风文章，头图可以偏信息化、概念化或场景化。
- 高端商务风文章，头图应专业、稳重，适合企业或行业语境。
- 文艺清新风文章，头图应柔和、有生活感或情绪感。

头图下方必须配一行小号灰色注释。注释不写图片来源、不写生成方式，而应像正文中的一句自然补充，帮助读者进入文章情绪或理解图片与主题的关系。

示例：

- 备考的日子，是青春里最专注的时光。
- 城市醒得很早，很多人的一天从通勤路上开始。
- 当技术变化越来越快，普通人的不安也变得更具体。

<a id="toc-13"></a>
### 3.2 文中插图要求

文中插图应根据正文内容自然插入，通常用于承接段落转折、补充信息、强化情绪、解释概念或呈现案例。

适合插图的位置包括：

1. 重要章节开始前。
2. 复杂概念解释处。
3. 数据、趋势、流程、对比内容之后。
4. 人物故事、品牌故事或场景描写之后。
5. 情绪转折、观点递进或结论强调处。

文中插图应具备至少一种功能：

1. 补充信息：帮助读者理解数据、流程、结构或背景。
2. 强化氛围：让文章的情绪、审美和叙事更完整。
3. 承接结构：在不同章节之间形成自然过渡。
4. 支撑观点：用案例图、截图、图表、场景图辅助论证。
5. 提升可读性：缓解长文阅读压力，让手机端阅读更轻松。

文中插图下方必须配一行小号灰色注释。注释应简短、自然、有画面感，成为文章整体表达的一部分，而不是机械说明图片内容。

示例：

- 很多改变不是突然发生的，而是在日复一日的选择里慢慢显形。
- 看似普通的工作流，正在被新的工具重新组织。
- 真正影响选择的，往往不是信息多少，而是判断成本。

不得出现以下问题：

1. 图片与正文主题无关。
2. 图片风格与文章基调冲突。
3. 图片只是机械插入，没有实际作用。
4. 注释写得像图片说明书，而不是文章的一部分。
5. 严肃内容使用过度娱乐化图片。
6. 商业文章使用廉价感、夸张感或强网感图片。
7. 知识科普文章插入无法辅助理解的氛围图。
8. 图片过多，打断正文阅读节奏。

<a id="toc-14"></a>
### 3.3 封面图要求

封面图应作为正文之后的独立模块输出，用于公众号列表页、转发卡片或文章封面。

封面图必须符合以下要求：

1. 与文章主题直接相关。
2. 视觉焦点清晰，适合缩略图展示。
3. 与正文头图和整体排版风格保持一致。
4. 适合承载文章标题或形成标题联想。
5. 不使用低清、侵权、误导性、猎奇化或与内容不符的图片。
6. 不为了吸引点击制造与正文不一致的视觉暗示。

不同类型文章的封面图选择逻辑：

1. 观点文：突出核心冲突、情绪张力或关键隐喻。
2. 干货文：突出主题关键词、工具感、结构感或问题解决感。
3. 品牌文：突出品牌调性、产品场景或用户利益。
4. 人物故事：突出人物状态、场景氛围或故事转折。
5. 行业分析：突出行业场景、趋势变化或数据感。
6. 活动转化文：突出活动主题、参与价值和行动入口。

封面图下方可以配一行小号灰色推荐语或导读语，用一句简短自然的话概括文章气质、核心问题或阅读价值。

示例：

- 写给每一个正在努力适应变化的普通人。
- 一个看似简单的问题，背后藏着新的消费选择。
- 真正的成长，往往从重新理解自己开始。

<a id="toc-15"></a>
### 3.4 图片来源与验真

图片注释中不直接写来源、出处或生成方式，以免破坏正文阅读体验。

但在文章最后的“内容验真”部分，必须对图片来源进行统一说明：

1. 如果图片由模型生成，应在内容验真中标明“AI 生成图片”。
2. 如果图片来自联网搜索，应在内容验真中注明图片来源和出处。
3. 如果图片来自用户提供素材，应在内容验真中标明“来源于用户提供素材”。
4. 如果图片为示意图、概念图或信息图，应避免让读者误解为真实新闻现场或真实数据截图。

不得：

1. 编造图片来源。
2. 使用来源不明的图片冒充公开资料。
3. 将 AI 生成图片标注为真实照片。
4. 将示意图、概念图写成真实现场图。
5. 使用可能侵犯肖像权、版权或商标权的图片。
6. 用误导性图片强化未经证实的判断。

<a id="toc-16"></a>
### 3.5 图片与正文的一致性

配图必须与文章本身形成自然关系，而不是简单“好看”。

模型在选择或生成图片时，应优先判断：

1. 这张图是否符合文章主题。
2. 这张图是否符合文章基调。
3. 这张图是否能补充信息或增强表达。
4. 这张图是否放在当前位置最自然。
5. 这张图是否会造成误解。
6. 这张图是否与前后文衔接顺畅。
7. 图片注释是否像文章中的一句话，而不是生硬的图片说明。

如果无法获得合适图片，应优先生成符合文章基调的原创图片，而不是强行使用不相关的联网图片。

好的公众号配图应让读者觉得“这张图本来就应该出现在这里”，图片注释也应像正文自然长出来的一句话。
