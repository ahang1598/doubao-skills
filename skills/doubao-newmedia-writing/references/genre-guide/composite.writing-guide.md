# 复合型创作方案排版格式

本文件只处理复合型创作方案的路由顺序、文档分区和排版组织。具体写作方法、图文要求、分镜要求和事实核查，仍由命中的创作类型主干及其 samples 决定。

## 适用范围

当用户一次要求 2 个或 3 个受支持创作类型时使用：

- 小红书图文笔记 + 公众号文章。
- 小红书图文笔记 + 短视频分镜脚本。
- 公众号文章 + 短视频分镜脚本。
- 小红书图文笔记 + 公众号文章 + 短视频分镜脚本。

用户只提到“抖音文案、快手文案、视频号文案”时，先确认是否需要 3 分钟以内可拍摄短视频分镜脚本；确认后再纳入短视频分镜脚本分区。

## 路由顺序

1. 先确认复合型类型清单，按用户明确要求保留类型；未明确的平台不主动扩展。
2. 读取 `lark-doc.writing-guide.md` 创建一份飞书文档/Lark Doc，并用“新媒体创作方案｜主题”作为文档标题。
3. 先写入复合型外层骨架；`方案总览` 必须是文档首个一级标题，并包含核心主题、目标受众、创作类型、核心策略等表格字段。
4. 按命中类型逐一读取对应主干：
   - 小红书图文笔记：`xhs.writing-guide.md`。
   - 公众号文章：`wechat.writing-guide.md`。
   - 短视频分镜脚本：`short-video.writing-guide.md`。
5. 每个类型生成完成后立即写入该类型独立分区，再进入下一个类型。
6. 最后写入复合型发布检查，核对各分区文本、图片、事实边界、发布素材和交付链接。

## 文档排版骨架

复合型创作方案文档使用“一份文档、多区块成稿”的结构。

```xml
<h1>方案总览</h1>
<table>
  <thead><tr><th>项目</th><th>概述</th></tr></thead>
  <tbody>
    <tr><td>核心主题</td><td>...</td></tr>
    <tr><td>目标受众</td><td>...</td></tr>
    <tr><td>创作类型</td><td>小红书图文笔记、公众号文章、短视频分镜脚本</td></tr>
    <tr><td>核心策略</td><td>创作思路、内容策略、平台差异、事实核查等差异化描述</td></tr>
  </tbody>
</table>

<h1>一、小红书图文笔记</h1>
<p>按 xhs.writing-guide.md 与小红书排版 sample 写入完整图文结构。</p>

<h1>二、公众号文章</h1>
<p>按 wechat.writing-guide.md 与公众号排版配图规则写入完整文章结构。</p>

<h1>三、短视频分镜脚本</h1>
<p>按 short-video.writing-guide.md 与短视频 output-format 写入完整 7 列分镜表及配套模块。</p>

```
只写命中的类型分区，分区编号按实际顺序顺延。

## 自检规则

交付前严格确认：

1. 文档只有 1 个入口，所有类型内容都写入同一份飞书文档/Lark Doc。
2. 文档首个一级标题是 `方案总览`，且先于所有类型分区出现。
3. 已命中的每个类型都读取了对应主干 guide。
4. 各类型分区都包含文本和图片/图片落位。
5. 小红书、公众号、短视频的排版差异被保留。
6. 文档没有把多个平台内容混成单一随机结构。
