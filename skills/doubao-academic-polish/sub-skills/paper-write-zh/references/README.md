# paper-write-zh 参考总表

本文件是 `paper-write-zh/SKILL.md` 的路由索引，只回答“这次该读哪个参考”。它不替代主入口的 S0-S5 阶段门控，也不放宽 S1 文献真实性、S4 审查和 S5 状态交付要求。

执行优先级：

1. `paper-write-zh/SKILL.md` 始终是最高优先级入口，负责任务粒度、Draft/Final Mode、S0-S5 阶段、文献真实性门控和状态交付。
2. 本目录的通用能力模块（common 层）负责跨学科共用的检索、证据、章节和语言方法。
3. 学科规范层（`stem/`、`hss/`）负责理工/医学与法学/人文社科的差异化结构、术语和合规。
4. 学科目录内的细分参考文件只在需要更细的体例、模板、期刊格式或清单时按需读取。

## 一、通用能力模块（common 层，跨学科共用）

这些能力不分文理，任何论文都可能用到。它们从原 `stem/` 子目录上提到 `references/` 顶层，避免文理各存一份造成重复。

| 模块 | 路径 | 何时读 |
|---|---|---|
| 文献检索与真实性 | `literature-search/SKILL.md` | 需要中文知网/CNKI 检索、GB/T 7714 引用、英文文献池、引用真实性核验时。中文 browser-task 强制门控在此。 |
| 文献综述组织 | `literature-review/SKILL.md` | 把真实文献组织成主题脉络、研究缺口和综述段落时（写作组织方法，与检索门控分工） |
| 证据驱动写作 | `evidence-driven-writing/SKILL.md` | 写引言、相关工作、讨论、结论等引用/实验/图表驱动段落时；证据图、实验协议、Mock/PLANNING DATA 边界和图表生成协议在此 |
| 章节写作方法 | `writing-chapters/SKILL.md` | 写作、改写、续写单章或多章，需要章节结构、任务卡、两阶段审查、公式表格生成时 |
| 中文语言与去 AI 味 | `writing-core/SKILL.md` | 需要自然中文表达、去 AI 痕迹、正文排版、语言自检时。中文语言收尾由本模块承担，不回流英文侧 polish。 |

## 二、学科规范层（个性下沉）

先在主入口 S0 判定学科分支，再进入对应目录。文理分开：

| 分支 | 路径 | 覆盖 |
|---|---|---|
| 理工 + 医学 | `stem/SKILL.md` | 工程、计算机、控制、材料、机械、电子、农工、自然科学实验、算法模型；医学、生物、临床、护理、公共卫生、药学 |
| 法学 + 人文社科 | `hss/SKILL.md` | 法律、法条、判例、司法解释、比较法；文学、历史、哲学、政治、经济、管理、社会、教育、传播、艺术、马克思主义理论等 |

学科交叉时按研究方法判定：有实验、基准、模型、算法、临床数据走 `stem/`；有文本、史料、概念辨析、问卷、访谈、法条走 `hss/`。

### hss 细分参考文件

`hss/SKILL.md` 下另有六份细分参考，需要更细内容时按需读取：

- `hss/citation-styles.md`：GB/T 7714、脚注注释、APA、Chicago/MLA、著者-出版年制等引用体例示例。
- `hss/section-templates.md`：标题、摘要、引言、综述、理论框架、方法、正文、结论等章节模板与范文技法。
- `hss/journal-formats.md`：人文社科常见目标期刊格式参考。
- `hss/literature-review-guide.md`：人文社科文献综述方法论、检索式和筛选标准。
- `hss/formatting-output.md`：公式、表格、章节、参考文献和 AI 痕迹的输出格式检查。
- `hss/submission-checklist.md`：投稿前结构、格式、引用和文献真实性检查清单。

## 三、S1 文献强制约束索引

以下约束是主入口 S1 的硬门控，落在 `literature-search/SKILL.md`，在这里索引以便快速定位：

- 中文真实文献、GB/T 7714、CNKI/知网来源必须显式调用 Doubao browser-task；未显式尝试 browser-task 前，不得先用 `scholar_search`、`general_search`、普通搜索工具或搜索面板建立中文候选文献池。
- 英文文献可使用 `scholar_search` 获取真实元数据与引用格式，用于正式参考文献、文内引用和元数据层判断；方法细节、实验结果、局限边界等全文级判断仍需摘要、DOI/出版社页面、arXiv 页面、browser-task 页面可见内容或用户材料。
- 英文文献池和中文文献池是两个独立门槛。技术类、医学/生物、法学和人文社科论文完成英文检索后，中文文献未达阈值时阶段保持 S1，必须继续走 `literature-search/SKILL.md` 的中文来源检索。
- 涉及文献综述、研究现状、related work 或任何带引用正文时，正文必须显式出现文内引用标识；不能只有文末参考文献列表而正文没有挂引用。

## 四、边界

本技能只负责中文论文的产出侧（写出来、改漂亮）。投稿前评审、稿件挑硬伤、判断研究想法值不值得做，走姊妹技能 `doubao-academic-evaluator`，不在本目录处理。
