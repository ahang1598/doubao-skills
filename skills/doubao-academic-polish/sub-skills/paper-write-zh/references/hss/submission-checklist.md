# 投稿格式与合规自检清单

交付前对照本清单逐项核对。

## 结构完整性
- [ ] 标题（中/英文）准确、凝练
- [ ] 作者信息、单位、基金项目齐全
- [ ] 中文摘要（报道性约400字/指示性约150字，硕博500–1000字；报道性四要素齐全）
- [ ] 英文 Abstract 与中文对应
- [ ] 中/英文关键词各 3–8 个（分号隔开，末尾不加标点，特定含义词加双引号）
- [ ] 引言含研究问题与创新点
- [ ] 文献综述有述评、指出空白
- [ ] 研究方法清晰、与问题匹配
- [ ] 结论回应研究问题，含局限与展望

## 文献综述专项（对照五大常见问题反向自检，详见 `literature-review-guide.md`）
- [ ] 未照搬他人综述，评述均基于阅读原文
- [ ] 有自己的分类、归纳与评述，而非罗列堆砌
- [ ] 文献选择有明确范围，研究性著作为主（知识性材料未混入）
- [ ] 综述与主题紧密相关，结论已回应综述提出的问题
- [ ] 围绕关键词聚焦重点，主线清晰
- [ ] 重要概念已追溯源头（最早提出者）
- [ ] 批评他人时引用了原作者/原著，未据二手或译本妄断

## 结论与建议专项（溯源原则，详见 `section-templates.md` 第 9 节）
- [ ] 每条结论都能追溯到正文（第 8 节）的检验结果或论证，无凭空结论
- [ ] 结论未引入新数据、新变量、新论点
- [ ] 结论表述不夸大（未把相关说成因果、未把局部显著说成普遍成立）
- [ ] 每条建议逐条对应某条结论，未超出论证范畴
- [ ] 无"应加强、应重视"式脱离本文证据的空泛套话
- [ ] 研究局限具体且与本研究相关，未来展望与局限呼应

## 引用与注释
- [ ] 默认采用 GB/T 7714—2015 顺序编码制（文内 [1]、文末按引用序号著录）；仅目标期刊明确要求时才用其他体例
- [ ] 全文统一使用一种体例
- [ ] 文内标注与文末著录一一对应
- [ ] 无遗漏、无编造引文
- [ ] 直接引用加引号并标页码
- [ ] 文献类型标识（[M]/[J]/[C]/[D]/[N]/[EB/OL]）、标点符合 GB/T 7714 规范

## 语言与格式
- [ ] 学术语言客观严谨，无口语化
- [ ] 术语前后一致
- [ ] 图表有编号与标题，正文有引述
- [ ] 数字、标点、计量单位规范
- [ ] 篇幅符合目标期刊要求

## 期刊格式专项（按学科选定模板，详见 `journal-formats.md`）
- [ ] 已按学科选对期刊格式（文史哲法→《中国社会科学》；经管→《经济研究》；前者未涉及按后者补充）
- [ ] 题名 ≤20 汉字 / 英文 ≤10 实词，中英文含义一致
- [ ] 摘要字数与人称符合所选期刊（《中国社会科学》120–300字、第三人称）
- [ ] 关键词 3–8 个，尽量取规范主题词
- [ ] 图表数量合规（《经济研究》：表≤6、图≤4、总数≤8）
- [ ] 数学表达式与参考文献条数合规（《经济研究》：表达式≤20、参考文献≤50）
- [ ] 引言+文献述评、结论篇幅达标（《经济研究》：前者≥5000字、后者≥1500字）
- [ ] 已说明本文与相关文献（尤其目标刊近年文章）的学术承启与区别
- [ ] 排版细则符合范本（字体字号、三线表、脚注编号、参考文献著录格式等）

## 输出格式专项（防 AI 生成稿常见排版问题，详见 `formatting-output.md`）
- [ ] 公式渲染为正式数学格式并编号，无 `_{i,t}`、`β_0`、`<sub>`、`<sup>` 等未渲染痕迹
- [ ] 公式符号解释与公式中系数完全一致（无"γ 和 γ"式重复/简写）
- [ ] 指标体系/描述性统计/相关性矩阵/回归结果均为标准表格，未串成正文或字符串
- [ ] 表格数字不粘连，显著性星号紧跟系数、未被换行拆散
- [ ] 各级标题（一、/（一）/1.）独立成行，未与正文或表题/表注/下节标题粘连
- [ ] 变量定义按被解释/解释/机制/控制变量分层编号
- [ ] 中文正文无必要英文普通词；术语首现给中文译名+英文括注
- [ ] 参考文献逐条换行、条目内部完整，无硬换行拆断
- [ ] 已清除"由 AI/豆包生成""请谨慎参考"等生成提示与占位符；如需 AI 声明按期刊规范处理

## 学术诚信
- [ ] 转述已改写并标注来源
- [ ] 数据/史料真实可溯源
- [ ] 查重通过（如有要求）

## 文献真实性专项（涉及联网检索/补充文献时必查，以 `../../SKILL.md` 的 S1 文献核验门控为最高优先级）
- [ ] 每条正式参考文献均已通过 CNKI/知网 browser-task 页面可见信息、知网导出引用、Google Scholar/DOI/出版社页面/arXiv 页面或用户可核验回填确认；百度学术、万方聚合页、普通搜索和 API 元数据只作为候选线索，不能单独作为正式引用依据
- [ ] 无任何凭记忆或推测"生成"的文献（作者、篇名、期刊、年份、卷期、页码、DOI/URL 逐项真实）
- [ ] 题录信息与检索结果逐项一致，无臆造或张冠李戴
- [ ] 无法核实的文献已剔除，未"宁滥勿缺"
- [ ] 关键文献保留了可回溯的检索路径（数据库 + 检索词 / 链接 / DOI）
- [ ] 转述他人观点忠于原文，未为论证需要曲解或夸大
