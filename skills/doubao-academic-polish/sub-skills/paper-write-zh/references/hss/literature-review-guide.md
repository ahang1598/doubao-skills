# 文献综述撰写指南（方法论）

> 本文件系统说明文献综述的本质、目的、常见问题、收集途径、撰写方法与注意事项，尤其适用于**人文社科（人类学、社会学、历史学、民族学等）**的青年研究者。其核心命题是：**撰写文献综述是发现和确立学术问题的关键步骤**。
> 与 `section-templates.md` 第 5 节配合使用：第 5 节给出综述的章节写法与范文技法，本文件提供更完整的方法论与避坑指南。

## 一、什么是文献综述

- "综"= 归纳"百家"之言，综合分析整理；"述"= 通过阅读文献，对作者的不同观点进行叙述与评论。
- 文献综述是在**全面搜集、阅读大量研究文献**的基础上，对某一问题在给定时期内的研究成果、存在问题、发展趋势进行**系统性、全面的叙述与讨论**。
- 它**不是单纯列出文章**，而是辨别资料、结合自身研究进行综合评估，对相关研究作系统分析评价并作合理趋势预测，为新课题确立提供支持和论证。
- 分类：**综合性综述**（以一个学科/专业为对象）与**专题性综述**（就某一论题为对象）。

## 二、作用与目的

1. **高度浓缩**几十、上百乃至上万篇相关文献，使杂乱无序的文献条理化、系统化。
2. **为自己的研究铺路**：了解题目与相关领域同行研究到了何种程度、有哪些不足，从而发现继续研究的空间。
3. **发现成就，更要发现问题**：问题可能出在方法、工具、材料或论证过程上。
   - 工具有问题 → 可改进工具；方法有问题 → 可换方法；**材料有问题 → 是"硬伤"，无法补救**。
4. **训练科研能力**：提高对学术概念的归纳分析与综合能力，培养独立科研能力。
5. **学会批判**：最重要的不只是学习别人的研究，更要学会批判、批评前人研究。

## 三、五个常见问题（务必避免）

1. **照搬他人类似的文献综述**（最致命、最低级的错误）。别人怎么评述就照样评述，自己没看过原文，必然出错。模仿可以，但绝不能照抄——每个研究的主题不同，时间与问题变了，分类方式也应改变。
2. **大量罗列堆砌文献，没有自己的评述、分类与归纳分析**；或只引导师及导师圈的文献。常以"没有资料"为借口，实则是检索不充分。
3. **文献选择太随意**：未界定选择范围。**知识性材料**（如年鉴、方志等背景资料）不必放进综述；综述主要放入**研究性著作**。
4. **文献综述与文章主题脱节**：写完综述后，后续章节与结论不再回应它，使综述"可有可无"。**结论一定要回应文献综述中提出的问题**，论文每一章都应与综述连接。
5. **文献综述没有重点**：洋洋百页却抓不住核心。解决办法是**建构一个关键词**，围绕关键词聚焦相关文献重点讨论。

## 四、文献资料的收集途径

1. **善用检索工具与数据库**：如专业古籍/史料数据库（按关键词检索历朝历代记载）、搜索引擎；大数据能体现关键词的使用频率，借此了解某问题的学术关注度。
2. **善用原始文献与三次文献**：
   - **学位论文**（尤其博士论文，其综述部分通常较认真）可联网检索，了解全国研究状况。
   - **三次文献** = 综述、述评、百科全书、年鉴、手册（多由权威作者撰写，是快速入门与把握脉络的捷径）。
3. **追溯源头**：人文科学研究要追溯到概念**最早是谁提出的**；这与自然科学不同（自科常认为五年前文献已不重要）。电子检索的缺陷是早期文献收录不全。
4. **谷歌学术**适合检索外文文献，能找到很古老的文章。

### 四·补：联网检索操作指引（检索式构造与筛选标准）

> 配合 `../../SKILL.md` 的 S1 文献核验门控使用。**所有文献须实际检索命中、逐条核实后方可写入，严禁编造。**

#### A. 检索前：拆解概念、准备检索词

1. **从研究问题中提取 2–4 个核心概念**（变量/对象/情境）。如"生态位宽度对电影市场绩效的影响"→ 核心概念：`生态位宽度`、`电影市场绩效`。
2. **为每个概念扩展同义词/近义词/上下位词，并备中英文对照**（中文库与英文库分别检索）：
   - 例：`生态位宽度` ≈ niche width / niche breadth；`市场绩效` ≈ market performance / box office（票房）。
   - 兼顾**学科术语差异**（同一概念在不同学科叫法不同）与**历时变化**（早期与近期用词）。
3. **确定检索范围**：时间区间（如近 5–10 年为主，经典源头另行追溯）、文献类型（期刊论文、博士论文、专著、综述）、学科领域、语种。

#### B. 检索式构造（布尔逻辑 + 字段限定）

- **布尔运算符**：`AND` 缩小（多概念同时命中）、`OR` 扩大（同义词并列）、`NOT` 排除（剔除干扰义）。
  - 通式：`(概念1同义词A OR 概念1同义词B) AND (概念2同义词A OR 概念2同义词B)`
  - 例（中文）：`(生态位宽度 OR 生态位宽度理论) AND (电影 OR 影片) AND (市场绩效 OR 票房 OR 满意度)`
  - 例（英文）：`("niche width" OR "niche breadth") AND (film OR movie) AND ("market performance" OR "box office")`
- **精确短语**：用英文双引号包裹固定搭配，如 `"文献综述"`、`"belt and road"`，避免词被拆开。
- **字段限定**（缩小到关键位置，提高精度）：优先限定**题名 / 关键词 / 篇关摘**字段，泛查再用全文字段。
- **各渠道要点**：
  - **中国知网（CNKI）**：用"高级检索"，按 `主题 / 篇关摘 / 题名 / 关键词 / 作者 / 文献来源` 分字段组合；可勾选来源类别 **CSSCI / 北大核心 / SCI** 限定高质量文献；可限定年份与学科。
  - **万方数据**：高级检索同理，支持按学科、年份、文献类型（期刊/学位/会议）筛选；适合补充知网未收录文献。
  - **百度学术**：聚合多源，支持按时间、领域、被引频次排序；适合快速查被引量与全文获取入口。
  - **谷歌学术（Google Scholar）**：检索外文最强。支持 `intitle:` 限定题名、`author:"姓 名"` 限定作者、`source:期刊名` 限定来源、年份范围（Since 2020）；条目下"被引用次数"和"相关文章"便于滚雪球。
- **迭代调整**：结果过多→增加 AND 条件或限定字段/核心库；结果过少→增加 OR 同义词、放宽时间或字段。

#### C. 文献筛选标准（先粗筛后精筛）

1. **粗筛（看题录）**：读**题名、关键词、摘要、结论**判断相关性；不相关直接排除（呼应第五节"先读摘要与结论判断相关性"）。
2. **精筛（五维标准，详见第五节）**：**代表性、显著性（被引/影响力）、相关性、适时性（时效）、简捷性**。优先选：
   - **来源权威**：CSSCI / 北大核心 / 权威集刊 / 高水平英文期刊（SSCI/A&HCI）；目标期刊近年文章尤其要看。
   - **被引与影响**：被引频次高、被权威学者反复引用者（"从文献中找文献"，看谁在引）。
   - **时效与经典并重**：近期文献把握前沿动态；同时**追溯概念最早提出者**（人文社科尤重源头，见第三、四节）。
3. **剔除标准**：非学术来源（普通博客、营销号、问答站、无出处网络文）、与主题脱节、信息无法核实者一律不用。
4. **去重与配额**：剔除重复条目；注意目标期刊的参考文献条数上限（如《经济研究》≤50 条），按重要性取舍，避免给非权威作者列过多条目。

#### D. 真实性核验与留痕（写入前最后一关，逐条执行）

> 核验的目标有二：**(1) 这条文献真实存在；(2) 它确实支撑你要引用的那个观点/数据。** 二者缺一不可。下列步骤对**每一条**拟引用文献都要走一遍——无论文献是自己检索来的，还是**用户提供的**（用户给的同样要核验，不可直接采信）。

**第 1 步 · 来源核验**
- 正式引用的最低门槛以 `../../SKILL.md` 的 S1 为准：中文文献需有 CNKI/知网 browser-task 页面可见信息、知网导出引用或用户可核验回填；英文文献需有 Google Scholar/DOI/出版社页面/arXiv 页面或用户材料确认。
- 多源交叉验证是高风险文献和 Final Mode 的推荐加强项，不是覆盖主入口的硬门槛。若同一条文献已由 CNKI 导出、DOI/出版社页面或期刊官网确认，可以作为已核验来源；若只有百度学术、普通搜索或 API 元数据命中，仍只能作为候选线索。
- 若各渠道题录互相矛盾（年份/卷期/页码不一致），以**期刊官网或出版社页面**为准，存疑处向用户说明。

**第 2 步 · DOI / 链接实地解析**
- 有 DOI 的：把 DOI 拼到 `https://doi.org/<DOI>` **实际打开**，确认跳转到的正是该篇文章且题录吻合——**不能只因"有一串像 DOI 的字符"就认定为真**。
- 无 DOI 的中文文献：在知网/万方打开**条目详情页**确认存在，记录可访问链接。

**第 3 步 · 题录内部一致性闭环**
- 核对**作者—篇名—期刊—年份—卷期—页码**彼此自洽：年份要与卷期对应、页码要落在该期的页码范围内。
- 特别警惕"**真作者 + 真期刊 + 错年份/错卷期/错页码**"的拼接式错误（这是编造或张冠李戴的典型形态）。
- 核对作者：该作者的公开著作列表中确有此文（可查作者主页、机构库、谷歌学术个人页）。

**第 4 步 · 引文内容核验（防错误归因）**
- 打开原文（或至少摘要/可见全文片段），**确认被引的观点、数据、结论确实出自该文献、且与你的转述一致**；不能只凭题名"想当然"。
- 直接引用须核对原文逐字与**页码**；转述须忠于原意，不为论证需要而曲解或夸大。

**第 5 步 · 期刊与文献状态核查**
- 确认**期刊真实且非假冒/克隆期刊**，警惕掠夺性期刊与预警期刊（可参考中科院国际期刊预警名单、目标数据库是否正规收录）。
- 核查该文献是否被**撤稿（retraction）或重要勘误**（如 Retraction Watch、期刊撤稿声明）；已撤稿文献不得作为论据。

**AI / 编造文献的可疑信号（命中任一即须重点复核或弃用）**
- 题名通顺、像模像样，但任何权威渠道都**检索不到**；
- DOI **无法解析**，或解析到的文章与题录不符；
- 作者是该领域真实学者，但其著作列表里**没有这篇**；
- 卷期、页码与该刊该年实际情况**对不上**；
- 全网仅在"参考文献列表"里出现，找不到任何其他痕迹（原文、被引、库收录）。

**核验失败的处理（铁律）**
- 任一条无法通过上述核验 → **剔除，宁缺勿造**；不得"静默补全"或凭推测修正题录。
- 若用户坚持使用某条无法核实的文献 → **显式标注"出处待核实"并提示风险**，由用户定夺，不擅自当作已核实写入。

**留痕**
- 为关键文献记录"**数据库名 + 实际检索式 + 命中链接/DOI**"，便于用户复核与回溯。

## 五、基本方法三要点

1. **概括归纳**：
   - 可按**时间顺序由近及远**（最新研究通常已包含早前资料）。
   - 先读**摘要与结论**判断相关性，相关则归入对应文件夹分类管理。
   - 对**结论不一致**的文献要对比分析：看差异何在、为何产生差异，从而找出自己的研究路径。
   - 每项资料须**注明完整出处**。
2. **评述**：对文献要有基本判断，评价其优点与不足。文献选择标准：**代表性、显著性、相关性、适时性、简捷性**。"从文献中找文献"，看哪些权威学者在引用，有助判断文献重要性。
3. **建议**：在总结前人研究后，对新的研究方向提出建议（常写"回顾与展望"），把问题列出并指出从方法或理论上如何解决，即发展趋势。

## 六、撰写模式与结构

### 6.1 导论与主题
- **导论**（介绍性）：介绍主题及其重要性，理清首要问题，简介各篇文章/作者及其异同。讲清学科发展面临的最大问题、讨论最多的议题、有哪些代表性作者、A 类与 B 类理论有何不同。
- **主题**：先分析 A 的观点理论，再比较 A、B，找出共性与差异，**提出一个中心议题**；结论往往要提出一个优于各家的、更好的理论立场。

### 6.2 标准组成部分
综述格式相对多样，但一般包含：(1) 题目 (2) 作者单位 (3) 摘要 (4) 关键词 (5) 前言 (6) 主题 (7) 总结 (8) 参考文献。可按"前言—主题—总结—参考文献"四部分**先拟提纲再撰写**。

- **前言**：阐述为何写此主题，给出概念定义，规定综述的**学科范围**（切忌过宽过杂）与**时间范围**（声明引用文献起止年份），最后点出现况或争论焦点，引出核心主题。
- **主题**（主体）：写法多样，可按**年代顺序 / 不同问题 / 不同观点比较**综述。无论哪种，都要归纳整理、分析比较，阐明历史背景、现状与发展方向并加以评述。注意代表性、科学性、创造性。课题申报常有参考文献数量上限，需谨慎取舍（避免给非权威作者列过多条目，注意评审者与所列学派的关系）。
- **总结**：扼要总结全文主题，列出有研究的作者，提出自己的见解。
- **参考文献**：先单独排序再并入全文；对同一人（尤其在世学者）的引用不宜过多。

### 6.3 学术史/理论类综述的四种组织方式
撰写理论类文章（本身即一种综述）可按：
1. **时间顺序**（做学术史常用）；
2. **理论流派**（人类学常用，但当代流派不易划分，"越近越难产"，给在世者盖棺定论也有学术伦理困难）；
3. **问题划分**（按学科历史上主要研究的问题）；
4. **地域划分**（如英、美、法人类学各有何理论）。

## 七、注意事项

1. **文献要尽量收全**：掌握全面大量的资料是写好综述的前提，不要随便收一点就开始写。
2. **注意代表性、可靠性、科学性**：引用文献出错将是"硬伤"；历史学研究尤其讲究**史料准确性**，史料错了论文即"判死刑"。
3. **比较分析、合乎逻辑**：不能教科书式地把理论、学派简单汇总陈述一遍，要让人感到研究是有规律、合逻辑的。
4. **批评须引原作者、读原著**：批评他人不足时要引用原作者原文，防止曲解；**不能仅凭二手/翻译资料就批判原作者**（如《共产党宣言》"工人阶级"被俄译为"无产者"之鉴）。博士尤其要少用译著，重要理论来源应读原著。
5. **注明来源**：采用文献观点须注明来源；模型、图表、数据必须注明出处，不能含糊（曾有学者因地图被他人无出处引用而打版权官司）。
6. **引用学术性期刊**：非学术期刊、报刊、纯网络文献的引用要谨慎（做互联网研究除外）。
7. 文献综述不同于读书笔记（仅摘要点）与读书报告（不需找问题、评论），也不同于一般科研论文（无须证明对错，只是"就文献谈文献"）。

## 八、案例：以《从乡村宗族到城市宗族——当代宗族研究的新进展》为例

周大鸣此文的操作方式（可作模板）：
1. **写作之初先确定主题**，划分若干部分：
   - 第一部分：讨论**概念与来源**（宗族的定义有哪些，对宗族分类的不同思路归类梳理；为何成为重要话题）。开篇先**给核心概念下定义**。
   - 第二部分：讨论该概念的**历史变迁**。
   - 第三部分（重点）：宗族如何**从农村宗族发展到城市宗族**——这是跨学科（人类学、历史学、经济学）的研究。
2. **将海量文献挑选分类后，放入这三大架构中撰写**。
3. **提炼核心观点**：综述要有学术价值，必须有一个核心观点。本文核心观点——中国正从乡村社会向城市社会转变，那么"从农村宗族变为城市宗族后应研究什么？"此前无人讨论过，这就是**创新点**。没有核心观点与探讨，文章就没有发表意义。
