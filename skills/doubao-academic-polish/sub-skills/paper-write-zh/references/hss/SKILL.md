# 人文社科与法学学科写作专项

本参考承载人文社科与法学两类学科的写作规则。它是主入口 `paper-write-zh/SKILL.md` 在 S0 判定为“法学”或“人文社科”分支后的学科规范层，不替代主入口的 S0-S5 阶段门控、S1 文献真实性门控和 S5 状态交付。

通用能力不在本参考重复：文献检索与真实性走 `../literature-search/SKILL.md`，文献综述组织走 `../literature-review/SKILL.md`，证据图走 `../evidence-driven-writing/SKILL.md`，章节写作方法走 `../writing-chapters/SKILL.md`，中文语言与去 AI 味走 `../writing-core/SKILL.md`。更细的人文社科引用体例、章节模板、期刊格式、文献综述方法论、输出格式和投稿清单，见本目录下的六份参考文件（见第四节）。本参考只写“人文社科/法学与理工不同”的那部分。

使用时先判断论文所属学科和任务类型，再只读取对应小节；不要把法学的规范分析结构套到普通人文社科论文，也不要把工程话语套到任一分支。

## 一、适用范围与分流

| 命中条件 | 分支 | 读取小节 |
|---|---|---|
| 法律、法条、司法解释、裁判文书、比较法、制度建议 | 法学 | 第二节 |
| 文学、历史、哲学、政治、经济、管理、社会、教育、传播、艺术、马克思主义理论等 | 人文社科 | 第三节 |

理工、工程、计算机、自然科学实验、医学、生物、临床不走本参考，改用 `../stem/SKILL.md`。学科交叉或用户只说“文科/社科”时按研究方法判定：涉及法条、判例、司法解释、制度规范优先走法学；实证/计量/问卷走社会科学实证写法；文本、史料、概念辨析走人文学科思辨写法。

## 二、法学论文

### 2.1 学科特点

法学论文以法律论证、规范分析、案例评析和制度建构为核心。引用对象不仅包括学术文献，还包括法律文件、司法解释、裁判文书和域外法材料；语言应严谨、客观、可争辩，避免口语化判断。法学不直接套普通人文社科结构。

### 2.2 专用结构

- 问题导向：问题的提出、现行规定及不足、域外经验与比较、完善建议、结语。
- 规范分析：概念界定、构成要件、法律效果、适用问题、结论。
- 案例研究：案情简介与争议焦点、裁判理由、学理评析、制度反思。
- 比较法：我国现行规定、域外立法例、比较与借鉴、完善建议。

### 2.3 引用规范

- 著作示例：王利明：《民法总则研究》，中国人民大学出版社2018年版，第156页。
- 期刊示例：张明楷：《刑法学中的概念使用与概念界定》，载《法学研究》2021年第3期，第75页。
- 法律文件示例：《中华人民共和国民法典》第1165条第1款。
- 案例示例：张某诉李某合同纠纷案，最高人民法院（2020）最高法民终123号民事判决书。
- 用户指定学校或期刊格式时，以用户格式为准；不能凭空补法条、案号或判决结果。更多体例示例见 `citation-styles.md`。

### 2.4 术语规范

- “法院判错了”改为“该判决在法律适用上存在值得商榷之处”。
- “这个规定不合理”改为“该条款在适用中存在以下问题”。
- 论证角度可使用文义解释、体系解释、目的解释、历史解释、比较解释，但每种解释都应对应具体法条或制度语境。

### 2.5 检查清单

- [ ] 问题意识明确，争议焦点可辨认。
- [ ] 法条、案例、司法解释引用准确完整。
- [ ] 论证层次区分规范解释、案例评价和制度建议。
- [ ] 域外经验没有被直接移植，已说明适用边界。
- [ ] 表达客观，不使用口语化或绝对化判断。

## 三、人文社科论文

### 3.1 适用范围与分流

人文社科分支覆盖文学、历史、哲学、政治学、经济学、管理学、社会学、教育学、传播学、艺术学、马克思主义理论等学术论文写作、修改、综述、开题和投稿检查。

进入本节后先判断任务粒度：

- 整篇论文或重大返工：回到主入口 Draft Mode，先定论文类型、学科、范式、结构、文献池和材料计划。
- 单独章节或段落：使用主入口 S0-S5 简化流程，只输出本轮章节，但仍保留文献、事实和格式风险说明。
- 润色、降 AI 味或格式诊断：结合 `../writing-core/SKILL.md`；若改动涉及引用、史实、数据或强判断，先做最小核查。
- 文献综述、研究现状或参考文献补充：先回到主入口 S1 和 `../literature-search/SKILL.md`、`../literature-review/SKILL.md` 门控，不能凭记忆补文献。

### 3.2 学科特点与研究范式

人文社科论文更依赖理论解释、文本证据、历史材料、调查资料和论证逻辑，结构比 IMRaD 更灵活。经典文献可以与近年研究并用，但必须说明其理论位置，不要只按年份堆砌。

范式按研究方法而不是学科标签判断：

- 文学、历史、哲学、艺术等常见文本细读、史料考辨、概念辨析和思辨论证，重点是材料解释、理论位置和论证层次。
- 政治、经济、管理、社会、教育、传播等常见问卷、访谈、统计、案例和政策分析，重点是研究问题、变量/材料来源、方法边界和结论可追溯。
- 同一学科可跨范式。教育学、社会学、传播学等若使用计量或问卷，按社会科学实证结构写；若做概念史、教育哲学或文本分析，按思辨/文本结构写。
- 用户提供目标期刊、学校模板或导师要求时，结构和格式以该要求为准。

### 3.3 专用结构

整篇论文默认结构：

| 类型 | 推荐结构 |
|---|---|
| 人文社科期刊论文 | 题目；摘要；关键词；引言/问题提出；文献综述或研究述评；理论框架/概念界定/研究设计；正文论证或实证分析；结论与讨论/政策建议；参考文献/注释 |
| 人文社科学位论文 | 摘要；Abstract；绪论；文献综述；理论基础/概念界定/研究设计；主体章节；结论与展望；参考文献；附录/致谢 |
| 思辨或文本分析论文 | 问题提出；核心概念与理论视角；文本/史料/材料分析；分论点递进；总体判断 |
| 社会科学实证论文 | 问题提出；文献综述；理论框架与研究假设；数据/案例/访谈来源；结果分析；讨论与结论 |
| 文献综述/述评论文 | 研究问题；检索与筛选说明；主题分类；主要争议；研究不足；未来方向 |

常用论证组织：

- 总分总：提出中心论点、分论点展开、回到总体判断。
- 递进式：现象描述、原因分析、深层机制、理论建构或方案。
- 对比式：观点 A、观点 B、比较分析、综合评价、本文立场。
- 问题导向：提出问题、文献回顾、理论框架、分析论证、回答问题。

各章节写作框架、范文技法和撰写顺序提示见 `section-templates.md`。

### 3.4 引用与格式规范

- 默认使用 GB/T 7714 顺序编码制；目标期刊或学校明确要求脚注注释体例、APA、Chicago/MLA、著者-出版年制时从其规定。
- 直接引用短句可融入正文；较长引用应另起段并保持格式统一；直接引用必须有页码或可定位出处。
- 间接引用也需要出处，不能把转述变成无来源观点。
- 古籍、译著、经典理论、现代论文、政策文件和统计数据的格式要区分清楚。
- 文献综述不只是罗列，应说明学派、主题、方法或争议之间的关系。
- 具体体例示例见 `citation-styles.md`；目标期刊版式见 `journal-formats.md`；输出格式检查见 `formatting-output.md`；投稿清单见 `submission-checklist.md`。

### 3.5 文献与材料真实性

本节不能降低主入口文献核验门槛：

- 中文文献、国内研究现状和 GB/T 7714 引用优先走 `../literature-search/SKILL.md` 的 CNKI browser-task 门控；没有页面可见来源、知网导出文本或用户可核验回填时，不得生成正式参考文献。
- 英文文献、国外研究现状、APA/Chicago/MLA 等英文引用格式，可通过 `scholar_search` 获取正式元数据与引用格式，也可结合 Google Scholar/DOI/出版社页面/arXiv 或用户提供材料补充更强证据；证据层级规则以 `../literature-search/SKILL.md` 和 `../literature-review/SKILL.md` 为准。
- 史料、政策文件、统计年鉴、访谈材料、案例材料和古籍版本必须说明来源、版本、时间、页码或获取路径；无法核实时只可作为待核验材料。
- 文献综述的方法论与避坑指南见 `literature-review-guide.md`。

### 3.6 术语与论证规范

- 论点应明确、具体、可论证，例如“鲁迅杂文中的讽刺手法体现了其批判现实主义精神”优于“鲁迅很伟大”。
- “我觉得”改为“本文认为”或直接陈述判断。
- “很多学者都这样认为”不能凭空改成具体学者姓名；若没有可核验来源，优先删除该来源性判断，或在正文外标为“待补具体来源”。
- 政策建议必须来自前文分析和证据，不得脱离结论另起宏大口号。
- 经验研究中的变量、样本、访谈对象、案例选择和统计口径必须有说明；没有材料时用“待补材料”标注，不得伪造数据。
- “绝对正确”“完全错误”等绝对化表达改为“具有一定合理性”“存在值得商榷之处”。

### 3.7 检查清单

- [ ] 已判断任务粒度、论文类型、学科分支和研究范式。
- [ ] 中心论点明确，分论点服务于主问题。
- [ ] 文本、史料、访谈、调查或理论证据与判断之间有分析环节。
- [ ] 文献综述有评述和分类，不是摘抄堆叠。
- [ ] 引用体例统一，文内引用与文末参考文献对应。
- [ ] 章节之间存在递进或回应关系，没有重复换词。
- [ ] 结论不引入新论据，政策建议或实践启示来自前文分析。
- [ ] 格式、章节标题、表格、公式和参考文献没有粘连或 AI 提示残留。

## 五、学科附录资源

以下数据库和规范只能作为检索目标或格式依据；需要引用具体文献、法规或判例时，仍必须通过 `../literature-search/SKILL.md` 的门控获取页面可见来源，或由用户提供可核验材料，不能用搜索/API 元数据替代。

- 法律数据库：北大法宝、威科先行、中国裁判文书网。
- 法学核心期刊示例：中国法学、法学研究、中外法学、法学家、法学、法律科学、政法论坛、现代法学、法商研究、法制与社会发展。
- 人文社科引用格式：优先遵循学校或期刊模板；未指定时保持 GB/T 7714 或用户已有格式一致。

## 六、交付提醒

使用本参考后，在回复中简要说明所采用的学科规则、仍缺的材料和需要用户确认的风险点。学科规范不能替代文献真实性、事实准确性和材料核查；涉及这些问题时仍应回到主入口 S1 和 `../literature-search/SKILL.md`、`../literature-review/SKILL.md` 的门控。
