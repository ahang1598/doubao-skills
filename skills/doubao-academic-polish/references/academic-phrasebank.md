# 学术表达备选库

需要更细的措辞支持时翻这里。这是一份按用途分类的常用学术表达清单，挑最贴切、最朴素的那个用，不要每句都堆衔接词。

## 按证据强弱选动词

让动词和证据强度匹配，是最实用的分寸把握。

**强（证据扎实、设计和数据都足够时才用）**
- 英文：show, demonstrate, establish, reveal, identify
- 中文：表明、证实、确立、揭示

**中（合理但不确定时用）**
- 英文：suggest, indicate, support the view that, be consistent with, point to
- 中文：提示、说明、支持……的看法、与……一致、指向

**弱（超出直接观察、属于推测时用）**
- 英文：may reflect, could arise from, appears to, seems likely, might be explained by
- 中文：可能反映、或源于、似乎、看来、或可解释为

形容证据本身：
- 弱：limited, scant, insufficient（有限、不足）
- 在积累中：growing, emerging, accumulating（渐增、新出现、不断积累）
- 强：robust, reliable, convincing, considerable（稳健、可靠、有说服力、可观）

## 描述缺口（精确，而不是夸张）

好的缺口陈述是精确的，不是戏剧化的。

- 用：remains poorly understood / has not been examined in... / has received limited attention / few studies have addressed... / evidence remains sparse for...
- 中文：仍不甚清楚 / 尚未在……情形下被检验 / 受到的关注有限 / 鲜有研究处理…… / 关于……的证据仍很稀少
- 避免：no one has ever studied / completely unknown / ignored by all previous work（从来没人研究过 / 完全未知 / 被所有前人忽视）——这类说法几乎总是站不住。

## 与前人工作比较

对齐（本文与前人一致）：
- These results are consistent with... / This finding accords with... / Our observations broadly support...
- 中文：这些结果与……一致 / 这一发现与……相符 / 我们的观察大体支持……

划清差异（公平地，不贬低）：
- In contrast to earlier reports... / This finding differs from... / One possible reason for this discrepancy is...
- 中文：与早先的报道不同…… / 这一发现不同于…… / 造成这一差异的一个可能原因是……

公平地点出缺口，而不把前人压成稻草人：
- Although previous studies showed..., their performance in... remains unclear.
- Earlier work established..., but did not address...
- 中文：尽管已有研究展示了……，但它们在……方面的表现仍不清楚。 / 早先的工作确立了……，但没有处理……。

## 限制与局限

- These findings should be interpreted with caution because... / A limitation of this study is that... / The generalisability of these results is limited by... / We cannot exclude the possibility that...
- 中文：这些发现应谨慎解读，因为…… / 本研究的一个局限是…… / 这些结果的可推广性受限于…… / 我们无法排除……的可能。

把局限和真实的不确定性来源绑在一起，不要写成空泛的谦辞。

## 含义与未来工作

含义（不要越出证据边界）：
- An implication of this is that... / These findings may help to explain... / This work has implications for...
- 中文：由此引出的一个含义是…… / 这些发现或有助于解释…… / 这项工作对……有启示。

未来工作（应当从一个真实的局限或机会里长出来）：
- Further work is needed to determine whether... / Future studies should examine... / A useful next step would be to...
- 中文：还需进一步工作来确定是否…… / 未来研究应考察…… / 一个有用的下一步是……

## 过渡词（用最小的那个就够）

- 对比：however, by contrast, nevertheless, despite this, whereas（然而、相比之下、尽管如此、与之相对）
- 递进：furthermore, in addition, moreover, also（此外、另外、而且）
- 因果：therefore, thus, consequently, as a result, thereby（因此、从而、结果是）
- 限定：notably, importantly, approximately, in part, at least in this cohort（值得注意、重要的是、大约、部分地、至少在本样本中）

段落之间不要老用 "This suggests..." 开头。换些办法接：重提名词（Such heterogeneity...）、用确定的名词短语（The resulting gradient...）、用分词小结（Taken together,...）、或者在逻辑本来就清楚时干脆不用连接词直接往下走。指示代词开头一段一次就够。
